How can I change the language in the App Store

Very often, users of Apple products are faced with a problem when everything in their favorite application store or iTunes is in English. The reasons for this are different, but obviously not worth the panic. Below we will describe some of the simplest ways that you can change the language to the one you need in the App Store and iTunes.

The essence of the problem

Surely almost everyone had a situation where in their favorite iPhone when entering the application store, the language changed from familiar and native to some unfamiliar one, for example, English, German, French or even Chinese.





Why this happens, and why it happens, is still unclear. But the user is faced with the question of how to change the language back to the one that was before in the App Store. In fact, there is nothing complicated in this, and anyone can cope with this task.

New Apple ID

The easiest and easiest way to change the language in the App Store is to create a new account (Apple ID), which will indicate the desired country with the desired language. You can do this as follows:

  • Go to Settings.
  • Find item iTunes Store, App Store.
  • In the menu that opens, click on your Apple ID and select "Sign out" in the window that appears.
  • Now you need to return to the main screen and go to the App Store app.
  • Next, go to the tab with free applications and download any of them.




  • When you try to download, a pop-up window will appear asking you to log in to your existing account or create a new ID. You need to choose the second option.
  • After that, the process of registering a new account will start, where it will immediately be offered to choose the country of residence. It is from this point that the language in which the application store will work will depend.
  • Well, then everything is simple: all the necessary forms and fields are filled in, a new mailbox is indicated, the account is activated. The new Apple ID is ready, you can now log in.

Change the language on an existing account

If you do not want to create a new account, but at the same time you need to change the language in the App Store on the iPhone, then you can use this method. He is quite working, does not require any special efforts or knowledge, and also significantly saves time. The only thing worth mentioning is that on some earlier and older versions of the iOS operating system this method does not always work. What it is connected with is still not clear, but the fact remains.





So, to change the language on an existing account, you need to do the following:

  • The first and most important thing is to go to the "Settings" menu.
  • Next, find the item iTunes store, App Store.
  • In the menu that opens, you will need to click on your account (Apple ID).
  • In the window that appears, you should choose the first option - "View account information" or "View Apple ID" (depending on the version of the operating system).
  • In the menu that opens, there will be an item "Country / Region", which you need to click.
  • After that, it will be possible to change the country and, accordingly, the language into the necessary ones.
  • All that remains to be done later is to accept the user agreement.




Well, if for some reason this method did not help, then you should not be upset. There is another method that explains how you can change the language in the App Store.

Change of language through the "iTunes" application

If the first way to change the language does not work, and the second simply does not work, then not everything is lost - iTunes will help. First you need to download the installation file of the program from the official site and install the application on your Windows PC or Mac.

Now directly about how you can change the language through the App Store in the App Store:

  • The first thing to do is start the program and enter the credentials of the account where you want to change the language in the window that appears.
  • After the login is completed, you need to click on the icon with the silhouette of a person, which is located next to the search bar in the upper right part.
  • In the drop-down menu, select Account info, after which a page with account information will open.
  • On this page we should be interested only in the first item - Change Country or Region.
  • After clicking on it, it will be possible to select the necessary country from the drop-down list.
  • When everything is done, it remains only to confirm the selection by clicking on the Change button and accept the user agreement.




As was said at the very beginning, there is nothing complicated here, and absolutely anyone can cope with such tasks. Good luck
