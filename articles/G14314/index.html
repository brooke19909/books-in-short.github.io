Import contacts on Android: options and instructions

After buying a new mobile gadget, it becomes necessary to transfer important information from the old device. And if the necessary applications can be reinstalled, then it is quite difficult to restore lost contacts. Especially if there are more than a hundred of them.

Of course, you can transfer contacts from Android to Android manually, but this process is rather laborious and unpleasant. It is much more effective to attract third-party services, utilities and other tools to this enterprise.

Let's try to figure out how to transfer contacts from Android to Android and make it as painless as possible for both the mobile gadget and the user himself. Consider a number of available and simple methods for implementing this procedure. So let's get started.

SIM card

One of the easiest, but at the same time time-consuming, ways to import contacts to Android is to copy the phone book to a SIM card. You should immediately warn that there is a limit on the number of subscribers - no more than 200 numbers.

To import contacts to Android, your old device must have a valid SIM card. If there are any software or hardware restrictions on working with the phone book, then they need to be removed. Permissions or restrictions are usually granted by third-party applications. If they are not installed, then you can proceed with the import procedure.

How to transfer contacts to a SIM card:

  1. Open the phone book and click on the option button (three-point icon / gear).
  2. The line ‚ÄúSettings‚Äù should appear in the drop-down list.
  3. Click on "Import / export contacts."
  4. After clicking on the ‚ÄúExport‚Äù button, select ‚ÄúIn SIM‚Äù.
  5. Wait for the process to complete.

Next, insert the SIM card into the new smartphone and perform the same actions. The only difference is that instead of export you need to select the import of contacts to the SIM card. We are waiting for the end of the procedure and check your phone book. All numbers should appear there. If there are more than 200 subscribers in your book, it is best to import contacts to Android using one of the methods described below.

SD card

For this procedure, we need any SD card. Even the medium with the smallest amount of memory will fit - 512 MB or 1 GB. Less are no longer being released. Immediately it is worth checking the performance of the card on both devices. If the external media felt great on the old phone, then it's not a fact that everything will be just as good on the new one.

How to transfer contacts from Android to Android using an SD card:

  1. We go into the phone book and open the options section.
  2. Go to the "Settings".
  3. Click on ‚ÄúImport / Export Contacts‚Äù.
  4. By clicking on the ‚ÄúExport‚Äù key, we select the SD card as the receiver.
  5. We are waiting for the end of the procedure.

On external media, all contacts will be saved in a specific VCF format. The file is located, as a rule, in the root directory of the SD card. If there are many subscribers in the phone book, several archives can be created (_002, _003, etc.).

To import contacts to Android, insert the SD card into a new phone and perform the same procedure, with the only difference being that we click on the Import icon instead of the Export button. In this case, there are no restrictions on the number of subscribers, as was the case with the SIM card.

Bluetooth

You can also transfer contacts using the bluetooth protocol. For this procedure, you will need to transfer all subscribers from the phone book to the memory of the old phone, if previously they were located only on a SIM card.

You must enable both mobile gadgets and establish a connection between them using the bluetooth protocol. If you encounter any problems, you should pay attention to the option ‚ÄúVisibility for other devices‚Äù in the settings of wireless networks.

Transfer contacts via bluetooth:

  1. Open the phone book and select "Options".
  2. In the drop-down menu, click on the item "Send Contact".
  3. Select all (or necessary) contacts for transmission.
  4. We enter the password (variably) and agree with the start of the procedure.

The process of transmitting data via the bluetooth protocol is quite lengthy, so do not be surprised if your phone freezes for a while. Especially when it comes to transferring a large number of contacts.

Google Account

Almost all firmware for Android gadgets are somehow tied to Google. The services of this search engine can also be used to import contacts to a new device. If there is no Google account, then you need to create one. The best way would be to register a mailbox. You can do this on the official website of the search engine.

Contact Transfer Process:

  1. Go to the default settings of the Android platform on the old gadget.
  2. Open the item "Accounts".
  3. We find the current Google account and click on it.
  4. In the synchronization window that appears, move the switch to the active position opposite the line ‚ÄúContacts‚Äù.
  5. Click on ‚ÄúOptions‚Äù and after - on ‚ÄúSynchronization‚Äù.
