World record - bench press. Powerlifting Champions

Bench press - the world record for this exercise today wants to set every powerlifter. And it should be noted, the goal of athletes can be understood. After all, the bench press is rightfully considered an unofficial indicator of strength in such a sport as bodybuilding, and in powerlifting itself. Therefore, many want to set a world record. Bench press is a very popular exercise. As a matter of fact, the first question that any athlete who visits the gym hears is: "How much do you squeeze from your chest?" It is not surprising that organized competitions for this exercise already exist today, and many athletes are preparing specifically for them.





It should be noted that today in the world there are a huge number of federations in such a sport as powerlifting. And almost in each of these federations you can compete in the bench press separately. So it makes no sense to describe every world record set in the federations. Many athletes perform the bench press with dignity. However, some records should be given special attention.
