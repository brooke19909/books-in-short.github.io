Wedding or Birthday Bottle Label

How to make a loved one an original gift? For example, present an unusual champagne for a birthday, the label on the bottle of which is made independently with a warm congratulation or a joke. This idea applies to other celebrations. How to implement it, we will consider further.

First of all, you need to come up with a layout for the future template: pick up drawings from the Internet, come up with beautiful words. Work can be done in Photoshop. For this procedure, no special skills are required; it is within the power of every person.

How to make birthday bottle labels



Consider the procedure using the example program Adobe PhotoshopCS3. When all the images for work are ready: flowers, background and other elements, we do the following.

1. Open Photoshop.

2. Download the pictures. To do this, click on the tab in the upper toolbar "File", then "Open" and select the desired image (background, photo, scenery).

3. Create a new document (future label). Click on the same tab ‚ÄúFile‚Äù, but this time select ‚ÄúNew‚Äù. In the window that appears, set the size to 12 x 8.2 cm, name: "The label on the bottle for Ivan's birthday."

4. We pass to the file on which the background is located (if we want to make it not monophonic). Using the tool in the left vertical panel "Rectangular Area" (dashed rectangle), select the desired area and click the tab on the top toolbar "Edit", then "Copy".

5. Return to the empty created sheet and click ‚ÄúEdit‚Äù - ‚ÄúPaste‚Äù. The background is ready.

If you want to create a solid color, you can select it using the Fill tool and click on a free section of a new file.





6. Now you need to process the photo of the birthday person. We pass to the corresponding file. The easiest way for this operation is to right-click on the Magic Wand tool, which is located on the left vertical toolbar, and select Quick Selection. A dotted circle will appear with a plus sign in the center. Press the left mouse button and select the part of the photo that you want to remove. By releasing the button, you can repeat the operation, continuing to mark the deleted area. If you accidentally touch the photo, the reverse procedure is activated by pressing the "Ctrl" key (in the lower left or right corner of the keyboard). A minus sign appears in the dotted circle. Holding down the key, reduce the area. Now just press the ‚ÄúDelete‚Äù key on the keyboard (or ‚ÄúEdit‚Äù - ‚ÄúClear‚Äù). You can edit the result with the Eraser tool.

By this principle, a bottle label is created with a photo of the birthday person. However, if you wish, you can insert anything you want instead of it: a bouquet of flowers, kittens, a funny picture, and more.

7. Now copy the cropped image, as written in step 4, and go to the file with the created background. Click "Edit" - "Paste". The size can be changed by clicking "Edit" - "Transformation" and selecting "Free Transformation". While holding the mouse pointer over the empty square around the photo, resize by pulling in the desired direction.



8. To add an inscription, we use the Text tool (letter T), with which we will write what we want. Highlighting the letters, you can choose any color from the palette. The bottle label for the birthday is ready!

9. If you want additional decorations (balloons, cake, a bouquet of flowers), then repeat steps 1 and 6.

10. Save the work using the combination ‚ÄúFile‚Äù - ‚ÄúSave As ...‚Äù. It is always better to do this in two formats: jpg and psd. The first one is for printing, and the second one, set by default, so that you can still edit.

Using the above instructions, you can issue other labels on the bottle. Wedding photos, for example, will help create celebratory champagne for the celebration of the newlyweds.
