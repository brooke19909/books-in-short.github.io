How to disable the "Tele2" menu on the phone?

What to do if there is no need for a specific function provided by the operator





So, how to disable "Tele2-Menu" on the phone? It should be noted that you cannot completely disable this service yourself. You can only deactivate all the applications you do not need. There are several ways to do this.

  1. Go to your personal account and uncheck all applications that you want to remove.
  2. Dial a three-digit number and turn off services. For example, to disable an option such as "games", you need to dial the number 468. All other numbers can be found in the directory of the subscriber.
  3. Use USSD requests. This is a combination of numbers, asterisks and lattices when typing. Having looked in the directory, you will find all the necessary commands. After dialing, you will receive commands on how to disable the "Tele2" menu.
