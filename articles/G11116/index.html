How to update iPhone firmware?

Apple manufacturers are constantly striving to bring improvements to the operating systems of iPhone and iPod devices. New functions are introduced, the battery life is increased, the speed of the devices is growing. These ideas are embodied in the form of periodically released new firmware. The main disadvantage is that updating the iPhone firmware using small separate files will not succeed, since releases are only released as whole new operating systems. OS size varies from 220 to 250 MB.

How to update iPhone firmware: it can be downloaded either from the links of specialized Internet resources, or via iTunes.

There are only four generations of the iPhone (this fall the fifth generation devices will go on sale), and, accordingly, separate firmwares have been released for each of them. If you have a 2G model, then it is impossible to install an OS suitable for 3G on it. Therefore, before updating the firmware on the iPhone, you must carefully check the name of the downloaded file.

For example, a file sounds like this:

iPhone2,1_4. 0_8A293_Restore.ipsw

How to decrypt it?

The first figures of 2.1 is the model of the phone. Then comes the firmware version - 4.0 and the build number - 8A293.

There are two ways to update iPhone firmware:

  1. full update;
  2. restoration (restore).

What is the difference between these methods? With a full update, all settings and data are saved as they were, and with a restore, all previous information is completely erased, and new information is written in its place.

And this is unlikely to have a positive impact on system performance. Therefore, it is still better before updating the iPhone firmware, create a backup in iTunes, then make a firmware restore, and then restore all your settings and information from the saved copy.
