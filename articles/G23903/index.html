If you want to activate Locator (Megafon) through a computer, you will have to use the official page of your mobile operator. Log in there, and then visit the Services section. "Locator" ("Megaphone") is activated through the computer.

Please note that the search can be carried out via SMS. It must be sent to 000888. The text should contain only the subscriber's number. In response, you will receive an email with a link to the map and a description of the user's location.

Navigation

And now a little about navigation. It‚Äôs very difficult to use the locator without additional work. You can only find your own phone. But not a stranger. For this you, as already mentioned, you need to get confirmation from a friend to implement the idea.



The service "Locator" ("Megaphone") allows you to add numbers to the search list. To do this, you need to dial a message on your mobile device and send it to number 000888. The text is as follows: + 792XXXXXXX, where XXXXXXX is the user's phone. After the request, your friend will receive an SMS notification with a verification code and instructions. He will have to send back a secret combination to the number 000888. And after that, the subscriber will be included in the list of approved.

You can also exclude a person at any time. In this case, further search on the map without confirmation will be impossible. Send a message to the number you already know with the text: + 792XXXXXXX. Here XXXXXX means the same thing as last time.

The list is completely deleted with the CLEAR or C command (it is important that the letters are uppercase), and can be viewed with LIST and L. That's how it is extremely easy and simple. As you can see, managing a locator from Megafon is not as difficult as it seems.

Cancellation of Services

True, sometimes there is a need to refuse the service. You can either simply not send new requests, having previously cleared the contact list, or refuse the offer altogether. How to disable "Latitude" on the "Megaphone"?

We confirm the actions (just say that you agree with the shutdown) and wait for the result. A notification about the successful completion of the process will come in about 5 minutes.

In addition, the "Personal Account" on the site of "Megaphone" is an ideal option. It is better to work with him through a computer. Go through authorization, select "Services". Next, find there "Locator" and press "Disable". A window for the verification code will appear on the screen. It remains only to enter it and click "OK."
