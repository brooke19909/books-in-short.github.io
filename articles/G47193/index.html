Accumulator battery

What is responsible for the performance of Nokia 1112? Battery! It is removable type. Located behind the rear panel. The user will not be able to get to it. The BL-5CA model has a capacity of 700 mAh. If you look at the instructions, you can see the battery life data declared by the manufacturer. These numbers are as follows:

  • Talk time is about 5 hours.
  • Standby time is approximately 15 days.

What did users get as a result of using the phone? Testing was carried out with devices connected to Moscow mobile operators. At an average load, the battery "sat down" after five days. In order to fully charge the battery, you will need no more than 1.5 hours.



The main menu items "Nokia 1112"

Owners of this phone separately note a simple menu. It consists of the main points:

  • Phonebook - saving caller numbers. The number of characters in a name is limited to 14 characters. You can enter only one number. It is possible to attach a picture to the subscriber. It will be displayed on a call.
  • Messages The user is given the opportunity to change the font - medium and large. There is a chat function. The phone memory stores up to 60 messages. Text templates and graphic pictures (emoticons) are provided.
  • Challenges This item combines missed, incoming and completed calls. Up to 30 numbers are stored in the memory.
  • Settings Here you can change the parameters of the Nokia 1112 phone, set by default by the developers. For example, a melody of a signal, network options and other.
