iTunes and Phonebook

How to save contacts from iPhone to computer via ITuns? This technique should be known to every owner of the "apple" phone. In the shortest possible time, a person will be able to copy contacts and save them to a PC.

The data transfer guide looks like this:

  1. Install the latest version of iTunes on your computer. This software is distributed absolutely free.
  2. Connect iPhone to PC. To do this, use a USB cable. One end is inserted into the computer, the other into the telephone.
  3. Launch "IT" and wait for the synchronization of devices.
  4. In the "Devices" menu, select the desired phone.
  5. In the left menu, go to the "Details" section.
  6. Check the box next to "Sync contacts with ...".
  7. Set Windows Contacts.
  8. Click on Finish.

It remains only to wait. After some time, information will be copied and stored on a computer. To transfer the contact book back, just synchronize the devices. Everything is extremely simple.

We figured out how to save contacts from iPhone to computer. But this is only one of the possible options. And after the operation, you need to think about where to look for the resulting file.

If the user acted through ITuns, then a special folder will be created in Windows. You can find it on drive C, in Users. Here you will need to go to the folder with the username, and then open the "Contacts".

iTools comes to the rescue

There is another pretty interesting trick. It helps to copy the phone book without difficulty. Faster than iTunes. It's about using a utility called iTools. Now you can find both Russian and English versions of the application. It is distributed completely free of charge, and working with it is a pleasure.

How to save contacts from iPhone? It is enough to follow these instructions:

  1. Install iTools on a PC. It is advisable that the latest version of the application is installed on the computer.
  2. Connect the USB cable to the iPhone.
  3. Insert the cord into the USB port on the computer.
  4. Run the utility.
  5. Open the "Information" section.
  6. In the left menu, select "Contacts".
  7. Select all the necessary phone numbers.
  8. Click on the "Export" button. It is located at the top of the program.
  9. Select a save format. For example, vCard.
  10. Click on "OK" and wait for the operation to complete.

The data will be copied to the iTools root folder. Here, in the "Contacts" section, the user can find the corresponding file. It can also be directly copied to email. For example, in Outlook or in Gmail.

Cloud services

And how to save contacts from iPhone to computer if the phone does not turn on? In this case, a proprietary cloud service comes to the rescue. The main thing is that it was initially turned on on the phone.

Apple offers all users a product called iCloud. With its help, you can back up information, as well as restore the OS. There is nothing difficult about this.



How to save contacts from iPhone to computer via iCloud? Required:

  1. Log in to your Apple ID profile on your phone. Usually this step is skipped, since all owners of "apple" phones are initially authorized.
  2. Connect your computer to the Internet.
  3. Open icloud.com in a browser.
  4. Log in to the service using AppleID. You must use the profile that is activated on the iPhone.
  5. Click on the "Contacts" button.
  6. In the lower left corner, click on the image of the gear.
  7. Select "Export vCard". In some browsers, this operation has to be repeated several times.
  8. Wait for the process to complete.

It is done. At this point, the user will be able to enjoy the copied phone book on the computer. But before you start this reception, you have to do something - to prepare the phone for surgery.

Sync settings

We found out how to transfer contacts from an iPhone to a computer if the iPhone does not work. For this, iCloud is suitable. For the operation to succeed, you must first set the synchronization settings.

This will require:

  1. Open the iPhone main menu.
  2. Go to "Settings" - iCloud.
  3. Scroll to the "Contacts" line.
  4. Press the switch. A green indicator should light up near it.

In this case, the phone book will be synchronized with the cloud service. If you neglect this operation, it will not work to realize the idea.

Mail services

The next trick works on almost all modern smartphones. It is about synchronizing the phone book with email services. For example, with Gmail.



In order to copy contacts from iPhone, you need:

  1. Go to "Settings" - "Mail, calendars ...".
  2. Select a line with Gmail.
  3. Enter the email address and password to enter the box.
  4. Mark the line "Contacts" with a tick.
  5. Save changes.
  6. Log in to Gmail on your computer. This can be done through any browser.
  7. Click "More" - "Contacts" on the left side of the mail task bar.
  8. Click on "Export".
  9. Set the format for copying vCard contacts.
  10. Mark desired phones.
  11. Click on the "Export" button.
  12. Wait for the operation to complete.

In order to transfer data to iPhone, you will have to select "Import" in the "Contacts" menu, and then specify the path to the file that we previously created. Everything is extremely simple and clear!
