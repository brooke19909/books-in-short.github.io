Alarm sensor

An important point worthy of special mention is the presence of a G-sensor in Cardinal G3. This is a module that allows you to accurately set the time of the accident. Due to the features of the work, the function is triggered by a sharp decrease in the speed of movement, rotation and increase in pressure with increasing speed.



The operation of the electronic accelerometer protects the records from deletion by placing them in the appropriate folder on the memory card. The direction of impact, type and time of the event are also recorded. Thus, the driver provides himself with all the necessary set of evidence, if any.
