Inkjet MFP Canon Pixma MG2540S: reviews, instructions

The most inexpensive and least functional peripheral solutions today include the Canon Pixma MG2540S MFP. Owner reviews about this economy-class device, its parameters, as well as recommendations for future use will be given in this article. An algorithm for connecting and configuring it will also be indicated.





Positioning. Technological features. Equipment

It should be noted that this device has very, very modest specifications. Therefore, it can only be used when implementing a small home printing system. Also theoretically it can be applied in a small office. In all other cases, the technical parameters of such an MFP will not be enough to implement a complete printing system. The multifunctional device under consideration is based on the inkjet method.

Its advantages include high print quality and minimal costs for various consumable components. There is only one drawback in this case - it is low speed. A key component of the delivery specification is the Canon Pixma MG2540S MFP.

A manual, a set of advertising booklets and a user manual is a list of the supplied documentation. Also on this list is a set of cartridges, ink, a CD and a power cord. The only negative of the delivery list is the lack of an interface wire. According to the manufacturer, he is already in stock with a potential buyer. If it is not, you need to purchase it additionally.

Printer Specifications

The Canon Pixma MG2540S MFP boasts quite good technical parameters for the budget segment. Reviews indicate support for photographic printing. But, again, it should be noted that such a conclusion can be made only on 10x18 or 10x15 media. Also in this case borderless printing is allowed.

The main format for such an MFP is A4. The highest resolution in printer mode is 4800x600. The print speed in black and white is 8 pages per minute, and in color - 4. The recommended media density should be from 60g / m 2 to 105g / m 2 . But it is also allowed to use photo paper with a nominal density of up to 275 mg / m 2 . The latter can be either matte or glossy.





Scanner Settings

Even more modest specifications are claimed for scanner mode. Only in 1200x600 resolution can you get digital copies on such a solution as the Canon Pixma MG2540S inkjet MFP. Reviews also emphasize its low speed, which is 4 A4 pages per minute. Like most similar devices, the type of sensor element in this case is CIS.

Specifications when used as a duplicator

Performance is further reduced for copier mode. In this case, no more than 1.5 A4 pages will be displayed in one minute. The resolution of the copy will be no more than 1200x600. But one certainly should not expect anything more from the most budgetary peripheral product.





Connection Methods

As noted earlier, the Canon Pixma MG2540S inkjet MFP is the most low-cost solution in the lineup of this manufacturer. As a result, the set of switching ports in this case will be minimized. It is represented by only one of them - USB. Of course, this wired interface is great for connecting various printing systems, but it was more convenient to equip such an MFP with a Wi-Fi transmitter. This would significantly increase the functionality of the device. But the manufacturer decided to save money and limited himself to only supporting one possible connection method.

Separately, it should be noted that the functions of the workflow station in this case are assigned to the control computer system. That is, you can use the MFP as a printer or scanner only when the computer is turned on. If this condition is not met, then the multifunction device under consideration turns into a multiplier. This is the only function that this solution can perform on its own.

Cartridges

Only two cartridges are equipped with a Canon Pixma MG2540S MFP. Owner reviews indicate a very modest resource of 180 pages. The black cartridge is labeled PG-445, and the color cartridge combination is labeled CL-446. The latter is filled with three types of ink - cyan, yellow and purple. They are included in the basic package of the solution in question. There are also improved versions of cartridges: PG-445XL and CL-446XL. In this case, the limit is increased to 400 sheets for monochromatic and 300 color. But in any case, these values ‚Äã‚Äãwill not be enough for the normal operation of such a printing subsystem. Therefore, most owners equip such MFPs with external CISS. This solution can significantly minimize the cost of printing a single page. But this requires additional capital investments of about 500-750 rubles.





Commutation. Setting procedure

The standard and quite familiar order of tuning boasts this MFP. This simple operation can be performed even by a low-skilled computer specialist. It consists of the following stages:

  1. Unpacking the box. Removing from it all components of the supply.

  2. Install a peripheral device next to a PC.

  3. Connecting the interface cord and power cable.

  4. Turn on the MFP. Installation of complete cartridges.

  5. Power on the PC. The end of the download process. Installing software, which includes drivers, control and diagnostic programs.

  6. Reboot a personal computer.

  7. Print a test page.





Cost

About 2300-2500 rubles must be laid out in order to purchase a Canon Pixma MG2540S MFP. At the same time, user reviews indicate that it is additionally necessary to purchase a USB interface computer wire, which costs about 100 rubles. But it will also be useful to equip MFPs with third-party CISS. The result is a full-fledged printing factory. That is, in order to maximize the possibilities of the solution in question, it is necessary to invest about 3000-3200 rubles.

Conclusion

Once again, it should be noted that the most budgetary solution in the lineup of printing devices of this manufacturer is the Canon Pixma MG2540S MFP. Reviews, too, on this of his features are constantly focusing attention. Therefore, something supernatural was not to be expected from him initially. For the rest, such a multifunctional device is perfect for home use. But it can also be used in the office. In other cases, it will be more correct to purchase more expensive printing systems with relevant specifications and parameters.
