What is an iPad and what is it famous for?

Modern technologies are designed to make human life easier and more convenient. With the help of modern mobile devices, we can keep in touch with our loved ones located on the other side of the world. A person is becoming more mobile every day, the number of tasks that he can perform in one day is growing. All this is facilitated by new devices, which include the iPad. This article will fully explain to the reader what an iPad is.



Ahead of all

iPad - a tablet computer developed by Apple, rightfully considered the discoverer of this genre. The device was first introduced to the world in 2010. Within a month after the presentation, over 1 million units were sold. Such a device allows the user to conveniently create and view high-quality photos or watch a movie while sitting in a cafe or at a party.

What is an iPad in terms of technical parameters?



This is a modern mobile device, which is a cross between a laptop and a smartphone. It weighs only 635 g, has a thickness of 13 mm and a list of features. The device is equipped with a Retina multi-touch display with a resolution of 1024x768, with which it works with the device. The text is entered using the virtual keyboard, but the iPad is also equipped with an additional hardware keyboard. iPad has the same functionality as the Macbook: you can type and edit texts, create drawings, work with tables and formulas. The relatively large - 9.7 inch - display gives ample opportunities to use the device. There are all the necessary slots for expanding memory, an input for USB (pairing with a computer, recharging).
A high-quality LED-backlit screen is ideal for watching movies from almost any angle, as well as for reading electronic books. Using the functions of 3G and WI-FI, the tablet has access to the World Wide Web, thereby making it possible to communicate on social networks, view a variety of content or simply search for information.
