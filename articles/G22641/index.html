"StarLine M21": installation instructions, diagram, reviews

StarLine car security systems occupy a leading position in the market due to its wide functional range and affordable cost. One of the most popular models is "StarLine M21" - a compact GSM-module with advanced functionality, with which you can remotely control a vehicle. The device can be used both independently and simultaneously with any car alarm. At the same time, you can control different points of the machine by connecting to the three built-in inputs of the device.

The vehicle is controlled by SMS messages, phone calls or special software on iOS or Android platforms installed on mobile gadgets. The battery of the car is significantly saved due to a thoughtful and efficient power system.





Equipment

Almost all StarLine modules have the same configuration and are easy to install. The configuration of the StarLine M21 module is as follows:

  • The system unit of the device.
  • Adapter for connecting to car alarms.
  • Wires to connect.
  • Resistor.
  • SIM card.
  • Installation instructions "StarLine M21".
  • LED sensor.
  • High sensitivity microphone.

Communication with the system is established after the installation of StarLine M21 and a SIM card using a mobile phone. The owner of the vehicle can choose the most convenient way of driving from the three proposed.

Design

GSM-module "StarLine M21" has a small size and is completely tight, which allows you to install it in hard-to-reach places of the car. The installation and operating instructions are supplied with the device, which describes in detail the process of installation, configuration and further use of the device.





Module features

"StarLine M21" works with any SIM-cards of all GSM-operators. In the module settings, you can specify four different numbers to which notifications will be sent, regardless of their location. For each number, you can configure individual message settings.

When synchronizing with a car alarm, the module instantly notifies the vehicle owner of any operation: opening doors, starting the engine, opening the trunk or hood or triggering shock or tilt sensors. Additional channels can be used to turn on when activating an alarm, enable and disable arming, listening to the passenger compartment.

When using StarLine M21 telematics as an independent security system, three main channels are used to control the car: hood, trunk, door, brake pedal or hand brake sensors are connected to them. The text and notification method can be programmed separately for each interface. The car owner is instantly notified in case of unauthorized operation of the controlled area.

Coordinate Definition

StarLine M21 tracks the location of a vehicle using cellular network stations. The accuracy in this case is 100-250 meters.

Control

The car owner can control the alarm and GPS module in three different ways:

  1. Application for mobile devices on Android, Windows and iOS /.
  2. Voice call to the module number "StarLine M21".
  3. By SMS message.

Alerts

You can add four phone numbers to the memory of the StarLine module, to which notifications about events occurring with the car will be sent. From each of these rooms you can manage the security complex. You can set an individual notification method for each of the numbers, as indicated in the instructions for StarLine M21. Additional channels can also be connected to the module.

Security functions

The StarLine module can be used as a separate signaling device, or in conjunction with other alarms. The "StarLine M21" circuit allows you to connect limit switches: the module has 3 separate inputs to which limit switches are connected. The car owner is notified by the module when any of the connected outputs is triggered.

When trying to hijack a vehicle, its owner can block the engine by SMS message or a call.

Engine start

As in any other automotive security system, the module provides 4 types of engine startup: by alarm, ambient temperature, remote and time intervals.

Pre-heater control

In their reviews of StarLine M21, the owners note that the module can be used as a means of controlling standard stand-alone heaters. They are connected to the device through a special channel for heaters.

Ergonomics

The compact size of the module and the integrated GSM antenna make it possible to install the main module unit in an inaccessible place in the car interior.





Heat resistance

The manufacturer creates StarLine modules taking into account the Russian climate, so the devices can operate in the temperature range from -45 to +85 degrees.

Modules "StarLine M21" and M31: similarities

Car alarms, coupled with Starline M21 and M31 security and telematics modules, turn into full-fledged multifunctional security and service systems. The modules can be used in conjunction with car alarms from the manufacturer "StarLine", as well as from other manufacturers. Systems can be controlled through such modules at any point in the presence of GSM network coverage.

Both models are characterized by the following distinctive features: determining vehicle coordinates using LBS technology and controlling systems using a mobile phone. Intuitive and convenient smartphone applications are offered by the manufacturer for free - you can download them from the official StarLine website.

Car alarm control through modules is carried out in the following ways:

  • Through applications for smartphones based on Android and iOS.
  • By phone call to the module number or by sending a text message.

"StarLine M21" and M31 read SIM cards of any operators. Up to four telephone numbers with individual settings for each of them can be entered into the module memory. When the security complex is triggered, the module sends detailed information about the cause of the alarm to the vehicle owner.

The modules can be used as independent security systems, since they have three inputs to which limit switches are connected. By sending a simple command via SMS or application, you can block the car engine when you try to steal it. The functionality of both models allows you to remotely control prestarting heaters and prohibit control of the system from all numbers, except for the number of the owner of the vehicle.





Differences between StarLine M21 and M31 modules

Unlike M21, the M31 module determines the coordinates of the car not only by LBS technology, but also by GPS. Actually, the presence of a GPS receiver is the main difference between one model and another. By sending an appropriate request, the owner of the vehicle can get the coordinates of the car and a link, clicking on which will allow you to see the location of the car on the map.

You can control and program the StarLine M21 module through 42 commands. In contrast, the M31 supports over 50 programming and control commands.

The advantage of StarLine M31 is the presence of a microphone, the sensitivity of which is controlled remotely from the owner‚Äôs mobile phone. The microphone allows you to listen to conversations in the car.

In addition, the M31 module notifies the owner about the car leaving the fixed control zone and starts the car engine without car alarms.
