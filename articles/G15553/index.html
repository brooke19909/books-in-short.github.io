What is this folder?

The folder with the intriguing name "Thumbnails" is a system folder. It is used by the operating system to store a cache of thumbnails of images and video files. This is to speed up the process of displaying them in applications such as Gallery. This program is known to many. Some other applications can also use this folder. If there are indecently many images on the device, then the cache size is also rather big. And he eats up a precious place.

What is useful to know for those who want to end this disgrace, to demolish the entire cache and everything that can be deleted? The Thumbnails folder is a system folder. Delete content using a PC

This is the first way you can delete the Thumbnails folder or its contents. In principle, it makes no sense to delete the directory itself, because the system will immediately create another one. But to demolish its contents is more effective. So, first you need to connect the gadget to the computer in media device mode. This will show not only the contents of the SD card (if any), but also the file system of your phone. However, it is not so easy to find the desired folder. In Windows Explorer, you need to enable the display of hidden files. Then she will be visible.





Now you need to open the folder, select all its contents and delete it in the usual way. If the system swears that "the file is read-only", then you need to go back one step, right-click on the folder, open the "Properties" item and carefully look at the file attributes. Uncheck "Read-only" and click "Apply to all files in the directory". Then repeat the removal procedure.

Uninstall via smartphone

Can I delete the Thumbnails folder using my phone? Can. But for this, you need a device that has gone through the rutting procedure, that is, you must have superuser rights. This can be done using the free Framaroot or Kingoroot apps. After obtaining superuser rights, you need to install the Root Explorer program. It is she who shows all the contents of the file system and allows you to change it. Without this conductor, nothing will come of it.





In the Root Explorer, you need to find the desired folder, select all its contents and click on the "Delete" button. The application will request superuser rights.
