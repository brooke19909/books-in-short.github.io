How to enable GPS on Android and configure it

Today, almost every device is able to replace the navigator. We are not talking about expensive models, where there are many different functions. Even the most budget smartphones install GPS. Enabling this feature is quite simple, probably even a novice user can handle it. But, as practice shows, even with such a seemingly elementary action, problems can arise. Let's see how to enable GPS on Android?





GPS enable

To turn on the navigation, you need to go to the menu and open the settings. In the new window you need to find the "location" section and open it. How to enable GPS on Android? To do this, click on the switch. Of course, it will be different in different versions, but you will have to choose an operating mode. What are they like?

  • High precision mode. If you select this item, then the device will use the built-in sensor and wireless networks to determine the location.
  • Saving charge. In this mode, the location will be determined only if there is an Internet connection. So you can save battery power.
  • The latter mode allows you to determine the location of the built-in GPS-module. If you are unable to use the Internet connection, then this is the only way to find the right place.

How to enable GPS on Android 5.1? In the new version, you can find the button to enable navigation in the "curtain". To enter the GPS settings, you need to click on this button and hold it a little. How to enable GPS on Android 4.4 and earlier? To do this, follow the above instructions.
