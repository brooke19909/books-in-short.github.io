How to install the indicator in MT4 - step by step instructions

If you are just starting to trade Forex, you have a lot to learn. One of the most important skills is the ability to set custom indicators on the MetaTrader trading platform. This program comes with a standard set of options, but often there is a need to add your own settings.

This allows you to expand the functions of Metatrader and adapt it to your needs. Before you start learning how to install the indicator in MT4, you should look at two different types of files that can work on this platform.

What it is?

There are two types of files associated with Metatrader indicators, so the settings may seem a little confusing for beginners.

The first type is .mq4, an important file that stores all programming. This is basically a text format containing source code that the programmer writes to the indicator.

You can see the source code in a text editor or in the MetaEditor program. As a rule, the volume indicator for MT4 is presented in this format.

The second type of file is .ex4. Unlike .mq4, it cannot be edited. Remember this, because if a developer or a free Internet resource provides you with such a file, you cannot change it - this provides copyright protection, which often includes candle indicators in MT4.

This is a compiled version of the indicator that MetaTrader creates from the.mq4 file. This format is readable by the user, but for this it must be converted to .ex4.

How to open MetaEditor

If you want to see the indicator encoding, you can open the .mq4 file and view its code. To open it, go to MetaTrader and click on the MetaEditor icon.

The program will open and you will find the indicator folder on the right side. When you open it, you will see all files saved in .mq4 format. Double-click on the indicator (levels for MT4, for example) that you want to edit, and you will see the source code. But if you do not understand this well, it is better not to touch the system settings.

Where to find new indicators?

The great thing about Metatrader is its widely used platform, for which you can find almost any indicator in the public domain.

However, there is a drawback. You cannot know for sure how well the indicator has been programmed. In no case can it be argued that these free applications are intentionally sabotaged, but the creators may have different programming skills. Therefore, check all indicators (including horizontal volumes for MT4) on a demo account before you trade for real money.

Where to look for indicators?

As noted above, there are many resources where you can find and download indicators for MT4. Below are the names of sites where you can find the right one. Of course, a regular Google search is always the best place to start searching for accurate MT4 indicators, but you can always use the ready-made offers.

ForexFactory. Since FF is one of the largest Forex forums, it is not surprising that they have a wide selection of indicators and lots of suggestions from users that can help you change and customize them.

FX Fisherman is another large Forex forum with a large repository of MT4 indicators (volume, levels and so on).

MQL4 - The creators of MetaTrader keep their own repository of indicators, which you can use for free.

Since all of these sources are reliable, you do not need to check each indicator. It is best to learn the trading method that works with it and test this method on a demo account.

How to get custom indicators

Another option is to get an indicator programmed for you.

Fortunately, much of what you can request has already been developed for free. For example, horizontal volume indicators for MT4 are always ready to use. Even if such settings are not available in the public domain, for little money you can get the functionality that meets your specific needs. Usually, minor improvements are required, such as adding a warning in the form of text messages or signals to the MetaTrader smartphone application.

Conclusion

Finally, even if you have all the possible indicators at your disposal, this does not mean that you should use all of them. The more indicators you add, the more confusing your chart becomes. If there are too many lines and scales, you will not be able to figure out the chart and make the correct forecasts.

Many newcomers believe that a larger number of indicators will give them a better idea of ‚Äã‚Äãthe market situation. The reality is that you should use them as little as possible. The more you install them, the higher the likelihood that they will contradict each other and simply confuse you.

Many of the trading systems recommended by specialists are based on two or three indicators. Sometimes you can even go through a net price action.
