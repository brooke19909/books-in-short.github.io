How to check Internet traffic on MTS: step-by-step instructions

For subscribers who do not have unlimited tariff plans, the question of how to check Internet traffic is always relevant. On MTS, this can be done in several ways, regardless of which device it is used on: on a smartphone, modem or tablet PC. It is recommended to regularly monitor the state of traffic: this will allow you to not be left without the Internet at the most inopportune moment and avoid automatically connecting options with additional traffic volume (this is especially true for tariffs of the Smart line). How we can control the amount of remaining megabytes on MTS on various devices, we will describe in this article.





How to check Internet traffic on MTS from mobile devices

You can view information about how many megabytes have been used up through the standard functionality of a mobile device. This feature is present in all modern gadgets. Let us consider each of them in more detail.





All ways

Checking traffic via SMS. In order to receive information on the remaining megabytes, it is enough to send a test request with a question mark to 5340. Sending a message is not charged. In response, you will receive a message with information about how much traffic the subscriber has on the number. You can check the data in this way an unlimited number of times.

Getting traffic information through a USSD request. Since the tariff plans may provide for a certain amount of megabytes, a number of packages are available for connection, there are several commands for receiving data on balances. We give them all.





  • You can check how much MTS Internet traffic remains on the Smart tariff plans (which is included in the monthly fee) through a request of the form: * 100 * 1 #;
  • after activation of additional packages (for the Smart line, after the limit on traffic included in the TP has been exhausted), you can check the remaining megabytes through the command: * 111 * 217 #;
  • for other options that are also connected separately to any tariffs of MTS (for example, ‚ÄúBit‚Äù, ‚ÄúInternet Maxi‚Äù, ‚ÄúSuperBit‚Äù, etc.), the request * 217 # is used.

Check on tablet

How to check Internet traffic on MTS via a tablet? Their owners have a bit more complicated, since not all models have the ability to enter USSD-requests. For them, only traffic monitoring via the Internet is available ("Personal Account" of the user). How this can be done will be described below.

If the ability to enter USSD commands is present, you should remember the combination of requests that may be useful for checking the remainder.

  • For tariff plans of the Smart line, you can check the traffic included in the TP by entering the combination * 100 * 1 # (the same request is used for smartphones).
  • For packages that were activated after the main volume of megabytes was exhausted according to the TP of the Smart line, the command * 111 * 217 # is available.
  • The balance of other packages that are connected to any tariff plan of MTS is verified through the request * 217 #.

Checking traffic through a modem

The principle of controlling the remaining megabytes according to the tariff plan or additional Internet options is similar to the previously considered options. The only difference is that instead of the screen of a phone or tablet, a laptop or computer monitor is used. A program is installed for MTS modems, which is installed on a computer and contains management tools. In particular, the USSD request input form for checking traffic. Since the MTS Connect TP is used more often for modems, you can check the remaining megabytes by entering the combination * 217 #. If the device model does not have an interface for entering commands, you must act through the "Personal Account".

Getting traffic information over the Internet

If the subscriber is inconvenient to use the options described above to view data on the remaining megabytes, you can use the "My Account".
