Visually, the NanoStation M2 is very different from what is offered on the market. This is a small white box with an elongated shape, designed specifically for installation on the street, next to cameras and on all kinds of poles. Key Features of NanoStation M2

The NanoStation M2 is a compact yet very powerful mobile transmitter with a 600 milliwatt MIMO antenna. The gadget is ideal for connecting to IP cameras and receiving information from them. One of the main advantages of the gadget is its support for Airmax technology, which prevents unnecessary changes when transmitting data over long distances. The NanoStation M2 has an intelligent prioritization system for transmitting data from cameras or audio.

You can configure the NanoStation M2 access point to organize a large-scale Wi-Fi network (for example, in a small city), to which any clients can connect. The same access points are used to establish a connection between several branches of the same company.

Due to the ability to endure harsh climatic conditions, NanoStation M2 has established itself as an ideal tool for organizing a reliable and uninterrupted network.

The same device can be used as a bridge that will transmit a signal at a remote distance (up to 7 kilometers). This will be effective for individuals who want to establish a single network between two apartments in the same city or a summer house and apartment. Also, in this way, NanoStation M2 is used by communication providers who can install a single workstation in the house and already lead wires from it to each individual apartment. The main signal, respectively, is transmitted using another mini-station NanoStation M2.





An important advantage of gadgets from Ubiquiti is the Air OS operating system, which helps to effectively manage multiple points of NanoStation. You can also adjust the frequency shift there so that owners of smartphones and computers with Wi-Fi support cannot catch the signal transmitted by your NanoStation bridge.

Ubiquiti NanoStation M2 Loco setup

If you want to use NanoStation as an add-on for your router, which will distribute the Internet to a large space according to the same rules and with the same security settings as your static router (or modem with a router function), then you need to configure NanoStation M2 ( router) in access point mode:

  • Before you start, you need to connect your main router to the computer using a cable (no Wi-Fi).
  • Then open the "Control Panel".
  • Go to the "Network Management Center" submenu.
  • Select the item ‚ÄúChange adapter settings‚Äù.
  • Open the properties of the network connection and change the IPv4 settings.
  • Enter 192.168.1.21 as the IP address, and 255.255.255.0 as the subnet mask.

Once you have finished configuring the connection, you need to configure the station itself:

  • To do this, open any browser and go to the address 192.168.1.20 (if you can‚Äôt log in, then someone has changed the address of the connection to the station).
  • You will see a dialog box where you need to enter your username and password (by default, this is ubnt).
  • Open the Wireless tab in the AirOS interface.
  • Change the Wireless Mode parameter to Access Point.
  • In the Country Code parameter, enter any country (the United States will allow you to overclock the connection speed).
  • In the SSID field, enter a name for the network (any).
  • We specify WPA2-AES as the type of protection and enter an eight-digit password.
  • To save the entered parameters, click on the Change button and reboot the access point along with the router.




Configuring the bridge in the NanoStation M2

Now let's talk about how to configure the NanoStation M2 in the mode of the subscriber point (bridge):

  • First, open a browser and go to the AirOS interface at 192.168.1.20.
  • Go to the Wireless submenu.
  • In the Wireless Mode item, in the drop-down menu, select the Station operating mode.
  • Then you need to look at the list of basic Wi-Fi stations and select the one to which you plan to connect your NanoStation (put a tick in front of the desired one and click Lock to AP).
  • Save the settings by clicking on the Change button.
  • This completes the basic setup process, and your bridge is ready to go.
