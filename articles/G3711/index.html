Nokia 3220: advantages and disadvantages

Nokia 3220 is an inexpensive phone that is equipped with all the necessary functions. In the line of devices of its class, it stands above its predecessor and was positioned as a telephone for young people.

The manufacturer improved the design, light indicators appeared.

Like other Nokia phones, the 3220 runs on the Series 40 platform.

The phone has been available since 2004.

Equipment

The kit includes the phone itself, a charger, 2 pads on the back panel, instructions.

Main characteristics:

  • weight - 86 g;
  • dimensions - 104.5 x 44.2 x 18.7 mm;
  • built-in antenna.

Working hours

3 hours while talking, 350 hours on standby - this is the maximum runtime of the Nokia 3220. The battery for the model is 760 mAh.

Charging time takes about an hour and a half.

Display

Color display, 65,536 colors, TFT screen with a resolution of 128 x 128.



The screen contains 5 text lines, a numeric keypad. Menus in the form of icons or a list, as in other Nokia phones.

Display size 27.5 x 27.5 mm.

Camera

A 0.3 megapixel VGA camera is provided, the maximum resolution when shooting is 640 x 480, in portrait mode - 80 x 96. There is a night shooting mode, 3 types of quality. Files are saved in JPEG format.



The video shoots in 3GP-format, each video lasting up to 15 seconds. Resolution 128 x 96, AMR sound. There is a setting for the duration of the shooting. With it, you can shoot up to 4 minutes. In the amount of memory is 1.5 MB.

There is an HSCSD.

games and applications

Java 2.0 version allows you to play. But this can only be done with games preinstalled on the phone.

Sound

The phone plays polyphonic ringtones of 16 tones.

Phone book

The maximum number of entries in the phone memory is 1000 names. If you fill out 6-7 fields for each number, you can save 500 numbers. For one name it is possible to enter up to 5 numbers (mobile, home, main, office, fax).

Each contact can be added an email address, website, postal address, note.

The numbers in the Nokia 3220 phonebook can be associated with voice tags (up to 10).

There is a standard speed dial feature.

Additionally, it is possible to set photos for 100 contacts (portrait mode).

All contacts can be divided into 5 groups, choose a ringtone for them.

Lists of calls are standard: missed, received, outgoing. Each of them saves 20 records with date and time.

Messages

Your phone can receive audio and black and white pictures using Nokia Smart Messaging. But only those devices can send and receive files that support this standard.

For such data transfer, the device has 10 graphic templates.



Color images, photos, melodies, text can be sent using MMS.

It is worth mentioning the ability of the phone to send a Flash message. This function sends a message, but it is not stored in the memory of the device to which it has arrived. It is only displayed on the screen.

Additional features

The Nokia 3220 has the ability to synchronize with a PC.

There is a built-in image editor, a key lock.

There are standard functions: clock, date, calculator.

You can set the alarm to repeat by selecting the day and ringtone.

To count the time there is a stopwatch and timer.

An organizer is available with the ability to make from 100 to 250 entries. Their number depends on the length of the text. There is a setting to automatically delete old records.

The function of viewing the calendar for different time periods, a quick transition to the desired date are also present.

In the settings you can find 5 user profiles. Each of them can be set temporarily by selecting the desired interval. After completion, the default profile is activated.

For the screen, you can choose the color, screen saver and wallpaper.

The speaker volume is automatically adjusted, it works depending on external conditions.

The recorder can record 5 minute clips. Their number is unlimited, depending on the availability of memory in the phone. During the conversation, the recorder also works. Voice recorders can be used as a call (AMR format).

Case, colors

Orange, black, black with blue accents, cyan, red - these are all available colors of the Nokia 3220 model. The case is a monoblock. The tube material is soft to the touch.

The keyboard is made of rubber, with the exception of the central button, which is made of hard plastic.

On top there is a plastic on / off button.

Bottom 2 connectors: standard pop-port and charger connector.



There are interchangeable panels. The back cover is transparent, any paper insert can be cut under it. Side inserts can also be changed. There are only 2 on each side. They are made of elastic plastic, which protects the phone from shock. Also, these inserts are highlighted, for example, with an incoming call.

LEDs are a good addition to the design. The manufacturer offers to edit lighting effects using a special program.

conclusions

The advantages of the phone are obvious. All available functions work at the proper level, no complaints.

Of the shortcomings, it can be noted that the phone does not have an FM radio and an infrared port. Data can only be transferred via the cable that needs to be purchased. Also, the device does not support MP3 files. These are the most significant shortcomings of the Nokia 3220 model. An overview of all the technical features allows us to conclude that the phone is justifiably called youth. The unusual design and the unique ability to use the backlight corresponds to this direction. But at the same time, the device is equipped with only basic functions.
