Rostelecom Internet settings: recommendations for users

In order for Internet access to appear, it is necessary that all devices, whether a computer or a home router, are configured correctly. In this article, we will examine in detail what settings are used by such a large provider as Rostelecom.

What you need to do first

In order to make it clear which Internet settings from Rostelecom to install, you need to know the following details, which can be clarified in the technical support of the provider or see the contract that was concluded when connecting.

Depending on the region, the following connection technologies may be used:

  • ADSL (copper telephone line connection);
  • GPon (the use of optics, which is carried out in the apartment to the subscriber);
  • FTTx (connection via Ethernet cable);
  • Docsis (using coaxial cable).

Depending on the technology, various network devices are used, and the connection scheme may differ.

The type of connection shows how the logical part of the connection is made (it is specified when configuring the equipment). The necessary information can be obtained directly from the technical support of the provider.

Additional data - IP address, subnet mask and gateway, or login and password to establish a connection - are usually prescribed directly in the contract.

Wiring diagram

Let's consider the main options of the connection scheme for setting up the Internet from Rostelecom.

  1. ADSL. The telephone cable from the telephone wall jack is connected to the splitter (divider), then to the ADSL modem. From it, the network cable through the LAN connector goes to the computer, or, if wireless devices are used, the connection is via Wi-Fi.
  2. GPon The optical cable is connected to the terminal, and then in the same way: either twisted pair cable to a computer or router is used, or a wireless network.
  3. FTTx (Ethernet). The network cable immediately connects to a computer or router.
  4. Docsis A coaxial cable goes to the modem, then a router or computer is connected.

Make sure that all cables connected to the devices are not damaged. The corresponding indicators (ADSL and LAN1, or PON and LAN1, or, respectively, INTERNET or WAN and the corresponding LAN indicator) should light up.

What to configure on a computer

Ask your provider for the type of connection. Most often when using ADSL, Ethernet and Docsis technologies, PPPoE (high-speed connection) is used.

Setting up Rostelecom Internet on a computer for this type is as follows:

  • Using the control panel, go to the "Network and Sharing Center" section, then click on the "Configure a new connection or network" link and follow the step-by-step prompts.
  • Now you need to mark "Internet connection" (the topmost item), and then go on to setting up a high-speed connection.
  • Enter the login and password received from the provider. Then complete the setup.

Also, the type of connection can be a static or dynamic IP. Then you need to turn to the network connection, which is located in the "Change adapter settings" menu, in exactly the same way through the network and sharing control center and find your connection. How to configure a router

Before configuring the Rostelecom router for the Internet, make sure that it is connected correctly and the POWER, WAN / ADSL and LAN indicators are on.

Setting up the Internet from Rostelecom is done similarly depending on the type of connection. If the type of connection is PPPoE, select it in the corresponding paragraph, below in the field you will need to enter your login and password.

For dynamic and static IP settings will also be similar to those used on the computer: either the data specified in the desired item, or the receipt of data automatically. After that, save the settings and reboot the router. Done!
