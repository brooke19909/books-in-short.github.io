4Good tablet: description and equipment

The 4Good T700i 7 3G tablet is an exclusive line assembled in China by 4Good by order of MVideo. The main and most important advantage is its price (from 1990 rubles), which is much lower compared to analogues of top brands.

The device measuring 188/108/10 mm (length / width / height) and a display diagonal of 178 mm (7 inches) is painted black, has ports for 2 mini-SIM cards in mini format and supports 3G networks. The kit includes a USB cable, charger, instructions for use and adaptation, headphones. The body of the gadget consists of high-strength plastic.



Hardware Base and Performance

The tablet runs on the Android 4.4 operating system. Together with Intel's dual-core processor (Atom x3-C3130 model), which delivers power up to 1 GGz, it provides high-speed performance in almost any operation. 512 MB of RAM also contribute to fast data processing. The internal volume is 4 GB, but thanks to the present microSD card slot (microSDHC), the disk space can be increased up to 32 GB.

However, in practice, when working with a memory card, problems may arise: applications are not transferred to the external, but are saved only on the emulated, which is built into the 4Good T700i 3G. Owner reviews indicate that, due to this shortcoming, there is simply not enough volume for full-fledged work.

Connections and Internet access

This device supports Internet access via SIM, and also has a built-in Wi-Fi network signal reception function, which provides data reception / transmission up to 300 Mbit / sec. To further enhance the capabilities of exchanging data, a Bluetooth device running on v4.0 is built into the tablet. Connection to a PC is made using the microUSB port with a cable included in the basic kit. It is worth noting that to connect the tablet to the computer, additional software is not required (all necessary components are built into the device).
