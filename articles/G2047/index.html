What is MMS? Decoding abbreviations

In the modern world, technologies such as SMS and MMS have gained great popularity. It is worthwhile to understand in more detail what is the MMS service in the telephone set, what is its role. What is MMS, you will learn when you read this material.

Familiarization with the MMS message function

What does this abbreviation mean? How to use such a service? MMS - what is it? Decryption explains that this is a multimedia message (Multimedia Messaging Service). The transmission of these messages takes place on a cellular network. Their feature is that they are sent not only to a mobile phone, but also to an e-mail mailbox. This option differs from SMS in that it does not have a restrictive framework in terms of the type and size of the transmitted information. Using these messages, you can send various multimedia files (music, voice messages, videos, various photos, etc.).

But how is SMS decrypted?

In this article, we will consider not only the MMS service (the abbreviation is decrypted above), but also SMS (SMS), which means the short message service. This technology allows you to send as well as receive small text messages to your mobile device.





Functional properties

The features of this type of messaging are more extensive than we think. The capabilities of these messages can be divided into several types of groups, which literally explain to us what MMS is:

  • Message exchange. Messages that do not fit in one or more SMS can be compactly formatted in MMS. For example, I want to send a birthday congratulation to a friend, but it contains a large amount of information that does not fit into SMS. You can simply send a voice message with congratulations - it will be even more realistic and pleasant.
  • Information services, that is, extensive opportunities for processing messages.
  • Business Services. This technology expands business opportunities. For example, a user can receive exchange information, their graphs and stability histograms for a selected period of time on their phone.
  • Entertaining side. British scientists have found that more than 70% of MMS in England contain information of a sports or romantic nature. In addition, the telecom operator has the opportunity to offer its users other interesting services, such as chat, games or dating and much more.




Using messages

To learn what MMS is and how to use it, the settings for this function of sending / sending messages will help us. Where to get these settings? To get them, you need to contact a special support service for your mobile communication and notify the specialist that they will send the necessary settings. After receiving, you need to save these settings. If the mobile phone does not support this function, then they will send a link that will be displayed with a normal message.

Transmission and processing of message data is controlled by the switching center. This center also has connections with other mobile operators.

If you send MMS from our phone to e-mail, then it will come to the specified mailing address in the form of a regular message. There are also devices with which you can send a letter, and it will arrive on your own mobile device.

MMS box

If you often use this service, then later the question may arise about where to store personal messages. The option with telephone memory immediately disappears.





This problem is solved as follows: the mobile operator creates a special information database, which is called the "multimedia box". This database allows the user to store private messages for some time. In addition to all this, the user has the ability to send and generate messages without previously downloading information to a mobile phone. To do this, the subscriber sends a message to the specialized service center that contains information about the recipient and investments, then the support center automatically makes a pickup from the user's multimedia box. And also the subscriber has the opportunity to access his messages through the World Wide Web.

Special Message Album

To fill the messages with their own information, the operator forms a special base, which is called the "multimedia album." This information base is located on a special server and stores video / audio files, photos that the user can freely download to write a message.

Is there a maximum size for an MMS message?

The size of this type of message has no limit. This was done so that there were no problems encountered in SMS with a limit of 160 text characters. In order to clarify what MMS is, it is necessary to indicate that its value is generally dependent on communication technologies, while it also depends on the mobile operator, which is entitled to indicate the standard message size for calculating its cost.





Can I send MMS to my phone without the support of the MMS message function?

MMS transmission to such types of mobile phones is supported by the TGW (Terminal Gateway). This system calculates the type of mobile device that receives a message, and without sending a message, saves it to a web page. Then an SMS message is sent to the mobile phone, in which a link to the page of the Internet resource is indicated.

What changes are made to the network for MMS?

There is a chance that the telephone that receives the message will be disconnected or not in range of the network. Therefore, to store message data until the time of receipt, it is advisable to refer to the latest network element - MMSC. MMSC also has such capabilities as connecting to networks and application management functions, ensuring the operability of various other services.





Is it possible to send MMS to a phone with a monochrome display?

The main type is a color image. On the basis of this, a multi-colored telephone system is required for full performance. But there are several approaches for reproducing color images on black and white phone models. If a mobile device with a black and white system supports MMS messages, then, from the point of view of theory, it has the ability to receive multimedia, and images in different colors can be viewed on a black and white display.
