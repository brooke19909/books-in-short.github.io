How headphones work without wires

So is it worth buying wireless headphones at all? We will analyze the principle of operation (a description of the main characteristics of the signal). The fact is that when using the usual Bluetooth 4.2 data transfer protocol, the data stream arriving at the transmitter is compressed in the most merciless manner. This is done in order to increase the transmission speed. But with compression, data quality deteriorates very much. So, if you try to listen on a FLAC smartphone with 24 bits of depth and 196,000 Hertz frequency, then in headphones you will still hear some strange MP3 with a bitrate of 128 kilobits per second. And therefore, for the widespread use of wireless technology in the transfer of sound is still too early. There will still be no quality.
