View data by number over the Internet

How to find out the rest of the traffic on the MTS Smart from the phone?

The most convenient and quick way to view information by number is to enter a USSD request. Using a special command, information on the balances of not only megabytes within the tariff plan, but also minutes and test messages will be displayed on the device‚Äôs screen. To get such data, just enter the request - * 100 * 1 #. If the main limit has been exhausted and the additional Internet package has been activated automatically, this command will not work. To view traffic information in this case, the following query is useful: * 217 #. By the way, when activating other packages that imply included Internet traffic, the same combination is used.

Contacting Technical Support Specialists
