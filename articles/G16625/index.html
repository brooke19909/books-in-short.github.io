Base types

Fluorescent fluorescent lamps, unlike incandescent devices, should not be directly connected to the electrical network. For this, ballasts are used. They regulate the supply of electricity. Devices are divided into two types: with an external device, as well as built-in.

We are talking about ballast equipment of a mechanical or electronic type. Ballasts can be built in both the cartridge and the lamp itself separately. If we take devices that have external characteristics, then there are two types: with two and four socles. If we are talking about the latter, then such devices are connected using either a special device or a throttle. The pin base can only be connected using the last element.

Often, lamps of this type are used for chandeliers and lamps. There are also models that have plinths with integrated electronic ballasts. The base is created with a thread that does not exceed a certain diameter (standard or small).

Advantages and disadvantages

Luminescence fluorescent white light, like any other, have their advantages. For example, they received a huge level of efficiency, long service life, excellent light output, low temperature of a glass bulb, when compared with an incandescent lamp. Several color shades are also sold.

If we talk about the shortcomings, it should be noted that such devices are too expensive. If they are broken or deformed, there is a danger that a chemical infection will occur. If the load on the network changes, the lamp will flicker. The device will not work at temperatures below zero, so it should be said that such devices are demanding on this indicator.

If we talk about load differences in the electrical network, then with regular surges this will reduce the life of the unit. It should be noted that lamps cannot be used with a dimmer. Unfortunately, this will only cause damage. It is best to use conventional switches, which will save a long life, but this is only if the person will properly operate the device.
