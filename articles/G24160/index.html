What to do if the sound is lost in the headphones?

Practical every modern person is a user of a personal computer. PCs, laptops, tablets and mobile phones have become an integral part of life in our time. Computers and laptops can be used for both business and leisure purposes: watch a video, a movie, listen to music or play games. But what to do if there is any malfunction in the PC, for example, the sound is lost in the headphones or speakers? You can find solutions to this problem in this article.





Possible reasons for the lack of sound

There may be no sound on your computer or laptop for several reasons. We list them:

  1. Sound settings are down.
  2. Headphones or speakers are broken.
  3. Windows Audio service is not enabled.
  4. Disconnected device in the Manager.
  5. Incorrect BIOS settings.
  6. Conflicting programs or viruses.
  7. Crashing sound card.

Check sound settings

First of all, you need to check whether the sound on the computer is turned on. This can be done by left-clicking (LMB) on the sound icon in the lower right corner of the screen. The volume slider should be in the middle of the band or higher. You also need to check if the sound is muted at all. If the icon is crossed out in red, then click on it.





If the volume slider was already at a high level, but there is still no sound, then right-click (PVC) on the sound icon in the lower right corner of the screen and select ‚ÄúSounds‚Äù, and then ‚ÄúPlay‚Äù.





There should be a green check mark next to the speaker or headphone icon indicating the connection of the device to the computer. If a red cross or a gray down arrow flaunts in its place, then you should click RMB on the device and select "Enable".

Check the device for functionality

What should I do if the sound is lost in the headphones, but is it in the speakers?

In this case, you can disconnect the headphones and connect them to another computer, laptop, tablet or mobile phone with a suitable connector. Next, you need to turn on any sounds or music, thereby checking the headphones for performance.

If the device is operational, it means the matter is in the computer or its settings. If there is no sound when connected to another device, then you should go and buy new headphones.

Make sure Windows Audio is turned on

It also happens that for reasons unknown to the user, Windows Audio settings get lost. To understand why the sound was lost in the headphones, you need to check the correct operation of this service.

First of all, you need to call the context menu for working with commands using the Win + R key combination. In the window that appears, register the services.msc command.

Next, in the opened "Services" window, you need to find Windows Audio.





The line "Status" should be set to "Running", and "Default startup type " should be "Automatic". If this is not the case, then you need to click RMB on this service and select the named properties.

Next, as shown in the photo proposed in the article, you should select the following sequence: "Automatically" - "Run" - "Apply" - "OK".





Device disconnected

Another reason that the sound is lost in the headphones on a laptop or computer is the disconnection of the device. In order to verify this, you need to perform the following steps:

  1. Go to the "Start".
  2. In the search bar, write "Device Manager" and click on it.
  3. In the list, find "Sound, video and gaming devices."

If any of the devices in this section is indicated by a gray arrow (as shown in the figure below), then you need to click on it with RMB and select "Enable".





If this does not help, then you may need to update the sound driver.

Incorrect BIOS settings

If the user has recently performed any manipulations with the BIOS, then perhaps the settings were accidentally lost or they were set incorrectly. You need to check it out.

In order to enter the BIOS, after turning on the computer, you must repeatedly press the Del, F2 or F10 key. Which of these keys starts the BIOS of a particular operating system is indicated immediately when you turn on the computer opposite the word Setup. You must press this key immediately after turning on the computer, but before the operating system starts.

After you managed to enter the BIOS, you need to open the Advanced tab and find the High Definition Audio line. Opposite should be set to Enabled. If this is not the case and the Disabled status is set, then it must be changed to the value indicated above. Next, you need to save the changes and exit the BIOS.

These actions in most cases solve the problem with the lack of sound on the computer or laptop. After restarting and turning on the computer, you should check if there is any sound.

System Restore

If the user knows exactly after which and when the sound on the computer disappeared, then you can try to fix the problem using system recovery:

  • In order to roll back the system to a specific recovery point, you must open the "Start" and in the search bar write "System Restore".
  • In the window that opens, read the warning that all recently installed programs and other documents will be deleted after system recovery, and click "Next."




  • Find and select a recovery point, after which the sound may have disappeared in the headphones or in the speakers. If such a point is not in the proposed list, you need to set the checkbox opposite "Show other recovery points", select from the additional list and click "Next".
  • After clicking on the ‚ÄúFinish‚Äù button, system recovery to the selected point in time will start.

It is important to remember that it is no longer possible to cancel this process after its launch. If the computer has important files and documents recently saved, then they should be moved to a flash drive or some other memory card.

After the system rolls back, the computer will restart. If the problem was in the recent installation of programs, then a sound will appear.

Virus programs

Perhaps the lack of sound on a computer or laptop is due to viruses. A thorough malware scan should be carried out and eliminated if it is detected.

Sound card malfunction

If after using all the previous tips the question - the sound has disappeared in the headphones, what to do - is still relevant, then it remains to use the last recommendation: replace the sound card.

This is an extreme method, which should be used only after all of the above tips have not solved the problem with sound.
