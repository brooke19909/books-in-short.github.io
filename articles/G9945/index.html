How to remove the pattern from the "Android"?

The screen lock is used to protect the device from strangers. In the latest devices, among other things, it allows the use of a fingerprint, in all others it is usually used a digital password or a graphic key. However, sometimes the pattern of the graphic key is forgotten. As a result, access to the phone is impossible. In this article you will find information on how to change the pattern in several ways.

How to change the lock if you remember the combination?

If it is possible to unlock the mobile device, then removing the graphic key in the settings is not difficult. To do this, follow the step by step instructions:

  • open the Settings tab on your tablet or phone;
  • select the "Security" section and go into it;
  • open the ‚ÄúScreen lock‚Äù item;
  • in the window that appears, select "No" or "Scroll".

After these steps, the mobile device is easy to unlock even to an unauthorized user. If necessary, you can set another graphic key or a digital password in the same menu. After changing the password, it is recommended to write it down somewhere in order to further eliminate problems with blocking the device.

Making a call

Sometimes it happens that the user forgot the pattern of the graphic key, especially if it is installed recently. What to do in this case? You can unlock the Android-based device by calling it. On older models, during an incoming call, it is possible to enter the menu without using a graphic key.

So, during the call we perform the actions:

  • You must press the Home key.
  • Then proceed to the device settings and, according to the previous scheme, change or remove the graphic key.

Low battery

You can try to remove the lock, taking advantage of the vulnerability of the device through a signal informing of a low battery. The process takes time, but sometimes there may not be another way.

How to do it:

  • When the display shows a low battery message, it becomes possible to open the Settings tab.
  • In the window that appears, open the "Security" section and disable the graphic key.

We must act very quickly. After all, before the device is completely turned off, there is not much time.

If there is a Google registration

To remove the graphic key, if you forgot it, and unlock the gadget, you can use the registration in Google. To do this, follow the step by step instructions:

  1. You must make at least five attempts to enter the pattern. Since they are all entered incorrectly, the device remains locked. At the same time, the display will prompt you to try again after a while.
  2. If there is a Home button on the mobile device, after incorrect attempts to enter a graphic key, this key is displayed. If it is not, then you need to try to enter the key several times. It is worth noting that this method only works if you have an Internet connection, in other cases you need to use other unlock options.
  3. The principle of solving the problem is that by clicking the "Home" button, the user gets the opportunity to use his Google account and change the picture. To log in, you need to enter the email login information and password, and then click OK. If the data was entered correctly, the user will be logged in and will be able to change the graphic key to a new one.

This option is suitable only if there is a registration in Google, the username and password are not forgotten, and an Internet connection is established.

File manager

During the installation of the graphic key, a separate file is created in the device system that stores information about the lock. If you delete it, the password will be canceled and access to use the device will be open. To do this, you need a special program, for example, File Manager, and your smartphone must also have CWM or Recovery. In order to remove the graphic key, you must follow the following step-by-step instructions:

  1. Download the file manager and leave it unzipped. It must remain in ZIP format.
  2. Using a computer, a program is downloaded to the device‚Äôs memory card. You can also use a card reader or other phone.
  3. The SD card is inserted into the device you want to unlock.
  4. Pressing the volume and power buttons at the same time reboots Recovery.
  5. Then select the item ‚ÄúInstall ZIP from SD card‚Äù with the archive.
  6. After that, you need to confirm the action, and the file manager will start.
  7. Then you need to open the DATA / SYSTEM folder and find the gesture.key file. or password.key and delete.

Thus, the password will be deleted, and the phone will be unlocked. All that remains is to change it and this time remember or write down the lock combination.

ADB program

If the previous method could not be implemented for any reason, you can apply an alternative method of a similar principle of action and remove the graphic key from the "Android".

To do this, it is recommended to use the computer program ADB. It is designed for a variety of operations, including deleting the gesture.key file. In order to use it on the phone, the USB function must be activated. Also from the Internet you need to download the corresponding script to perform all the actions. To unlock the device, you need to follow the instructions:

  1. Install the ADB program on the computer and then save it to drive C.
  2. Through the USB cable, the device is connected to the computer.
  3. Then run the downloaded script, which will open the command line.
  4. The file deletion is confirmed, and after the offer to restart the device, click "allow".
  5. At the moment when the reboot begins, the smartphone is disconnected from the USB cable.
  6. After rebooting, you will need to enter a new password, then go to settings and delete the pattern.

Thus, you can open the smartphone without using a password. It should be clarified that there are no programs that can pick up a graphic key, so only such options are used.

We use third-party SMS Bypass applications

You can remove a forgotten graphic key from the "Android" using a third-party application. To use this method of unlocking a smartphone, you must have administrative rights and the ability to install such a program.

The principle is simple: when the display is locked, a special message must be sent to the smartphone. SMS at the same time serves as a code that allows you to deactivate the pattern. To implement this method, you must follow the instructions:

  1. Access your account using root privileges.
  2. Go to the Play Market and enter in the search engine SMS Bypass, then install the utility in Android. It is worth noting that the program is paid.
  3. Open the application and go to settings, change the password and provide administrative rights.

The application is ready to go. How does it work? Messages with the text XXXXreset are sent to the blocked device (x in this case, the password for resetting the key). After a message arrives on the device, an automatic reboot will begin. You can then reinstall the pattern or use a different locking method.

It is worth noting that the application can be installed on a locked smartphone, but only through root-rights.

If nothing helps

If after applying all kinds of methods it was not possible to unlock the device, the last option remains - to reset to factory settings. The main drawback is the loss of all data: contacts, photos, music and other files that cannot be restored if there is no backup.

To perform a reset, you must perform several steps:

  1. Turn off the smartphone.
  2. By simultaneously pressing the volume and power buttons, go to the engineering menu.
  3. Find and select the Clear eMMC line or wipe data / factory reset.
  4. After the removal is complete, select the Reboot System line.

After completing the described steps, the mobile device will return to the original settings, and the graphic key will disappear. After a reboot, the device will be completely clean, like new.

If you have problems remembering the pattern, you may need to choose a different locking method.
