If you do not use Bluetooth, then the smart device can work up to two days. If you activate this function, the time is reduced by half. There is also an active mode I'm Watch. Important information

Most of the smartwatch features announced by the manufacturer require an Internet connection via Bluetooth. Therefore, before purchasing a device, you should make sure that your phone easily meets the specified criteria. It will not be superfluous to check the support of the described function with the mobile operator whose services the buyer uses: not every company has an Internet connection via Bluetooth. Also, not every version of the operating system built into a smartphone provides this mode. Do not forget that even though I'm Watch is a smart gadget, it will not work without connecting to a smartphone. The cost of the Italian device is different.
