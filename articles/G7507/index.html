"MegaFon Mail" light version - what is it? How to activate and deactivate the service

Not all users know about the ability to use a mobile phone number as a mailing address. Moreover, at the moment there are a lot of different services that provide such an opportunity. However, some subscribers are aware that the MegaFon Mail service exists (light version and full version) and use it quite successfully.





Service Description

The essence of the service is that the subscriber number becomes the address of the mailbox, which has the following format 7YYY-YYY-YY-YY@megafon.mobi. On this box you can receive all electronic correspondence in the normal mode. MegaFon Mail, the light version (what it is and how it differs from the full version, we will consider later) has several advantages. One of which is the lack of tariffing - information about incoming letters comes in the form of a text message.

The advantages of the light version of mail from MegaFon

There are a lot of them, and everything has been done so that it is convenient for subscribers to use the service:

  • Simple and well-remembered email address.
  • No limit on the storage period for accumulated letters.
  • An optimized interface for mobile devices that makes it easy to view, select and create new messages.
  • The absence of any tariffing (the use of the box is not subject to any payments) - this property applies only to the option of "easy" mail, which has some restrictions.
  • The presence of notifications of incoming correspondence with the ability to configure receipt of notification. For example, the period: the subscriber can set the desired time when the MegaFon Mail service alerts will be received (light version).

To activate the service, you need to dial the following request on the number * 656 * 2 #. In the response message you will find information about the personal login and password that will be used to enter the mailbox. Follow the instructions from Megafon.
