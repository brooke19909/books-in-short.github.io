Hat with cat ears, knitting: how to knit a baby hat, schemes

A headdress is a necessary element of the image of every girl. The needlewomen know that it is easier to tie it with your own hands. Today, simple, cute hats with cat ears are especially popular. It is very simple to knit such a product. The main thing is desire and minimal knitting skills.

There are several models of children's hats with ears that look attractive on girls.





Where to begin?

You need to pick up yarn, knitting needles (you can stocking 5 pieces or circular), a hook, a needle with a wide eye, markers or pins, scissors. Consider knitting hats with cat ears with knitting needles as an example.

Everyone knows that the main purpose of the cap is to protect against cold and wind. Therefore, preference should be given to yarn, which includes wool. The example uses bobbin yarn (30% wool, 70% acrylic). You can choose any other. The main thing is to tie a sample that will help you calculate the required number of loops for the cap, given the individual density of knitting. How to tie a hat with ears for a girl or girl?

Loop calculation

Remember that the pattern should be knitted according to the pattern of the hat for the girl. In the example, 18 loops per 10 cm of sample. So, in 1 cm = 1.8 loops. We measure the height of the bound sample and calculate that in 1 cm 3 rows.

Often needlewomen knit a hat with ears of a cat with a lapel. It is such gentle and warm hats that are in fashion today. The height of the main sheet of the cap from the set of rows to the place of decreases is 20 cm, taking into account the lapel. You can knit a more or less long hat. We calculate the number of rows that should be knitted: 20 cm * 3p = 60 rows. So, we need to knit 60 rows of the main fabric before the start of the decreases.





We calculate the required number of loops for the set of rows. To do this, you need to know the head circumference. In the example, the cap is tied to a head girth of 51 cm (50 cm * 1.8 p. = 90 loops). For knitting we need 89 loops for 2 knitting needles and 88 loops for knitting.

Remember that the pattern should be knitted according to the pattern of the hat for the girl. So the calculations will be correct, which means you do not have to bandage the product.

Knitting hats

Knitting a hat with cat ears is easy. You can use both circular knitting needles and hosiery. The example presents the option of knitting this model first on 2 needles. And not surprisingly, for many beginning needlewomen, circular knitting needles are less convenient.

Before the start of the cuts, the fabric is knitted in rotary rows. Remember, the main number of loops should be added 2 edge on the sides. The seam of the cap will be neat, located at the back. All odd rows are knitted with facial loops, all even ones are purl.





If it is more convenient for you to use circular knitting needles, you can immediately dial 90 loops on them, close the knitting in a circle. Now it remains to knit 60 rows of facial stitch. Remember, you must use a marker that will indicate where the new row begins.

Preparing to perform a decrease

Recall that in the example, the knitting of the main fabric occurs on 2 knitting needles.

Having knitted the required number of rows before decreases, all loops should be divided evenly into 4 knitting needles. At the end of the 60th row, all the loops should be closed in a circle, knitting the first and last loops of the row together front. So we will have stock for the formation of a beautiful seam.

If knitting is done on circular knitting needles, you can divide the main number of loops into 4 parts using markers. They can replace pins. Thus, there must be 22 loops on each spoke (88: 4 = 22).





Decrease

All decreases on children's hats occur in the same manner as described below.

61 row. We begin to form the crown of the cap. The first 2 stitches of the knitting needles should be knitted together behind the front wall of the front. The loop is gently tightened so that a hole does not form in the future. Next, we knit with facial loops. We knit the last 3 loops like this: remove the first one without knitting, the next we knit the front one behind the front wall. With the left knitting needle, we pick up the untied loop removed and throw it over the previously knitted front loop. We tighten the loop. We knit the last face. The last loop on each spoke will be the central path between the wedges.

So we knit all the odd rows in a circle, performing the same decrease in each wedge according to the same principle.

62 row. We knit without reductions with facial loops. We knit all subsequent even rows in the same way.





When 3 loops remain on each spoke, the crown can be pulled back. To do this, cut the working thread, pull the hook through all the loops. The thread should go into the first loop twice. We bring the working thread to the wrong side of the product, tighten it well. Now you can put it in the needle and gently hide it in the product. The ends are cut off.

At this stage, you can sew a hat if knitting occurred on 2 knitting needles.

Knitting ears

Children's hat with ears is almost ready. Now it remains to knit beautiful neat ears.

It is necessary to count 12 loops from the crown on the central strip of loops between the wedges located on opposite sides. Mark the place with a marker. Visually assess the location of the ears, they should not be too low or high.

The number of loops for knitting the ear can be different (depending on the yarn). Most often there are from 12 to 20 loops. The number of loops can also be calculated visually.

Crochet from the track of the vertical chain 10 loops one at a time. Knitting 10 faces, we re-knit them to smaller needles than we knitted the main fabric.

In the same direction, we collect 10 loops, knit them with the facial ones.

Turn over the knitting. From the remaining lobes of the same vertical chain, we collect another 20 loops in 2 needles in the same way. We knit in a circle with front loops for the front wall. From the second round we perform the reduction in each row.

Reduction of the eye: we introduce a knitting needle behind the front walls and we knit together the first 2 loops of the front on the first knitting needle. We tighten the loop. Some needlewomen perform this operation crochet. We knit to the end with the front ones, we also knit the second needle with the front stitch to the last 2 stitches. The penultimate loop is removed on the right knitting needle without knitting, the last one is knitted with the front one for the front wall. We throw an unknotted loop on the last. We do a broach.

So we continue to knit to the end, making reductions at the beginning of the first and third knitting needles and at the end of the second and fourth knitting needles of each row until there are 3 loops left on each. We hide the thread with which knitting began, inside the eyelet. Now you can use only 2 knitting needles, not 4.

We continue to make decreases until 4 loops remain on the needles. We cut the thread, stretch it through the loops.

On the cap, symmetrically to the first ear, knit the second in the same way.

Baby hat is ready! Carefully thread the hook, grab the thread and stretch it inside the eyelet. We erase the product, in a straightened form in a horizontal position, dry the hat.

Conclusion

As you can see, there is nothing complicated in knitting a hat with cat ears. The main thing is to prepare the necessary tools and a good mood.
