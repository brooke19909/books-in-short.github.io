How does voicemail work with MTS?

It is not always possible to answer the call. How then to find out what the person who dialed the number wanted from you? Should I call back and waste my time? An answering machine or the voicemail service will help out in such situations. On what conditions does the MTS operator offer it?

How to connect an answering machine and general information about the service



The "voice mail" service allows callers to leave messages when there is no answer or when the subscriber is offline. On your mobile device, you can use the advanced settings - add the ability to record, provided that the main line is busy, set a time limit in the absence of an answer. MTS voice mail allows the subscriber to personally create a message that callers will hear.



To activate the service, just call service number 7744. You can also use a short interactive command - asterisk, 111, grille and call. In the menu that appears, select the "services" item. The next level will be the voicemail we need. From this menu you can turn it on or off and get general help, as well as check the status of the service. You can use the short command: asterisk, 111, asterisk, 900 and the grille. Voicemail can be connected via SMS. For this, the text ‚Äú90 9‚Äù must be sent to number 111. MTS voicemail: smartphone app



For iPhone devices and Android phones, there is already an official application from the MTS operator for managing an answering machine. A convenient interface allows you to delete irrelevant messages, listen to new ones and set up voicemail. The time saving is obvious - before listening to the message, you can see the recipient and the duration of the recording. Accordingly, much less is spent on the use of the service - obsolete and uninteresting messages can not be listened to, but deleted immediately.

Like any other similar service, voice mail has a monthly fee. Connecting it is possible only with a positive balance of the subscriber. Listen to the received messages also will not work with a minus on the phone. If you use the application for managing voice mail, you can turn off the notification using SMS messages. Traffic while using the client is not charged. The program also works in roaming within the network throughout the Russian Federation. Among the additional features of the client is the function of sending messages. So, you can send the received message to your second number or to another subscriber. You can download the client from the official website of the operator or through proprietary application download services. The application is distributed on a non-commercial basis, you pay only for downloading data from the Internet according to the tariff plan. During installation, no codes or password are required.
