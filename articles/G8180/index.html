How to create a folder on a tablet. Guide

You do not know how to create a folder on your tablet on your desktop? Organize the home screen on your device so that it contains the shortcuts of absolutely all your favorite applications, regardless of their number. You save space on your Android home screen by placing multiple applications in the same directory.

Instructions for Jelly Bean

Depending on the version of Android, file display features may vary. Therefore, the algorithm for each of them will be given separately. For Jelly Bean, it will look like this:

  1. On the main screen, drag one icon over another. When you release your finger, they form something new. A circle will appear near it, which indicates that the files did not collide in a virtual accident, but instead formed a folder.

  2. Drag additional shortcuts to the directory icon to add these applications to the inside of it. As you drag and drop them, space will be freed up on the main screen, which will create even more folders, thereby increasing the overall capacity.

Here's how to create a folder on your tablet. However, there are some more important nuances.

Click the folder‚Äôs shortcut to view its contents, and then on the application itself to open it.

For further editing, you need to click on the program icon and hold it, then move it to the desired location. Or you can drag it back. Touching the name "Untitled", you can rename the folder.

Directories can be deleted, like any other shortcut on the screen. This will lead to the disappearance of everything inside.

Instructions for Ice Cream Sandwich

For this version of Android, directories look literally like folders. Exactly the same you could see on your PC. Here you have to create them manually and then drag and drop application icons into it.

You can do it this way:

  1. Click on a blank part of the screen.
  2. In the menu that appears, select the New Folder command. Next, you will see a new directory, but it is still empty.
  3. Drag the icon of the application of interest to the newly formed icon. Release your finger to add this file inside. Repeat this step as many times as necessary.

File managers

File managers can be considered one of the most important programs for comfortable work with your device. Using them, you can view available data, downloads, manage the space for storing information, move things from one place to another.

Organizing files may seem boring, but finding the right information on the device will always be relevant. Here are some examples of such programs: Asus File Manager, ES File Explorer Pro, Total Commander, Tetra Filer.
