Auto update

Update "Android" free for every modern person. To do this, it is enough to adhere to a certain algorithm of actions.

The first and easiest trick is the implementation of auto-update software smartphone. How to use it?

To do this, follow these steps:

  1. Connect to the Internet on a mobile phone or tablet. For example, via Wi-Fi.
  2. Go to the main menu of the mobile device.
  3. Open the "Settings" option.
  4. Select the menu item "About phone".
  5. Click on the "Update" button.
  6. Tap on the inscription "Check for updates."

Next, you have to wait for the completion of the procedure. If "updates" are detected, the user will be prompted to download and install the appropriate software. Just a few minutes - and the job is done.

Manual update

The next solution is to use manual updates. A utility called Odin comes to the rescue here. With its help, users will be able to download the latest official versions of Android.

The algorithm of actions in such circumstances provides for the following steps:

  1. Install Odin.
  2. Find and download the appropriate Android assembly anywhere.
  3. Include "One."
  4. Connect the mobile device to the computer. For example, through a USB cable. It is important that the gadget is correctly detected by the operating system.
  5. Turn off the smartphone.
  6. Put your device in boot mode. To do this, hold down the Home, Power, and Reduce Sound buttons.
  7. Confirm activation mode. You need to hold the volume up button for some time.
  8. In "Odin" select prepared files. It is necessary to adjust the parameters PIT, PDA, CSC.
  9. Click on the "Start" button in the previously mentioned program.

Now what? You just have to wait. If the update was successful, a green button with the word "Pass" will appear in Odin.

We figured out how to update Android on the phone. This is not the most common way to solve a problem. But do not forget about him.

The difficulty with Odin lies in finding suitable builds of operating systems. Otherwise, the reception does not cause any difficulties. It takes only a few minutes.
