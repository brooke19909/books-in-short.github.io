How to remove protection from a flash drive: simple methods, step by step instructions

If the user finds that he cannot add or delete documents on the media, it is likely that write protection is enabled on it. Sometimes the media is protected by the manufacturer, or a similar situation appears due to a malfunction of the disk itself. And then you need to know how to remove the protection from the flash drive.

If the user is not prompted for a password to make changes, you can unlock them using the Windows DiskPart utility. And before you remove the protection from the flash drive as a precaution, it is better to reformat the disk. This process is available on any version of Windows.

Login with administrator account

First you need to connect an external drive to your PC and enter the OS with an administrator account. In Explorer, expand "Properties." Select the disk file system - either NTFS or FAT32, and its capacity. Now you need to copy all the files from the media to the computer. After that, experts strongly recommend reformatting it. The operation will delete all data on the USB flash drive. Even if the user does not plan to reformat the media, Windows DiskPart itself can delete some files, so it is very important to prepare backup copies of documents first.

The sequence of operations to remove write protection on a USB flash drive is as follows:

  1. After starting the command line, typing cmd in the search, enter diskpart and Enter. DiskPart opens.
  2. Specify a list of disks in the DiskPart window and then Enter. Disks will appear, and you can determine the flash drive by its size in the "File Properties" window.
  3. Select USB, then a space and the number of the drive, for example, type select disk 1 and then Enter.
  4. Type the attributes disk clear readonly and then Enter.

Now the protection should be removed.

If it is still active, you need to go through another procedure:

  1. Before removing the protection from the microflash, simultaneously press [Windows] and [R] to open Run.
  2. Enter diskpart and click OK. The command line opens.
  3. Wait a few seconds, then enter the command ‚Äúlis dis‚Äù and confirm with Enter, after which all volumes will be listed.
  4. Enter sel dis X.
  5. Replace ‚ÄúX‚Äù with the displayed drive number.
  6. Use the create partition primary, select partition 1, format fs = FAT32 quick and ‚Äúactive‚Äù functions one after the other.

The flash drive is now formatted and has open access.

Preformatting a Disk

To format the external media in the search, type clean and press Enter. This function deletes all volumes and partitions from the disk, as well as folders and files.

The order of operations in order to remove the write protection on the USB flash drive:

  1. Enter the initial section and then Enter.
  2. Format the USB flash drive in NTFS or FAT32 format, as it was indicated earlier in the properties window. To do this, enter one of the following commands, and then press Enter: format fs: fat32 or format fs: ntfs. Progress is displayed in a window, and DiskPart will notify the user of the completion of the partition. It takes about a minute to format a 1 GB disk.
  3. Enter exit when the volume is formatted to close the DiskPart window.
  4. Now you can copy or move data that was previously located on the disk or add new files to it.

Removing write protection from exFAT card

Sometimes hard drives and flash drives formatted with exFAT become read-only after a computer crash. As for the exFAT system, most likely, you will need to remove the dirty bit from it, since this system has automatic protection.

There are several common fixes to remove a lock. Before removing protection from a USB flash drive, you first need to check whether there is a lock slider on its adapter. If so, turn it off to remove the protection. Next, you need to fix the existing error on the card and reset it or completely format it, and if necessary, change it to a new file system.

Using CMD before removing protection from a USB flash drive:

  1. Open Run. Type CMD and then Enter. Run as administrator.
  2. Type chkdsk / ff: and Enter. f: This is the drive letter from the exFAT section of the SD card. The program will find and fix the error on the map.
  3. If you receive the following error message: "Operation could not be completed." You need to run this command: chkdskf: / f / r / x and then Enter, after which the dirty bit and the security attribute will be deleted.

The user will not be able to format the device if you try to do this after a normal boot in Windows and receive a message that the device is write-protected. To counter this:

  3. When Windows boots up and the desktop appears, press WinKey + R and enter CMD in the "Run" dialog box to get DOS.
  4. In the black window, enter the format followed by the drive letter of the removable storage.
