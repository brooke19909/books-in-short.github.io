Configuring the DVR for viewing over the Internet: step-by-step instructions

Knowing how to configure the DVR to view over the Internet to the Web is crucial to install and connect to the World Wide Web video surveillance system. Working with a network of cameras with one router is not technically difficult. To do this, you need to know the main computer networks and security system. In the past, they were controlled by a security guard who was locked in the room all day, watching the monitors to make sure that nothing illegal was happening on the premises.

Modern devices have the ability to view the security system remotely, in real time. The technology has made such progress that it can send notifications of an altered state directly to a mobile phone or tablet so that the user knows what events are recorded by the security camera.

Wireless Camera Router

A router is a device that transfers data between network connections; it provides the ability to control several points simultaneously from one Internet connection. Without it, it is impossible to configure the DVR for viewing via the Internet. Today, many homes have Wi-Fi routers that connect devices to the Internet wirelessly. Almost any router is suitable for remote access to the DVR. Some of the major brands are: Linksys (Cisco), D-Link, Netgear, Belkin, and Apple.

CAT5 Ethernet cables are used to connect the system to the Internet. Digital devices with settings for remote access to the DVR come with a network port for connecting cat5, which is sometimes sold bundled with the system. But in most cases it is too short. Therefore, it is better to purchase an additional desired length. In addition, you will need one Ethernet cable to connect the modem to the router. They also usually come with their own short Ethernet cable, which is not enough to bundle a security system. And it is better to pre-purchase a new one of the required length.

Monitor connection

For the initial setup of the Tantos dashcam for viewing via the Internet, you will need a monitor, and in the future, the system can be viewed remotely. Digital video recorders have outputs that can connect a TV as a monitor using BNC, HDMI, VGA, or even composite RCA connections, depending on the type of equipment purchased.

Algorithm for setting the DVR for viewing via the Internet:

  1. Connect the modem to the Internet. Usually they have a number of indicators on the front panel that show the status, so that the user knows how the device is currently operating. Typically, a router has one port for connecting to the World Wide Web. Which is located far from others and is located on the rear panel of the router for devices connected to the Internet.
  2. The connection is made with a CAT5 cable.
  3. Connect the DVR to the monitor via the HDMI or VGA port, both on the DVR and on the monitor. The first of these is preferred for use.
  4. Now that you have the equipment configured and connected, make sure that everything works.
  5. DVRs may have menus that look a little different, but the settings are usually the same. The same goes for routers.

Setting dynamic DNS

Dynamic DNS (DDNS or DynDNS) is a method of automatically updating the name server in the DNS domain name system in real time, with the active DDNS configuration of its configured host names, addresses or other information.

Dynamic configuration is required if the SIM card has a dynamic public IP address. You can find more detailed information about what a dynamic external IP address is, but in the general case this means that it can change over time, as a rule, when a network is disconnected or re-registered. Dynamic DNS assigns an IP host name and constantly updates this name. But, even if the IP changes, DDNS will assign the same hostname to the new IP, making the router accessible through the same name at any time.

You must use an external DDNS service to create a host name and assign it an IP. Routers support many of these services. You can find a complete list of DDNS services in the "Services" ‚Üí "Dynamic DNS" section of the router‚Äôs web interface.
