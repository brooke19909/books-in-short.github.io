Radeon HD 5650 Mobile Graphics Accelerator: Purpose, Settings, and Performance

In 2010, AMD officially introduced the mobile graphics accelerator Radeon HD 5650. This device has advanced technical parameters and this accelerator in HD resolution allows you to run most of the toys of that time. Therefore, such a video card can be safely called high-performance, and it is precisely its characteristics and test results that this material will be devoted to.

Appointment

As previously noted, the discrete graphics adapter Mobility Radeon HD 5650 can be attributed to the group of gaming devices of the middle level. With such hardware, all 2011 toys can be launched. In some cases, of course, the quality of the displayed image will not be maximum. But this accelerator allows you to play comfortably in HD-resolution. This is also true for those projects that were presented in 2012-2013.

To date, this adapter has lost its relevance. He is not able to show sufficient speed in modern toys. Therefore, such PCs already need to be replaced.

In a test called 3D Mark 06, this device scored 6469 points. In Dirt 3, it can give out 104 fps, and in Diablo III - 79 fps. But these values ‚Äã‚Äãare valid for the lowest possible image quality.

GPU Features

The Radeon HD 5650 graphics adapter is based on a microprocessor code-named Madison. This chip was manufactured according to technological standards of 40 nm and included 627 million transistors. The GPU includes 400 stream processors. The clock frequency of the latter varies from 450 to 650 MHz. Among other features of the accelerator in question, we can single out support for DirectX version 11 and Shader 5.0. These technologies allow in some cases to significantly increase the speed of a personal computer.

Memory

The Radeon HD 5650 accelerator is equipped with a separate video memory subsystem. At the same time, the recommended RAM size is 1 GB, but you can also find laptops that are equipped with a 512 GB video card. The type of memory used is DDR3. The clock frequency of the used chips in the nominal mode should be 800 MHz. The capacity of the physical bus connecting the video buffer is 128 bits, that is, the bandwidth of RAM in this case is 25.6 Gbit / s. But, again, this value is theoretical. In reality, the RAM bandwidth will be lower.

Conclusion

As part of this review, an accelerator for laptops Radeon HD 5650 was considered. At the time of the start of sales, it showed a good level of performance. He is also able to launch later toys, which appeared on sale until 2013 inclusive. Now this accelerator model is out of date. Its speed is only enough to perform the most undemanding tasks. Therefore, owners should change such mobile computers urgently to newer ones.
