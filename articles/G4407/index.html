Error 21 on iPhone: causes, how to fix

Any modern phone will function normally only on condition that all its systems work without interruptions. This applies to both software and hardware devices. If the software was flashed or updated improperly, then there is a big risk of seeing an error on the smartphone screen. Therefore, installing third-party programs and utilities is not recommended when using phones such as the iPhone.





By the way, it is iPhones that most often bring inconvenience to their owners. It's not that Apple is releasing low-quality products. Quite the contrary, the phones of this manufacturer are considered one of the most durable and reliable devices. The problem is that users very often install applications on smartphones that are not compatible with the iPhone system. Most often, owners of "apple" products encounter an error 21. Let us consider in more detail what this may be connected with. How to fix the situation?

Causes

If error 21 appears on the phone, then this is not the most dangerous problem. Nevertheless, it brings many inconveniences. Such a code may appear on the phone screen if:

  • An error occurred while downloading updates. Especially often, error 21 occurs when updating iTunes packages.
  • Atyuns registries were damaged, for example, if the owner of the phone updated the software manually.

  • There was a "conflict" of programs. For example, if the user installed a new utility that deleted or blocked part of the files associated with iTunes.

However, most of these errors in one way or another are usually associated with a failure during the software update process.

Backup creation

Before starting any manipulations with the phone, you must copy all important information from it. Of course, for this you do not need to rewrite everything in a notebook. It is enough to create a backup. Fortunately, the iPhone allows you to do this very quickly. In order to save files from a smartphone, you must:

  • Connect the phone to the PC.
  • As soon as the computer recognizes the device and notifies about it, you must click on the icon with the right mouse button.
  • Select ‚ÄúCreate Copy‚Äù in the context menu.

You can also save all data using any of the cloud services. Fortunately, there are a lot of them now. After that, you can proceed to recovery. Error 21 should disappear.

Standard reset

This method is a universal way that will get rid of the "glitches" of the smartphone. However, it is worth considering that when performing a reset, all user information (pictures, photos, videos and other files) will be completely deleted. Therefore, it is so important to create a backup first. It is also recommended to immediately download the latest versions of iTunes and iOS from the official website of the manufacturer.





After that, you must perform regular firmware as follows:

  • Go to the iCloud section and turn off the iPhone search function. If the phone version is lower than 5, then this action will not be needed.
  • Connect your smartphone to your PC and turn on iTunes. The program may start automatically.
  • Next to the iTunes Store button there will be an icon with the image of the device. You need to press it and go to settings.
  • Click on the ‚ÄúRestore via iTunes‚Äù button and simultaneously hold down the Shift key on the computer. If recovery is carried out through Mac, then hold down Alt.
  • The following dialog box will appear on the screen, in which you must specify the path to the previously downloaded files of the fresh OS.

After recovery, the device will go into Recovery Mode. The settings will be reset to the factory settings. At the next stage, it will be possible to return files from the backup to the phone. If error 21 reappears, then continue.

Recovery mode

In this mode, you can also try to fix the error. True, in this case, too, you will have to first save all the necessary documents and contacts from the phone book.





Next, you need to perform the following steps:

  • Disable device.
  • Hold down the Home key.
  • Without releasing the button, connect the smartphone to the PC.
  • The media player icon will appear on the screen, and the system will inform you that the gadget has switched to Recovery Mode.
  • If iTunes does not load automatically, you must force the program to run. A notification should appear on the screen that the device has been recognized by the media player.
  • After that, the same actions are performed as during normal recovery.

At the final stage, you must specify the path of the new firmware and enter the Apple password.

To summarize

If error 21 on the iPhone has not disappeared, then the problems may be more serious than it might seem at first glance. In this case, you must contact a specialized service center and diagnose the device.
