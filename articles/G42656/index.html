How to record an iPhone screen: instructions and tips

Sometimes users ask how to record the iPhone screen on video. There are several methods that will help you complete this procedure. The most interesting thing is that in iOS 11 everyone can record from the phone itself. Therefore, the need for third-party programs disappears. Other options considered will be for those who can not install such software on their iPhone. Their essence is the use of a PC, to which the gadget is connected.

How to record iPhone screen on video?

Owners of version 11 of the software can rejoice, because they do not need to bother using other options. For the procedure, only the iPhone itself is needed, in the settings of which there is this function.

To activate it and understand how to record what is happening on the iPhone screen, you need to follow the basic steps:

  1. First of all, the user enters the device settings, where he searches for the inscription "Control Point".
  2. Next, you need to visit the section "Configure controls".
  3. There we already find "More controls" at the end of the list, where the "Screen Recording" function will already be located. Near the inscription, click on the plus sign.
  4. When the user returns to the desktop, he should call up the menu by pulling the bottom of the screen. There will already be a recording function.

Procedure result

To activate the process, you must click on the appropriate button. The screen recording itself will begin, but it will be made without sound. If you need sound, you can activate it by calling the additional menu with a long press on the record button.

When the recording is finished, stop it by clicking on the icon again. The finished file will be automatically saved in the "Photo". Here's how to record the iPhone screen on the phone itself. The procedure is very simple and available for each owner of version 11 and higher.

PC Recording

Before recording video from the iPhone screen, you need to make sure that the gadget and PC are connected to the same Wi-Fi network.

If there is a need to use this method, you need to additionally download a special application that can allow you to broadcast on AirPlay. You can search it online on the official website.

There is a proven utility LonelyScreen AirPlay Receiver, which must be downloaded on the official website and then installed, following the instructions. It is effective in making the process convenient and fast.

The basic steps of how to record an iPhone screen:

  1. First, activate the LonelyScreen AirPlay Receiver program.
  2. After that, already on the gadget itself, you need to go to the "Control Center" and activate the screen rotation function.
  3. A list appears where all available devices on which it is possible to broadcast will be located. Click on LonelyScreen.
  4. The device screen will appear on the monitor in the window with the program.

When everything is set up, you can already directly record the screen. It occurs due to programs stored on Windows (10). If there are none, then you need to download them from the Internet.

QuickTime on MacOS

For Mac owners, the procedure will be different. Recording takes place thanks to the QuickTime Player application, which is built into the device.

  1. First, we connect the phone itself to the device using a cable. We synchronize the work of gadgets.
  2. Run the actual program on Mac. After it has opened, in the menu we look for the "File" section, and then "New video recording".

  4. Before starting, we press the "Record" button, when shooting is completed, click on the "Stop" button.

When the procedure is completed, the finished file can be saved to the phone if you click on "File" and "Save" in the main menu.
