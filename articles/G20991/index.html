CPU and its features

Smartphone Lenovo A859 Gray, like its other modifications, is equipped with the same type of processor. This is a single-chip system based on the architecture of the MTK 6582 workstation. It consists of 4 revision cores A7. The maximum allowable clock frequency is 1.3 GHz, which is automatically achieved in peak load mode. If maximum performance is not required, then the processor automatically unloads up to 300 MHz. In this case, idle kernels will automatically shut down. MTK 6582 managed to prove itself from the best side, but it can not be attributed to the new products. It has long been successfully sold, but still a little outdated. So do not forget about it at the stage of choosing a new smartphone.
