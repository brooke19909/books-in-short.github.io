How to hide whatsapp visit time? Several ways

WhatsApp is a pretty popular messenger with tons of extra features. By the way, not all users know about some. For example, how to hide the visit time in WhatsApp, so to speak, enable the "invisible" mode. And this option is quite useful, as there are situations in which you need to go unnoticed. This article will describe how to use this opportunity in several ways.

Hide Visitor Information

Those who have been using the Vatsap messenger for a long time may have noticed that authorized users in it have the corresponding status. For example, if your interlocutor is currently on the Web, his Online status is on, and if he left it a while ago, the time of the last activity will be displayed. However, this alignment of affairs does not suit everyone, and someone would like to remove this indicator.

After installing the program on a smartphone, it automatically sets the privacy settings, and they are configured so that your status indicator is constantly active. However, the user can independently change the corresponding item in the settings.

Let's figure out how to hide visit time on WhatsApp on Android:

  1. Launch the app.
  2. Go to settings.
  3. Open the "Account" section, and then go to "Privacy."
  4. From the "Visit time" drop-down list, select the desired value. for example, if you select None, then no user will be able to observe your status.

This was the first way to hide whatsapp time on WhatsApp. By the way, if you have an iPhone, then the instructions will be the same, only the name of some items will change.

How to remove status on WhatsApp

In addition to the visit indicator, many users would like to turn off the display of their status. But there is one problem - you simply cannot leave it empty; you need at least one character. By the way, this can be used:

  1. Open the messenger settings.
  2. Go to the "Status" section.
  3. Erase the established status.
  4. Set any character, for example, a period.

Thus, users will see a dot instead of status. But this does not suit everyone. Fortunately, the developers thought this through, and you can hide the status completely so that it is not visible to others.

  1. In the settings, go to the "Account" section.
  2. Then tap on the line "Privacy".
  3. From the Status drop-down list, select None.

After that, your status will become invisible. By the way, if you would like to hide it for individuals of your address book and not only, then add this user to blocked ones.

We remove visiting time in "WhatsApp"

There is a third way to hide the time of the last visit in WhatsApp, and it consists in using a special application. You can install it directly from the Play Market by entering a name in the search bar.

After the application is installed, you need to do the following:

  1. Launch the app.
  2. Give permission to use the resources of other programs.
  3. Wait for the scan process to complete.
  4. Read messages and chat with friends in stealth mode.

The program simply takes and downloads all messages from the messenger. Thus, you do not communicate with people through the program, so your status will be "Offline". So we figured out all the ways how to hide the visit time on WhatsApp. Well, how to use it is up to you to decide.
