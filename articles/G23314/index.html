You can try to prevent or correct them yourself. In this article, we consider all the characteristics and malfunctions of the device.

Inclusion and use

In order to use the machine, you should adhere to a certain action plan. You need to open the tap with water, wait until the device turns on, and all indicators on the control panel light up. Next, open the door, get the dispenser and pour detergent into it. After that, you need to fold all the dishes and close the appliance. Now you can choose a program of work and pick up all the necessary additional functions. The system is launched after pressing the "Start" button. After the machine completes its work, a light bulb with the inscription "End" will turn on. Next you need to close the tap, unplug the power cord and the dishwasher itself.





It is recommended to remove the dishes after 10 minutes. During this time, the machine will cool. Unload from the lower basket. You need to understand that if the wrong program was selected, then it can be changed if the machine started working a few minutes ago. Moreover, during the washing process you can load up the dishes. For the best result, you should choose an expensive and high-quality detergent, and also choose the right dosage. It should be remembered that a huge amount of this chemistry will not improve the quality of washing, but only pollute the air.

rules

To avoid any malfunction of the Indesit DSG 0517 dishwasher (instructions will help to understand the error code), you must follow the rules. First, you only need to choose the detergent that is created for the dishwasher. Using what is designed for a manual process is prohibited. After the machine finishes washing, you need to check whether the foam remains. If yes, then the detergent was used in excess.





Cart loading

Often, if the basket is not filled correctly, the machine ceases to function properly. In order to prevent such problems, you need to remove all food debris before loading the dishes. Then you should check that all devices are installed correctly and stably. It will be useful to pay attention to whether the blades rotate freely.

How to download?

The lower basket is made in order to install pots, lids, plates, salad bowls and so on. If any of the devices is large, then it should be placed above the basket. A sprinkler is installed at the top, so the dishes that were installed should not complicate its work. In that basket where it is necessary to establish spoons, forks and so on, there are special fastenings. These shelves are sliding. The devices should be placed with sharp ends down or laid in a horizontal position.





For light dishes, which are particularly fragile, an upper basket is provided. So that glasses, cups, salad bowls, pans and pans of small sizes can be properly installed, the manufacturer has provided for the adjustment of this shelf. Correction should be carried out without load, as the process can be significantly complicated. In addition, the basket must be raised and lowered on both sides simultaneously.

What dishes should not be washed?

There is a list of those devices that are prohibited for washing in the machine. It is a plastic dish that does not withstand high temperatures. Wooden fixtures and those that have small inserts of this material are also unsuitable. If the dishes have glued components, then do not wash them in the machine. Accessories made of antique silver, copper, tin, porcelain and hand-painted should not be placed in the device.





There is a slight nuance. Modern silver dishes are washed, however, cleaning should be carried out only in a delicate mode and make sure that the device does not touch other devices.

Dishwasher care: reviews

Users advise: to keep the dishwasher in working condition as long as possible, it is necessary to carry out proper care. This will prevent malfunctions. Buyers in the reviews warn that the tap should be closed after each process so that there are no leaks. Before starting maintenance and cleaning cycle, unplug the plug from the power outlet. If the machine did not turn on for a long time, then before washing the dishes, you need to make an idle run so that the washing process resumes. In order to clean the control panel and other external parts of the machine, you need to use a special rag. It should be wet and soft.





Users note that abrasive cleaners must not be used, like any solvents. If there is a need to remove stains from the surface of the chambers, you need to resort to using a rag pre-moistened with vinegar. Using a damp sponge, periodically clean the door gum and all containers. During downtime, the door must be open, then air will not stagnate. Leftovers can adhere freely to the blades. This entails difficulty draining the water. Therefore, it is necessary to check the condition of the sprinklers and, if necessary, clean them. Use a brush that will not scratch the parts. All sprinklers are taken out. What should be done to get them? First, you need to pull the bottom sprinkler up. If there is a need to remove the top, then you have to unscrew the nut. It is made of plastic, and it should be twisted counterclockwise.

If the water supply system has not been used for a long time, or the fluid supply system has been replaced, check that it is transparent. If this rule is neglected, then the machine can quickly become unusable due to clogging of the filter. The latter consists of three parts that perform their functions. The first cleans the water, the second washes from food, and the third resumes the circulation of water. Filters should also be cleaned periodically. How? It is not necessary to use a brush made of metal, but it should be washed under running water. First you need to turn off the fluid supply to the device, then unscrew the sleeve at the end of the pipe. The filter is freely removable and cleanable under running water. After fixing the device, you need to re-tighten the part from the pipe. Do not wash dishes without a filter.
