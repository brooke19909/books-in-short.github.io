Connection in the department

Via telephone

The next option is to use the phone. As we already said, a person can call the bank and activate the studied option. This alignment is found in real life more often than the first method.

How to unlock Mobile Bank? In the case of Sberbank, you will need:

  1. Dial 8 800 555 55 50 or 900 on the phone.
  2. Press the "Call Subscriber" button.
  3. Wait for operator response.
  4. Announce that you want to connect to mobile banking.
  5. Name the cardholder's full name, secret word, bank plastic data.
  6. Report the mobile number to which you want to bind.

It remains only to wait a while. After a couple of minutes, a citizen will receive a message on the mobile phone about the successful connection of the studied option. Sberbank's Mobile Bank number is 900. This combination will help in processing requests made with the card via telephone.

Main teams

It's time to talk about the basic operations of the Mobile Bank. How to transfer money? And to recharge a mobile phone from a card? We will talk about SMS requests and USSD commands. There are a lot of them, so we will focus on only a few operations.

Here are the main USSD requests that help you work with Sberbank's mobile banking:

  • * 900 * 1 # - information about the status of the plastic account;
  • * 900 * replenishment amount # - transfer money to a phone attached to the card;
  • * 900 * recipient_number * amount # - payment of any mobile phone;
  • * 900 * 12 * receiver_phone_number * money # - money transfer to a card linked to the number specified in the request.

That's all. These are the main operations that interest Sberbank customers. But how to send an SMS request of one type or another?

The generated message is sent to number 900. Here are a few forms that allow you to use mobile banking (message text options):

  • transfer amount - transferring money to a phone attached to a card;
  • balance XXX, where XXX is the last 4 digits of the plastic - receiving SMS about the status of the account;
  • tel. recipient_number sum - replenishment of another's number;
  • transfer recipient_number money - money transfer to plastic, attached to the specified phone number.
