Service "My Services"

Using it, you can get a list of both paid and gratuitous services connected to your room. To use the service, you need to send a message to number 8111. If you need a list of free services, write the number 0 in the message. For paid options, the text will be the number 1. If you need a complete list, you can send an empty SMS or a message with any text except for "0" and "1".

In response, you will receive an SMS with a list of your services. But there is one point. If the list takes more than 5 messages, then you will not be able to see all the connected options. Have to use other methods. In addition, the list will not include information and entertainment services, including GOOD'OK.
