PC installation





It will be much easier for the user to deal with updating the system through a computer. With this installation method, you will need: a USB cable, a battery charge of about 50%, firmware, drivers.

First you need to find a suitable version of the system. You can do this on the official website or on other Internet resources. It is advisable to choose the official firmware, but some "custom" ones will also show themselves well. After downloading the OS, you should prepare your PC for installation. The user needs to install drivers to connect the device via USB.

You can find drivers on the network or using a special program from LenovoUsbDriver. You can also use the device manager. This method allows you to automatically find the driver.

Then the user will need a firmware program. For "Lenovo A328" you can use the simple and reliable application SP Flash Tool. Having launched the program, you need to click on the Scatter File, specify the path to the previously saved firmware and open it. After that, check the box next to DA DL ALL WITH and click on the Firmware Upgrade button. Now the user must connect the disconnected device via USB. The process will start automatically and its progress will be displayed below in the status bar. The action takes about five minutes, at the end of the program will notify the user about the result.

After flashing, disconnect the device from the cord and start it. The first inclusion of the device on the new system takes a long time, so you should wait.
