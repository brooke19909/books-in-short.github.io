The simplest pattern

In this section, we explore how to knit the simplest, but at the same time very original pattern. The name he has is quite memorable - ‚Äúgum‚Äù. And the options look very beautiful:

First we need to dial a certain number of loops. Their number should be a multiple of the number of loops in the elastic band to make the pattern look complete. And plus two more edge loops. The number of readers should be determined independently. However, it is important to mention that the width depends on how many loops the scarf consists of.

Now proceed directly to the knitting of the first row, forming a pattern:

  1. We remove the first loop.
  2. Then we knit one, two or three face, and then the same number of purl.
  3. The last loop of the row is the wrong one.
  4. Next, we knit the second row. We also remove the first loop.
  5. We knit the rest according to the drawing. Where in the previous row there were front loops, now you should knit the wrong ones. And vice versa.
  6. So we move for a certain number of rows. It depends on how long the scarf with knitting needles we want to knit.

This pattern is suitable for a warm scarf, and for spring or summer. The main thing in this case is to select the appropriate threads. The product should be warmer, the thicker the thread should be chosen, perhaps even need a woolen. It is also important to note that the gum pattern is suitable for male and female knitwear.

The finished drawing, as well as its knitting technology, is similar to that which we presented to the reader in the previous paragraph. However, there is still a significant difference. Otherwise, the drawings would be identical. And then you can verify this:

  2. Having decided on the width of the scarf, we proceed to the formation of the pattern. To do this, as usual, remove the first loop. Recall, it is called edge. It is quite possible that the reader does not understand why to do this at all. Therefore, we will explain. So, the essence of this technology is to alternate front and back loops in each row. Forming an interesting volumetric pattern "speckles".
