How to set up Viber on iPhone, Android and PC

The Viber program allows you to chat with your friends for free. Moreover, the application has become so popular that now it can be installed on both a smartphone and a personal computer. True, far from every user can configure Viber independently. But, thanks to the information from this article, you can successfully cope with this task.





Version for Android

For starters, it‚Äôs worth considering how to configure Viber on Android. Smartphones on this platform are considered the most affordable, so you need to pay attention to them first. So, to configure Viber, do the following:

  • Download the application using the Google Play service. Of course, you can do this on a third-party resource, but in this case you are in great risk of catching some malicious program.
  • The application will ask for permission to access some functions of your smartphone. Give your consent to this, otherwise the program will not be able to work normally.
  • Wait until the installation of Viber is completed, and then run it.




  • A window will appear in which you will be asked to enter your phone number. Do this, and then choose one of two ways to activate your account: call or SMS.
  • Regardless of your choice, you will receive a special code. Enter it in Viber (the input window will open automatically), thereby confirming the authenticity of your account.

On this setting "Viber" can be considered complete. You only need to attach the contacts from the phone book to the program, after which you can enjoy free calls, messages and other functions.

IPhone Version

If you are a happy owner of an Apple smartphone, then you will probably be interested to know how to set up Viber on an iPhone. In principle, this procedure is not very different from that for Android OS devices, but there are still some features. To start using Viber:

  • Download the app from the App Store. For convenience, you can use the search.
  • Install Viber. The program will ask for access to the address book, and you will have to agree to this. This will allow you to immediately add the contacts of all your friends in the Viber directory.
  • Next, indicate your phone number. A message will come to him (or a call will come) with a code that will need to be entered in the program.




  • Now enter your name or use the data from Facebook.

That was all you had to do. The Viber program is configured and now you can use all the functions provided to it.

PC installation

As mentioned earlier, you can install and configure Viber not only on the phone, but also on the PC. First of all, you will need to download the program to your computer, and then do the following:

  • Run the installation, and then accept the license agreement.
  • A dialog box opens asking if ‚ÄúIs Viber installed on your phone?‚Äù Click ‚ÄúYes‚Äù and enter your number in the line that appears.
  • Next, you will be asked to verify the correct phone number. Do this and click ‚ÄúConfirm‚Äù.
  • After that, a QR scanner will automatically start on your smartphone, and a QR code will be displayed on the computer monitor. Just point the camera at it and the information will be read.




You will see a message about successful activation. Now you will need to wait until the PC version of the program is synchronized with the mobile application. This usually does not take much time.

Installation on a PC (if there is no smartphone)

As you probably already understood, in order to install the PC version of Viber, you will first need to configure this program on your phone. But what if you do not have a device that meets the requirements of the application?

Fortunately, there is an alternative option that allows you to install the program without a smartphone.

  • Since you cannot configure Viber on the phone (due to the lack of a suitable device), you will need an Android OS emulator. This program is called Bluestacks and can be downloaded for free from the official website of the developers.
  • Install the application on your computer and log in using your Google account (create a new one if you don‚Äôt have one).
  • Go to Google Play and download Viber from there, and then launch it.




  • Enter your phone number and confirm it via SMS.
  • Now minimize Bluestacks, download Viber for PC and start installing the program.
  • The application will ask for your phone number. Enter it, and then select "The camera does not work."
  • Return to the Bluestacks. Viber will automatically start here and a message will appear with an activation code. Copy it, and then use it in the PC version of the program.

That's all. Now you can remove Bluestacks and safely use Viber on your computer.
