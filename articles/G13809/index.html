How to disassemble iPhone 3GS: instructions

Before you begin to resolve the issue of how to disassemble the iPhone 3GS, you need to understand that you carry out this operation only at your own peril and risk. If you accidentally damage any parts, in this case, the technical service may refuse to help you, and this happens in most cases. Therefore, first we recommend that you think about whether you will carry out repairs yourself or whether it is worth trusting specialists.

Necessary equipment



In order to disassemble the iPhone 3GS, you will need to have a set of special tools on hand. A collection of watch screwdrivers, a spoke or a plastic spatula with a thin end will be involved in the work, however, you can also use the pick. A thin metal knife will also be required, but the presence of a scalpel is still recommended. For better illumination, use a carry or a desk lamp. If you need to replace the protective glass on the screen of the device, in this case, a hairdryer must be at hand.

Order



So, let's get started on how to disassemble the iPhone 3GS. We will act in a certain sequence. If you use our instructions, in this case, be sure to remember that all points must be observed in strict sequence, otherwise you can only harm your device.

Display

The first thing you should do is inspect the iPhone. If you carefully studied it, then at the bottom you can notice two bolts, which first need to be unscrewed. When this is done, you can open the screen unit, this is done as accurately as possible, because otherwise you can damage the cables. To remove this element, you can use a thin plastic or metal object, but still we recommend using a suction cup, so you will not damage the edges of the glass. If you needed to learn how to disassemble iPhone 3GS, then in no case should you be in a hurry and be sure to follow the instructions. When the partial screen is moved and raised, you should start by disabling the motherboard cable. There is a necessary element in the upper right part of the device. After this loop is turned off, you should deal with disconnecting the second one, which is intended for the sensor. The third leads to dynamics. You should also disable it in order to be able to completely disconnect the screen part. If everything is done correctly, then the module of interest to us can easily be separated from the bottom of the device. Now you know how to disassemble the iPhone 3GS to replace the screen or protective glass, but if you should continue the process, for example, to install a new motherboard or battery, in this case, follow the steps that we will give below.

Deep repair



Pick up the bottom of the device. On the front side you can find screws, on each side there are three of them. We turn off these elements and separate the display from the glass with the frame. In fact, care must be taken when separating the display. If you want to replace the glass, in this case it will need to be heated with a hairdryer, but if you do not plan to do this, we will not need to use a heating device at all. How to disassemble the iPhone 3GS, now you know, and if you act strictly according to the instructions given, then you can do without the services of specialists and not harm anything, but rather, fix the problem yourself.
