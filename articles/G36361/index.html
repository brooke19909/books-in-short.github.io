Xerox Phaser 3117 Printer. Technology, Specifications, and Setup

The Xerox Phaser 3117 Printer aims to create entry-level printing systems. At the same time, very high requirements are put forward to the speed of work and the monthly resource. But the quality can be average. It is about such a peripheral device that will be discussed further.





Technology, positioning and delivery list

The Xerox Phaser 3117 printer is based on the laser printing method. The essence of this method is that the toner is "pressed" into the surface of the carrier by heating. Depending on the amount of black powder, different grades of gray are obtained.

The largest print limit for this device is 5000 sheets. Therefore, such a solution is best used to create small workflow systems. The following solution can boast of a complete set of the following type:

  • a printer;

  • interface cord for connecting to a PC;

  • compact disc;

  • corporate cartridge (starter);

  • user's manual;

  • power wire;

  • warranty card;

As will be shown later, this list is quite sufficient for connecting, configuring and using such a printing subsystem.

main parameters

The Xerox Phaser 3117 printer can print on only one standardized media format - A4. But in the application management software, you can set various page sizes that cannot exceed the maximum allowable 210mmx297mm. The resolution when printing on a page with A4 dimensions is 600x600 dpi. Such a peripheral device takes 30 seconds to warm up before printing, and only 10 to output the first sheet. After that, the laser printer can output up to 16 pages per minute. The density of the sheets can be from 63 g / m 2 to 163 g / m 2 .

Cartridge

The Xerox Phaser 3117 Laser Printer can work with only one cartridge model. It is labeled 106R01159. Its limit is 3000 pages at one gas station. Among other features of this cartridge, it can be noted that it, in addition to ordinary office paper, can also print on cards, transparencies, labels and envelopes. But such a printer can only output information in monochromatic format.

Step-by-step setup

In order to install the Xerox Phaser 3117 printer, you must perform the following manipulations:

  1. At the first stage, install the device on a computer table next to the system unit. The maximum distance to the last must not exceed the length of the interface cord.

  2. Next, install the cartridge by removing from it all the protective films and cartons.

  3. Then do the switching. Connect the interface cable with USB connectors on the one hand to the printer, and on the other to the printing system. Connect the power cord with a plug to the power outlet, and the connector to the printer.

  4. Turn on the PC.

  5. After this, you must wait until the download is complete.

  6. Turn on the printer and wait for the driver installation to complete. If the operating system could not install them, then install them from the CD.

  7. Select the option whether the printer will be available in a networked environment.

  8. Print a test page.





Conclusion

Just today, the Xerox Phaser 3117 printer boasts a good balance of parameters and economy. It is really great for home or office use. It is in such cases that it is most optimally used both from a technical point of view and from an economic point of view.
