Brief technical specifications. Software





First of all, we note the operating system. On board the Lenovo Vibe X2 Gold, we have the Android family of OS installed. More specifically, this is version 4.4 of KitKat. But some companies do not stop using ordinary operating systems. This was the case with HTC. cellular





The device works in GSM networks, HSDPA + is supported. CPU





In this case, as a chipset, we have a MediaTek device, this is the MT6595m model. The processor works with eight cores, the maximum clock frequency can reach two gigahertz. Very, very good for such a device for the appropriate cost. The graphics subsystem is PowerVR G600. Two gigabytes of RAM are built into the smartphone, which is more than enough for 90 percent of games and applications. There is an opportunity to successfully use the phone in multifunctional mode and open several programs at the same time. At the same time, the operating system runs smoothly and clearly, without any hint of failures.

For storing user data (i.e. text documents, music, photos, video clips and movies, as well as games and applications), the phone has 32 gigabytes of long-term memory.
