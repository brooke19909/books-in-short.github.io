The simplest crochet skirt for a girl is not at all difficult to make. Patterns are not needed. The "future waist" of the baby is measured and the canvas is knitted. If you decide to knit a skirt without seams, then you do not need to add, if the crochet skirt for the girl is planned with a fastener, then add 3 cm.

It is desirable to knit the first rows of columns with a thread of elastic yarn or Hungarian. Having tied 3 cm, add one column for each row at an equal distance twice. The foundation is ready.

And then it all depends on imagination and desire. You can finish the skirt according to this scheme, or you can connect the frill with the same columns with a crochet or use any scheme for a napkin or panama, if only it was with an extension.

Assembling a skirt is easier than knitting. If the fabric is not circular - fold 1.5 cm on each side, sew and sew on the button fastener. If the fabric is tied round-the-way, the skirt is ready.
