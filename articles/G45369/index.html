How to blacklist an unwanted number on Tele2?

The desire to block the receipt of calls and messages from a specific number may appear for various reasons. The ability to refuse to communicate with the subscriber exists in most modern mobile operators. Tele2 is no exception. How can I get rid of an obsessive fan or avoid communicating with an unpleasant person? How to blacklist Tele2? We will talk about this in this article.





Terms of Service

Before answering the question of how to blacklist Tele2, I would like to familiarize potential users of this service with its terms:

  • You can activate the service simply by adding the first number to the list of unwanted subscribers.
  • Deactivation of the service can be carried out upon exclusion from the list of all numbers or forced shutdown.
  • If the “Black List” was forcibly disabled, then the existing subscriber numbers are stored in the memory for one month (that is, if the service is re-activated during this period, the numbers stored in the list will still be blocked).




  • When connecting the service, all calls and text messages from the selected subscriber will not be received.
  • In total, a limited number of numbers can be added to the black list: 30 pieces (if other numbers need to be added, then some of their current ones should be deleted first).

Terms of Use of the Black List Service

How to blacklist Tele2 and how much will it cost? Commands for activating and managing the service will be given below. Blacklisting is free. In this case, the first is the connection or the secondary. There is a daily subscription fee of one ruble (it does not depend on the number of subscribers on the list). Also, every time you add a number, you will be charged 1.50 rubles. (regardless of which account number is first or thirtieth).





How to blacklist Tele2?

Earlier it was mentioned that the service is connected at the moment when the first number is added to the list. Thus, special commands do not need to be entered. How to blacklist Tele2? Adding numbers and managing the service is done from the phone. So, to enter the first number in the list and activate the service, dial the following combination: * 220 * 1 * <blocking number> #. The subscriber number should be indicated in the format “8”, followed by a ten-digit number. In response, you will receive a message about the activation of the service and the successful completion of the operation.

Managing the Black List service

In addition to the fact that the subscriber can add the number to the list of “blocked” (we talked about how to put the subscriber in the “Tele2” blacklist), a number of commands are available for him to manage. In particular, you can view the entire list of unwanted subscribers by entering the request * 220 #. Information from the number will be received in a text message. If you need to add the second and subsequent numbers, the same command * 220 * 1 * <subscriber number for blocking> # will be useful. At the same time, you can remove a subscriber from the list by changing the unit in this command to “0”. If the question of how to put the phone in the “Tele2” blacklist, everything is clear and you need to find out if calls were made to your number from blocked subscribers, then the following recommendation will be useful. You can get information on calls from unwanted numbers by dialing * 220 * 2 #. In this case, it is possible to receive data for the last 48 hours (in other words, if a call from a blocked subscriber was made three days ago, then information about him will not be displayed in the message).





Deactivation of the Black List service

If the question of how to blacklist Tele2 is no longer relevant and there is no longer any need to block the subscriber, you can turn off the service. As mentioned earlier, its deactivation can be carried out in two cases:

  • If all numbers are deleted from the blacklist.
  • A forced shutdown was carried out, while the numbers remained blocked.

With the first paragraph, everything is clear - just erase all the numbers from the list. By the way, this can be done not only with the command * 220 * 0 * <number for unlocking> #, but also by sending an SMS message to number 220, the text of which requires the following combination: 0 ** <number for unlocking> . When you force the Black List service to be disabled, you need to enter the * 220 * 0 # command from the phone. In this case, calls from previously blocked numbers will again come in the same way as messages. The subscription fee will cease to be deducted from the next day. All numbers that remain on the list are stored for thirty days. During this period, you can reactivate the service while maintaining the previous settings. How to blacklist Tele2 if more than a month has passed after the service was turned off? The commands for managing the blacklist remain the same. All numbers that were previously present in the list of undesirables, if necessary, will have to be added again, paying for each 1.50 rubles.





Conclusion

The Black List service provides an opportunity to independently outline your social circle. Now you can refuse to accept calls and text messages from unwanted subscribers. In this case, there remains a way to check whether calls to your number from the black list have been made in the last 48 hours.
