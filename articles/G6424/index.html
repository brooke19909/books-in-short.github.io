Elastic band in knitting is one of the most famous and common patterns that is used to create many products. In general terms, it is a special combination of seamy and facial loops that form vertical stripes, giving the canvas the ability to stretch.



Different knitting methods

There are a large number of ways to knit elastic. This pattern can be simple, consisting only of purl, forming concave lines, and facial loops forming convex. In the diagrams, it is indicated by numbers that show the number of loops (for example, 2 x 2). In addition to simple ones, there are more complex patterns, with elements of a beautiful openwork knitting, crossed loops or braids. Gum is used in the design of the neck, bottom of the product, sleeves and collar. Often it is used as the main pattern for knitting sweaters, scarves, leggings, mittens and socks. In this article we will tell you how to knit a double elastic band, as well as the French elastic band "Snake". For beginner needlewomen, we will present photos and give a detailed description of the technique for making these patterns. We advise you to adopt these simple methods of knitting. They will be useful to you in the manufacture of a wide variety of things.



How to knit a double elastic band? What is it used for?

The canvas, made with a double elastic band, is elastic, dense, hollow. This pattern is often used to create cuffs, inside pockets, starting elastic bands, as well as when designing the hem - in hats, mittens and children's pants. Such an elastic band allows you to create an elastic canvas folded in half. Therefore, it is successfully used when it is necessary to create a cavity in the product, for example, to insert a linen gum. How to knit a double hollow elastic band we will tell further. In order to learn the technique of this pattern, we suggest that you perform a small sample. Prepare any yarn and knitting needles. Remember that elastic bands are always made with knitting needles, one to three sizes smaller than those used to make the main fabric.

Step-by-step description of the double hollow elastic pattern





How to knit: first we collect loops. They need to be made an even number. In this case, do not forget about the edge. Usually, two times more loops are required to complete this pattern than to create an entire product. How to knit a double elastic band? We collect 40 loops. We knit row number 1 according to the following pattern. First, remove the edge loop. Next, we knit one front. We remove the second loop as the wrong one on the other knitting needle. The third - again we knit as a facial. We complete the entire series to the end, alternating these elements. The second row begins with the removal of the edge loop. The first front is also removed on the knitting needle. The second, shot in the previous row, is knitted with the front one. We continue knitting by analogy to the end of the row. Remember, the working thread should always be between the front and removed loops, that is, before work. Similarly complete another 12 rows. That's all, you got a sample.

Some more tips for making this pattern





How to knit double elastic with knitting needles you already know. It remains to talk about increasing the number of loops, as well as closing the row. If this pattern is created after the main canvas, often there is a need to increase the number of loops. There are two ways to do this:

  • having done the crochet, which in the next row should be knitted with a facial loop;
  • stretching out through the already knitted loop a new one.

If you knit a double elastic band with knitting needles at the end of the product, the last row will need to be closed. You can do this in any way convenient for you, for example, knitting together two loops of the front for the back wall.

Another interesting way to finish: French gum "Snake"





If you have learned how to knit a hollow double elastic band, we advise you to learn how to perform another unusually beautiful pattern. It is called the French gum "Snake". This pattern can transform the product, make it more interesting and original. In order to learn how to knit "Snake", follow the pattern. Prepare threads and knitting needles for work. Dial the number of loops to be divided into four (plus two edge loops). We scored 22 loops for knitting a sample. Remove the first (edge). Next, knit one loop of the wrong one. And then repeat the pattern "2 facial loops - 2 wrong loops". As a result, the last two loops of the first row should be purl. Knit the second row by analogy. Now we proceed to the direct execution of the "snake". The first two rows we made to facilitate the learning process. In knitting the main product, these first two rows do not need to be performed. We knit the third row (odd) like this: first we remove the edge loop, then we perform 1 wrong side, 2 crossed front and 2 wrong loops. We continue knitting according to this pattern to the end of the row. The fourth row (even) is performed as follows: remove the edge loop. Further we work according to the scheme: "1 front, 2 wrong crossed, 2 front loops." Then we alternate the knitting pattern for even and odd rows. As a result, you get a beautiful three-dimensional pattern. Good luck
