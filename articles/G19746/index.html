Technical Specifications of the Megafon Mobile Router

More and more innovations are offered to its subscribers by the leading domestic mobile operator called Megafon. A wireless router with support for fourth-generation networks is one of them. With its help, you can create a mobile wireless access point to the global web without any significant problems. The provider offers users several varieties of such equipment, but the principle of operation of any of the available routers is, by and large, the same. In this article, an example will be considered a mobile router model MR150-3.





Device purpose

The portable device considered within the framework of this material is intended only for connection to the Megafon mobile network. The router model MR150-3 can work with any modern cellular communications from the second to the fourth generation . With their help, data and information are exchanged with the global web. The local wireless network is implemented using Wi-Fi technology . Up to 8 devices can be connected to this transmitter. The result is a mobile point that has a long battery life and allows in almost any case access to the Internet. This solution is very convenient in cases where a person travels a lot, goes on business trips. This is the main scope of its use.





Design. Display and control system

On the top plastic cover of black color MR150-3 there is a logo of the Megafon company. The router also has a group of indicators in the same place. These include battery power, wireless signal quality, and information regarding received messages. There is also a power button. The back side is closed by a removable cover. Under it there is a battery and a slot for installing a SIM card operator. Since the software shell of this solution is sharpened only for MegaFon numbers, it is unacceptable to install SIM cards of other operators in this case and they will not work in such equipment.

Equipment

Modest equipment in this mobile solution. But, again, in most cases it will be enough to start using such a router immediately after purchase. It includes the following:

  • Actually the router itself.

  • Interface cord with USB format ports.

  • User guide in brief.

  • Warranty card.

As noted earlier, everything you need in the delivery list is for the first launch of such a solution as a Megafon router. 4G - signal in some cases, requires an external antenna, for which in this case there is a necessary socket. Only the antenna itself is not in the supply. It will have to be purchased separately. This is the main disadvantage of this configuration.





Key technical specifications

As noted earlier, the Megafon mobile router Model MR150-3 n is left in a black plastic case. Its dimensions are 97x58 with a thickness of 13 mm. The weight of the mobile router is only 75 grams. The capacity of the removable battery is 1500 mAh. One of its charges, according to the manufacturer, is enough for 8-10 hours of active use.

Communication kit

The main wireless connection method in this case is WiFi. Router Megaphone Model MR150-3 is capable of transmitting information in the 802.11 standard version b, g and, of course, n. In this case, 2.4 GHz is supported. But the most advanced sub-standard of this connection method is 802.11 rev. ac this solution no longer supports. Like the frequency of 5 GHz. The transmission rate ceiling in this case corresponds to 60 0 Mbit / s in peak mode. This router has 2 wired ports. One of them is micro-USB, which is used to charge the battery or configure this router. The second activation in cases where switching with an external antenna is implemented. As noted earlier, this component is purchased separately at an additional cost.





List of supported networks

An excellent list of supported cellular networks boasts such a WiFi router from Megafon . Its transmitter is universal. It supports all current networks. These include:

  • Standard 2 G, or, as it is sometimes labeled, GPRS / EDGE. In this case, support for the frequencies 850/900 MHz and 1800/1900 MHz is declared. The speed of information exchange in peak mode can reach the maximum possible 500 kbit / s in this situation.

  • 3G networks are supported in UMTS format with a frequency of 900 MHz or 2100 MHz. The speed of receiving and sending in the presence of such coverage can reach 42.2 Mbit / s.

  • 4th generation cellular communication in the LTE standard is supported at frequencies of 800, 1800 and 2600 MHz. If the equipment of the Megafon company allows, then the peak speed in this case will reach 150 Mbps.





Setting procedure

Maximally simplified the process of configuring the Megafon mobile solution in question. Router setup reduced to the following stages:

  1. Removing it from branded packaging.

  2. Removing the back cover and battery.

  3. We install a SIM card in the slot.

  4. We return the battery, put the back cover in place.

  5. We turn on the device by pressing a single control button on the top cover of the router.

  6. After initialization is complete, the device is ready for use.

The default wireless network name and password for access to it are located on the bottom cover of the device. The only caveat to this is that the PIN code must be disabled on the SIM card. Otherwise, you need to connect the router to the computer and in the settings of the first you need to set this parameter using the browser and the network address (it is 192.168.1.8 in this case). It is with the help of the login and password that by default you can connect any device to such a wireless network, starting with a smartphone or tablet and ending with a laptop and a multifunctional device. If necessary, the network name and even the password for access can be changed. For this, the solution in question must be directly connected to a personal computer or laptop using a complete interface cord. Next, launch any browser and using the previously given address, go to the settings. Here we adjust the necessary parameters and save the changes. Exit the configuration interface.

Reviews Pros and main disadvantages

There are essentially no shortcomings in such a device as a Megafon router. 4G coverage is supported, like all other current modifications of cellular networks. And both 3G and GSM. The removable battery capacity provides up to ten hours of continuous use. The setup procedure is as simple as possible and comes down to a minimum number of actions. Moreover, the cost of this device is very, very affordable. There is even the possibility of connecting an external antenna if necessary. But still there are two drawbacks to this decision from Megafon. The Internet router of this model comes in a plastic case and it can be easily damaged if it falls. That is, it must be used carefully and strive to avoid possible falls. The second minus is that the external antenna is purchased separately. But it is not always needed and it makes little sense to include it in the basic configuration.





Summary

The router model MR150-3 allows you to create high-speed mobile access points to the global web based on high-quality coverage of Megafon. The router has excellent technical characteristics, and its cost is much lower than competitors. All this makes it an ideal tool for creating such mobile local area networks.
