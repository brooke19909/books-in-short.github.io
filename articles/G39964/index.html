Nokia 225: specifications and features

Phone management and communications

The standard telephone keypad allows you to control all functions with maximum convenience, it is clear and simple. Nokia 225 features convenient SIM card management features. From this phone you can send MMS to any numbers and from both SIM cards. To start sending, just a couple of keystrokes. Using the mobile Internet is possible using the standard pre-installed browser Xpress, which speeds up page loading.
