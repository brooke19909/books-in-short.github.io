Router TP-LINK Archer C20: reviews, description

Now on sale there are many gadgets that work with the Wi-Fi network. They are able to transmit data at a certain speed, which allows you to comfortably surf the Internet. In order to connect to the network, you must purchase a router that supports the new standard.

This article will provide an overview of the TP-LINK Archer C20 home router . It transmits data at a speed of 733 megabits per second. Its cost is small - only about 3 thousand rubles. Consider what the device is, what advantages it has and what can surprise the user.



Description

The described model is the youngest and most budgetary in the lineup from the manufacturer. What is the difference between the device and more expensive options?

It does not have gigabit Ethernet ports and the option of transmitting HD-video. In all other options, the TP-LINK Archer C20 router is not inferior to anything even more expensive counterparts. This unit will be useful for those who can use the Internet channel, the width of which is 100 Mbit, also for those people who use several wireless devices. Thanks to this router, you can establish high-speed communication between them.

Under the glossy glossy cover of the router is a 20 dBm power transmitter, a Wi-Fi module, and an Internet switch. Outside, you can see 1 WAN port, 4 LAN ports, a button for making settings, as well as resetting parameters. The device has 2 fixed antennas.

The capacity, which was declared by the manufacturer, will be enough to provide communication with an apartment for 2-4 rooms, a small office or a house. The main thing is to correctly configure the TP-LINK Archer C20. On the bottom of the router you can see a special sticker. It contains all the information for entering the admin panel, as well as the name of the device itself and the version of installed software. It also shows the serial number, password from Wi-Fi (factory), poppy address and name of the wireless connection. The device cannot be attached to the wall, since there are no holes for fastening.

The complete set of the device includes a power adapter, a patch cord , a mini-disk with installed software.



Setup and management

The device can be prepared for work using three different methods.

  • Traditionally, the configuration is carried out through the admin panel. All data in order to get to this menu is indicated on the label, which is located on the bottom of the device.
  • Using the "Quick Setup Wizard". This program is available on the disk that comes with the package.

In the admin panel, you can see that all parameters are divided into subgroups. On the right side of the screen is additional. information (reference), which describes the characteristics of each element.

If the user is a beginner, then he can use the quick setup mode TP-LINK Archer C20, Internet and wireless. Difficulties will not arise, since each action has a hint. If the user is experienced, then he can configure all sections and parameters himself.

In addition, there is special additional reference information that you can always use. If you want to practice creating your own network in order to prevent mistakes, you can go to the website of the developers of the router and use a free simulator.

Help is also available here, which allows you to understand how to connect the device, install a print server. The reviews for the TP-LINK Archer C20 say that this method is as convenient as possible. It works offline and does not require much user intervention.

The Tp-Link Tether mobile utility is not included, but it can be easily installed from Google Play. Router Connection

Connecting this router is performed according to the usual and familiar scheme. First you need to connect it to the mains and turn on using the key. The successful process will be notified by a lit indicator on the front panel. Further to the Internet port, that is, to the Internet connector, you need to connect the cable from the provider that provides the network.

If you want to make router settings on a network cable by connecting it to a computer or laptop, then you can use it. It comes with the kit. With it, you need to connect the computer to the router. In the settings you need to go to the ‚ÄúNetwork‚Äù tab, then select the WAN menu. Here, the first thing to do is select the type of connection. As we said above, this is often a dynamic IP.
