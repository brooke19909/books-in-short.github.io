How to disable voice mail on Megafon? Step-by-step instruction

When the subscriber cannot receive the call, voice mail starts to work. The MegaFon mobile communications company with the Voice Mail service will allow customers to stay informed about missed calls, and it will be possible to listen to messages from other subscribers who could not get through. The service works independently and is activated when the number falls outside the coverage area, or if the phone is simply disconnected. Not all customers need such a service, so you need to know how to turn off voicemail on MegaFon.

Option Description

Before disabling voicemail on MegaFon, you should understand its features and capabilities. The service starts to work if one of three situations occurs:

  • Mobile device is disconnected.
  • The subscriber number is in a place where there is no network coverage.
  • User number is busy.




If there is at least one of these situations, then voicemail (MegaFon) will be included. According to the standard conditions of call forwarding, it has code 62. This is when the client is out of coverage or the device is disconnected. If necessary, users can independently change the parameters and conditions of service activation:

  • Set unconditional call forwarding by code 21. In this case, the option will forward all calls in the incoming direction, in addition, the client number can work.
  • Set call forwarding if there is no answer to the call (setting code 61). In this case, if the subscriber does not answer the call within 30 seconds, the call will be directed to voicemail.
  • Set the busy parameter by code 67. In this case, calls will be directed to voicemail when the subscriber's number is busy.

Disconnect via mobile phone

If there is no need to use the service, then you need to know how to turn off voicemail via a mobile device on MegaFon:

  • Clients can dial * 845 * 0 # from a mobile phone and press the call button to send it to the network. After disconnection, the user receives a confirmation notification.
  • The second method is the use of a special interactive menu "MegaFon". To call it, you need to enter the command * 105 # in the gadget and make a call. After deactivating voicemail, a message with confirmation information is received on the mobile phone.
