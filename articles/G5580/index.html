Washing in LG E1296ND3

Before loading the laundry into the washing machine, it is necessary to sort it. After that, you need to open the powder box and put detergent there. Now you need to turn on the LG E1296ND3 washing machine. After that, a specific washing mode is selected, and then additional options are set depending on what type of laundry was loaded (optional). Now you should start the process itself by pressing the start button.
