Smartphone user guide for beginners: basic functions

Desktop

On smartphones, the user can create many desktops for comfortable calling various applications and widgets. They look like a regular desktop computer, which also has shortcuts to programs and games. To control the desktops, you must simultaneously touch the two fingers of the screen and begin to bring them together. In this menu you can add a widget or change the wallpaper. If you need to add a new field, click on the "+" button. To remove the desktop, click on it and hold it, after a few seconds you can move it to the trash.

The user himself can configure the location of the icons of any applications and widgets on the desktop. To do this, click on the program or game that you want to transfer, and wait a few seconds until the shortcut changes color. Next, simply move the application with your finger to another place. To remove the program shortcut, you need to drag it to the very top of the work surface, where there will be an inscription ‚ÄúDelete‚Äù.





Many programs have widgets that work in the background and provide some information without starting the utility itself. These are quite useful gadgets. To add a widget, simply press and hold your finger on an empty spot on the screen. After a few seconds, in the new window, click on the "Widget" button. We select the necessary one, transfer it to the desktop and configure it by following the steps that the application requires.

You can add your contacts to the desktop, for this you need to hold your finger in an empty place on the screen. In the window that appears, select the "Contact" and select the desired one. You can quickly navigate in them if a picture is installed.

The instruction manual for the smartphone for beginners (LG, HTC, any other phone) is almost the same for any of them. These devices have the Android operating system, which makes them almost identical in operation.

Menu

The instruction manual for the smartphone for beginners (Samsung is not an innovator in this regard) no longer provides for the possibility of using the menu. All necessary applications are contained on the desktop. But there are phones in which this function remains. The menu contains all installed applications. They can be added to the desktop or deleted. To add a shortcut to a place convenient for you, just click on it and after a few seconds transfer it to another.

Notification screen

The notification screen contains basic functions that can be quickly turned on or off. The most common are Wi-Fi, sound, Bluetooth, GPS, auto-rotate screen, Airplane mode, and others. The instruction manual for the smartphone for beginners (Lenovo, Fly or Samsung - no difference) shows us the different functionality on the notification panel. It depends on the Android OS version and the manufacturer.





Smartphone user guide for beginners: basic settings

the Internet

To use the Internet, you need to go to the notification screen and turn on either Wi-Fi or Data Transfer. In the early days, it is better to turn on Wi-Fi, since the entire configuration of the smartphone is usually performed via the Internet. Also, many installed applications can begin to be updated, and this will consume a lot of traffic and time. To turn on the Internet, go to the settings and click on Wi-Fi. After that, turn it on and select the appropriate network. It remains only to write a password and click on the "Connect" button. Before including the ‚ÄúData transfer‚Äù item, you need to make sure that your tariff plan provides for the inexpensive use of the Internet. After that, you should receive an SMS with the Internet settings, save it.





Google play market

Play Market is an app store. In it you can find all the popular games and programs that are installed in a few clicks. All applications that Play Market presents have been tested for viruses and are completely safe. To use this service you must register. First, go to the Play Market and click on the "Register" or "Sign in" button if you already have a Google account.
