Checking the rest of the traffic through a USSD request

You can learn about the status of the Internet package by typing a command from a mobile phone: * 122 #. The system will display the number of available kilobytes of Internet traffic, the validity period of the current package until the monthly subscription fee has been charged, and information about the state of Beeline additional traffic in Kazakhstan.

If the service is unavailable, you can use the command: * 106 #. This method allows you to quickly obtain information about the state of Internet traffic from any mobile device and does not require access to the Network. For the correct display of information before entering the USSD-command, you must complete the current Internet session in the settings of the device.
