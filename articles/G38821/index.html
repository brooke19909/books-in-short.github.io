How to enable iMessage on iPhone 5?

Product developers for Apple devices are always striving to somehow improve and make it easier for users to use their proprietary gadgets. So there are various useful applications and options. For example, iMessage. How to enable this option? What it is? Why is it necessary and is it necessary at all? There are answers to all these questions.

What it is

Before you enable iMessage, you need to figure out what, in fact, is this application. First of all, this is a useful option that allows owners of Apple-technology to communicate with each other for free. Any user of a new generation iPad, iPhone, iPod can send text messages, photos, videos to the interlocutor. Naturally, the application only works in the "home" company - Apple. Any other company simply does not have the ability to connect to the dialogue in iMessage. How to enable this application? More about this later.

Activation

Many have already appreciated all the charms of such a useful application as iMessage. How to enable it on the device? Yes, there is nothing easier! You need to go to "Settings", in the "Messages" section. The right side menu will say iMessage. How to enable this option? You just need to drag the toggle switch sideways so that it changes color to green. All! The application is activated, you can configure it for yourself. Attention! To activate, there must be an Internet connection (any: at least 3G, at least Wi-Fi). Everything, after that you can use the application anytime, anywhere. The latter is especially true. Especially for those who travel a lot.
