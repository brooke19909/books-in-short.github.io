What is the difference between a smartphone and a communicator?

It may be difficult for an average user to distinguish a smartphone from a traditional mobile phone. And if we talk about the difference between a communicator and a smartphone, then new difficulties appear, since these devices are very similar. Let's see what the problem is. This is due to the fact that the smartphone is a mobile phone that simply grew to a handheld computer, while the communicator, on the contrary, is a PDA in which the telephone module appeared. If you are interested in knowing how the communicator differs from a smartphone, then we can say that they can be distinguished in appearance. Usually they inherit the features of the devices from which they originated.

The communicator is essentially a handheld computer. Usually he has a large touch screen, there is a full QWERTY-keyboard.
