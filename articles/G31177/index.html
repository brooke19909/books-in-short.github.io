Call connection

Well, the first connection option is a call to the operator. Perhaps the simplest, but not very common method. You need to dial 0890 on your mobile device and click on the call button.

As soon as they answer you, let us know that you would like to activate the MTS Locator service. You will be asked who you want to track. What is the name and number of the subscriber. An application for confirmation of actions will be sent to him. If he approves of her, you can easily and simply follow the interlocutor. In principle, the work with the operator can be completed on this. Wait for SMS alerts with the result of the request.

Message

Another option that will help answer how to connect Latitude to MTS is to use a special request. You must form a message and send it to 6677 short number. What exactly needs to be written?





In the text of the message type the name of the subscriber, and after the space - his number. That is how it will be displayed on the map. We send a message and wait for a response. Your interlocutor will receive an SMS with a confirmation code. It must be sent to number 6677. Only after that you can see the user's location at any time. Please note that all calls to 6677 are free within your home region.

Pause function

But how to refuse this service? "MTS Locator" reviews in this sense earns very good. After all, the company has several alternative approaches. For example, you may not completely refuse the service, but only temporarily suspend all requests.

How to do it? An SMS request will come to the rescue. As in the previous case, it is necessary to form a special message. In the text write "PACK STOP". Make sure that all letters in the message are uppercase. It is important. Otherwise, the system may not recognize the request. Now we send SMS to the already familiar number 6677. That's all, it's done. You can resume working with the program immediately. It is enough to send a request to display the subscriber on the map.
