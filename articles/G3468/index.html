"Wave peacock tail" knitting. Schemes and Description

This style has many names. Someone calls it "waves," and someone calls it a "fan." But the most common name is Peacock Tail.

Now we will present the scheme and description of the openwork peacock tail with knitting needles.

This pattern has many details, which include:

  • Front loops. They are located at the edges of the product, as well as in some internal parts.
  • Wrong loops. In one ‚Äúwave‚Äù there are 3 wrong loops, which are located through 1 row.
  • Double and double front loops. Together they form one row - 3 double at the edges and 6 double in the center. There should be one front loop between each loop. Each row is located after the wrong one.

Before learning the patterns and descriptions of the peacock tail with knitting needles, you need to improve your technical skills. This is a very complex pattern, and therefore it is necessary to constantly practice in creating loops. If you are a beginner needlewoman, then practice creating the wrong side and front knit.

You can learn how to knit inclined rows. The loop tilted to the left is tied through the back wall, and the one that tilts to the right through the front.

The training scheme will help you apply beautiful patterns to various elements of the wardrobe.

"Peacock tail" can be used in the following types of clothing:

  • summer dresses and skirts;
  • T-shirts and tops;
  • sweaters.

Now consider two different descriptions and patterns of the peacock tail with knitting needles.

Lightweight peacock tail

This method will give clothes elegance and lightness. Great for creating shawls. All you need is ordinary knitting needles, as well as knitting threads.

The total number of connected loops should be equal to 17, excluding edge loops. One wave should be connected in 4 rows. Observe just such a quantity.

Instruction:

  1. We knit 17 loops, and also do not forget about 2 hem.
  2. The first row consists of only facial loops.
  3. Edge loops should have a different arrangement. The first should be fixed on the right knitting needle, and the second is knitted with the wrong loop.
  4. The second row consists of purl loops.
  5. We knit the third, which consists of 3 identical cycles: the first loop is tied, the other two are knitted together with the wrong one.
  6. We make a series of five cap and front loops.
  7. We knit the fourth row only on the wrong side.
  8. The fifth row is repeated similarly to the fourth.

Wave Peacock Tail Pattern

The second method is not as "openwork" as the previous one, but no worse. It also looks great and is very easy to perform.

It uses not 17, but 18 loops. But in this pattern, 2 rapports are used, which means their number increases to 36.

In addition to them, there are two edge loops.

Instruction:

  1. The first four rows need to be knitted with satin stitches from the front and back loops. The first and third are facial, the second and fourth are wrong.
  2. In the fifth row, you need to observe the exact number of loops. Work at this stage needs to be done as carefully as possible.
  3. We knit six double crochet stitches.
  4. We carry out the sixth row. All its parts, including the flare, consist of purl loops that fit on the front side.

Having tied the first six rows, they received the first rapport. Knit a few more, and get a gorgeous pattern.
