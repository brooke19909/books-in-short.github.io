Radio wave and wire

Rarely does anyone have a mobile phone now. Even more: for many models work simultaneously with several SIM cards of various mobile operators. The convenience of wireless solutions is obvious - you can call from any place where there is coverage of base stations. However, a landline telephone is still in use in some cases. For example, in order to save money on the cost of a call. Imagine that you need to call from a wired phone to a mobile number. It is expensive. However, if instead of a conventional device, a hybrid model is installed at home (the so-called GSM gateway), then all problems will be solved.





The usual call scheme is as follows: landline telephone - telecommunication company of wire communication services - mobile operator - "mobile phone".

If it were possible to remove the second link from the chain, then the tariffing would be completely different.

This is exactly what GSM gateways do. Inside such a landline phone is a standard cellular module and a SIM card. Depending on the number to be dialed, a particular call method is selected.

As a result, the connection is first made to the wireless unit of the gateway located at home, which dials the required city number. Thus, the scheme takes the form: cellular call initiator - wireless gateway unit - wired module - classic wired call.
