Elastic thread: how to sew it correctly?

Probably, some of the lovers of needlework have already tried to work with an elastic thread, which is often called spandex or rubber thread. It is possible that the first experience was not successful, so today we want to tell you about some secrets of working with this interesting material.



How to sew a gum thread is of interest to many lovers of sewing. Indeed, with its help it is possible to obtain uniform, elastic, stretchable assemblies that fit tightly to the body in the finished product, but do not constrain movements. Such assemblies can be used on the neckline, on the waist line, on the bottom of the sleeve or to completely stitch the whole part (for example, a sundress bodice).

So, today we will talk about how to sew with a rubber band, in other words, about using it as one of the threads that will be refilled in the machine. To obtain an elastic assembly, it is used as the lower one.

Instead of the usual thread, the elastic band should be wound on the bobbin . How to sew on a typewriter? You will learn more about this later. You can wind it on a bobbin manually or automatically. There is no difference. It is not worth it to stretch the gum specifically, it will be wound with some stretching. But this is not so important. The main stretching of the thread will occur when feeding it from the bobbin. Here it is necessary to dwell on this point in more detail.



It is necessary to correctly adjust the elastic tension. This can be achieved with the screw on the bobbin case. The thread should come from it not tight and not weak, but with some effort. Pull the tip of the thread - it should stretch first, and then start to reel off the bobbin. Using the screw, you can get different tensile, and therefore different degrees of assembly.

So, now the elastic thread is correctly and neatly wound around the bobbin. How to sew the necessary part and what to use for the top line? From above we use an ordinary thread, but preferably more durable than cotton. It is better to use reinforced thread. The fact is that during operation of the product the line will experience significant loads. The upper thread should be slightly loosened so that the weave is on the wrong side of the fabric. Set the maximum stitch length.



Is the gum thread set correctly? How to sew so that the assembly is uniform? You will be able to understand this using a prototype made exactly from the fabric from which the product is sewn. Stitching is performed on the front side. Its quality must be evaluated precisely from the front side of the fabric. If the assembly is not very uniform, adjust the stitch length and thread tension.

After completing the first line, you may get the impression that the line is a bit weak. In order to get a stable assembly, you need to make at least 3-4 lines.

Today you learned about what a gum thread is, how to sew using this original and very useful material for every good housewife. We hope that the knowledge gained will help you create interesting models with assemblies or buffers.
