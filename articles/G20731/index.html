LG 42LF652V TV: owners reviews, review, specifications

In this article, we consider the LG 42LF652V TV: reviews about it, features and advantages with disadvantages.

Specifications

This unit is equipped with a LED display. Its screen size is 42 inches. Resolution 1920 by 1080 pixels. It should also be said about the refresh rate. It is 100 hertz. There is a system of "Cinema 3D."





Such a feature as a curved screen is missing. Built-in backlight type Direct. Image display index at 550 PMI. Supported Smart TV function. Built-in Wi-Fi module.

An optional universal remote control is not supplied, but it can be purchased separately. Built-in 3 HDMI connectors and the same number of USB ports. There is a speaker system version 2.0.

The TV is part of the manufacturer's model line in 2015. Several broadcasting systems are integrated: satellite, cable, terrestrial. Processor type Triple XD.

Among the image modes, 8 built-in options should be noted. Among them there is the option "Bright", "Cinema", "Sports" and so on. Screen proportions also work in 8 modes. HEVC is supported with a score of 2K 60p. There are several decoders. The acoustics system is built in such a way that it has one lane and two speakers.

Continuing the review of LG 42LF652V it should be said that there is no subwoofer. Sound modes built-in 6. Smart Sound option supported. Moreover, there is the possibility of wireless and optical sound synchronization.

Disadvantages: reviews

The device does not have critical flaws. LG 42LF652V TV features excellent. In practice, what the manufacturer is talking about is fully confirmed. But not everything is so smooth.

It should be noted that of the most urgent complaints of buyers, it should be noted that two pairs of 3D glasses are supplied in the kit, although it is stated 3. Also, the kit includes an ordinary remote control that is not registered in the device's features.

It is also noted that in order to watch movies on TV in normal quality, you must have high-speed Internet. Some people have a marriage on the TV, in which the 3D mode does not work. However, this problem is not common, so we can assume that this is a developer error that has already been fixed. Reviews of the LG 42LF652V confirm this.

These shortcomings are not serious. Most of the buyers do not find any problems, which means this equipment is really convenient and in demand. You should pay attention to it when a person will make a purchase and choose this type of TV.
