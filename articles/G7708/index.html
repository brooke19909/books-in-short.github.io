Android Pay electronic payment system in Russia. How to use Android Pay

Android Pay in Russia appeared recently. But now this system has won the hearts of many. We have to deal with all its features. What is Android Pay? And how to use it?

Description

Android Pay in Russia is nothing more than a service for making contactless payments. It works easily and simply. A citizen binds a bank card to a mobile device through a special application. After that, a person has the opportunity to pay for goods and services using the phone.



The main thing is that the Android operating system is installed on the device. Otherwise, the reception will not work. In fact, we are dealing with an application that replaces bank plastic. But what should every modern resident know about Android Pay in Russia?

Installation

Now consider the process of working with the application. How to use Android Pay? In fact, everything is simpler than it seems. The hardest part is linking a bank card. This will be discussed later.

First, install the appropriate application. It is done like this:

  1. Open Google Play.
  2. Pass authorization in the application.
  3. Type in search Android Pay.
  4. Select the proposed result.
  5. Download the corresponding application.
  6. Run the received document.
  7. Follow the on-screen instructions to complete initialization.

That's all. Android Pay electronic payment system is sometimes tried to fake. It is worth paying attention to the developer of the program. The original application will have the signature "Google Inc." It is advisable to avoid downloading the utility from third-party sites.



Customization

At the above steps, the preparation for the use of contactless payments via phones is not downloaded. Now that Android Pay is installed on your smartphone, you need to properly configure the program.

The setup guide will look something like this:

  1. Launch Android Pay.
  2. In the menu that appears, enter the bank card details (validity period, number, CVV).
  3. Provide information about the owner of the plastic.
  4. Enter the address of the place of residence.
  5. Prescribe a mobile phone number.
  6. Enter the operation confirmation code. He will come to mobile in the form of SMS.
  7. Click on the "Allow" button. It will automatically be displayed after entering the code to add a card.

It is done. Now you can use Android Pay. Sberbank, like any of the banks listed above, will allow you to pay by phone for purchases in stores or online after completing the steps.

Other cards

Android Pay has support for additional cards, discount plastics and gift certificates. They can be added to the program menu if desired. Then there will be no need to always carry the listed cards with you.

How to cope with the task? You will need to follow the manual of the following type:

  1. Open Android Pay.
  2. Press the round plus button.
  3. Select the type of plastic to be added.
  4. Scan a barcode or manually dial a card number.
  5. Save changes.

Quick, easy, convenient. In reality, everything is much simpler than it seems. The studied option is not supported by all banks. Android Pay appeared in Russia in early 2017. And therefore, so far the list of financial organizations is limited. But discount and gift cards can be added to the program any. About use

We got acquainted with the main stages of preparation for using Android Pei. Now what?

You can pay using your mobile device. To do this, you need only 2 actions. Namely:

  1. Unlock phone.
  2. Bring the smartphone to the payment terminal.

In just a few seconds, data will be read from a mobile phone. Money will be debited from the added bank plastic. Everything is very simple.

Multiple cards

What should I do if a user has several cards added to the Android Pay program? What plastic will be debited from?

This parameter is set by the user. When linking several cards to Android Pay, you will have to open the application and select the necessary plastic there. It will not work otherwise.



Not all stores

It is also important to remember that Android Pay in Russia does not work in all retail outlets. Although theoretically, the application should function equally well in every store.

Firstly, the studied opportunity cannot be implemented in retail outlets without the support of bank transfer.

Secondly, the reader must have a chip for contactless payments. Fortunately, almost all modern non-cash machines are equipped with this part.

It is advisable to look at the entrance to the store (or at the checkout) support for "Android Pay". Then the user will be able to accurately answer whether he will be able to use the studied program or not.

Watch and Android Pay

Surprisingly, modern citizens can make contactless payments using special watches. They are manufactured by mobile phone manufacturers. Not all gadget models have support for Android Pay.

Today, this option is available on Huawei Watch 2 and on AlGi Watch Sport. How to act in such circumstances?

There is no difference between the phone and the clock in our case. The user will have to download Android Pay for the gadget, install the program and attach the plastic to the device. To pay with the help of the listed devices, simply turn them on and bring them to the reader.
