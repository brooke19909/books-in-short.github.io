PSP 3008: reviews

What fate awaited the new portable game console from Sony? As mentioned above, the gaming community took the PSP 3008 well. The reviews about the console are mostly positive. Users praised the console‚Äôs high performance, portability, amazing ergonomics, interesting design, the availability of useful services (the same Network PS) and a huge number of high-quality and exclusive games.





However, there were also disadvantages. Perhaps the main problem with the PSP 3008 was the screen. As you can read above, a new technology was used to create the display. Its essence was the use of the so-called interlaced scanning method. In accordance with this technology, each frame is divided into two parts, which are assembled from rows selected through one. This allowed you to display the image with the PSP directly to the TVs. And it sounds pretty promising. However, in practice, everything turned out to be much more complicated. Due to this technology, a comb effect arose during the game. Objects that move horizontally began to grow in halos, which made the picture very soapy. It was not possible to get rid of this bug by releasing new software, because the problem was in the hardware. However, Sony later released an update in which it was possible to disable interlaced scanning.
