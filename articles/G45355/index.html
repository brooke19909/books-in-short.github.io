How to create a Windows Live ID?

Windows Live ID is an account through which contacts and work tools are synchronized on your phone. It is also needed to download games and programs from MarketPlace, because it is used for single sign-on to all Microsoft network services.

If you own a Windows Phone smartphone, you just need to create a Windows Live ID. In addition, when you decide to change your phone to a new model, this account will help you restore your contacts and data.





It is worth noting that, starting with Windows 8, you can use the account on the official Microsoft page to log into the account of the above service. Windows Live ID authentication includes email and password verification.

How to do it?

So, there are two ways to create a Windows Live ID on HTC, Xiaomi and other smartphones:

  1. From a personal computer, if there is a continuous Internet connection. You can create an account on an existing subscriber email account. You can still register a completely new mail, which will be your account in this service.
  2. When you turn on the phone for the first time on Windows. To create an account using this method, you need a good Internet connection (via mobile network or Wi-Fi).

Both of these methods (how to create a Windows Live ID on Samsung, Nokia and so on) are almost identical. The differences are only the registration of mailboxes. As already noted, you can create an account for both an existing one and a completely new e-mail.





Register for an existing e-mail

To create a Windows Live ID, go to the Microsoft website to register your own account and fill in all the empty fields. It takes literally five minutes of your time. Mandatory age is required (the user must be over 18 years old), since subsequently there may be problems with downloading various content from the application store (more precisely, you need to configure the "Parental Control").

You must carefully fill out all the fields, and most importantly - correctly indicate your email account and phone number. You will need them if you want to regain access to your account in case of hacking or losing passwords. It is important not only to create a Windows Live ID, but also to maintain access to the functionality in the future.

After filling in all the fields you need to pass the verification of the email address you specified. This is to verify that you actually own this mailbox. You will receive a letter. To complete the setup of your Microsoft account, follow the instructions in the email.





Once you confirm your mailbox, the registration process in the service can be considered successfully completed. Now you only need to click "OK", as soon as the interface displays the page of the new personal account. After that, you get the opportunity to enter account information on your phone. This is done in the Settings menu, where you should select the Mail and Accounts item, after which you need to go to Add Account, selecting Create Windows Live ID.

Security and passwords

When creating an account (if you use a pre-existing email address) you should not write the same passwords. Otherwise, when hacking, the attacker will have access to both of these accounts.
