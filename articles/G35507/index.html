The smartphone is equipped with a 3.4-inch touch screen. The housing has jacks for connecting headphones and inserting microUSB. Screen resolution - 480x320 pixels.



Multimedia

The smartphone is equipped with two cameras - the main and front. Each of them performs a specific function. The main camera can be used to make video calls and shoot videos. Device advantages

Today, the market has a lot of budget smartphones. Why is it worth giving preference to the Megafon Login 2 device? We list its main advantages:

  1. Unique price. It is unlikely that you will be able to find a smartphone with a good filling for 1600-1800 rubles (without connecting the Internet XS option). This is a very good offer.

  3. The presence of a front camera. If you regularly use Skype, then you just can not do without it.
  4. Installing Android OS 4.2.2. This is a very modern version, but you can upgrade it if you wish.
  5. Support for 3G mobile Internet.

Smartphone flaws

Like any mobile device, this model has certain disadvantages. It is better to learn about them before buying a smartphone. So, we list its shortcomings:

  1. The device looks cheap. Although there are certainly those who find it elegant. The lid closes tightly enough. Therefore, efforts have to be made to open it. For girls with long nails, this is a real torture.
  2. Not enough internal memory. The box indicates 4 GB. But in reality, the size of the memory is several times smaller.
  3. Screen . It is small and not very high quality. The screen resolution is only 480x320 pixels. To watch movies and clips in HD-format, this will clearly not be enough.
  4. The quality of the main camera is poor. This can be attributed not to the minuses, but, most likely, to the features of budget smartphones.
  5. The device only works with a Megafon SIM card. Like all branded devices, this model is locked. Sims of other operators are blocked.
