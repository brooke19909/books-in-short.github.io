Setting procedure

A fairly simple procedure must be followed before using the 32 LG 32LJ600U TV. The reviews suggest that the switching and configuration algorithm in this case is much simpler than in the case of competing Tizen OS shells or Android OS. It consists of the following stages:

  1. Removing the multimedia center from the cardboard box with the rest of the delivery.

  2. Carefully lay the device with the screen on a flat surface.

  3. We remove special transport locks from it.

  4. We install complete supports. When installing them, we use only those fastenings that the manufacturer included in the delivery.

  5. At the next stage, we install the TV on the horizontal surface on which it is planned to operate the multimedia center in question. Again, a signal television cable should reach this point. There should also be a socket nearby. The third important condition is the availability of an Internet connection. That is, either a Wi-Fi signal should be present, or twisted pair should come up.

  6. After installing the multimedia center at the place of operation, it is necessary to perform full switching. First, we connect the power cord, with which the power supply of the device is implemented. Then we bring the signal cable to the antenna input. In the case of using a wired connection to the global web to the RJ-45 port, we bring in a twisted pair.

  7. Next, turn on the device, you must wait for the process of its initialization. Then you need to set the current date and time. You must also specify the territorial location of this device. If you skip this step, then not all the features of the Smart TV software option function will be used.

  8. Then you need to go into the settings menu and start the auto search for all available TV programs. Again, you need to set the type of input signal in the first query window (digital or analog, satellite, local broadcast or cable TV). After the end of this operation, we save the received list of television programs.

  9. Next, go to the main menu and select ‚ÄúNetwork‚Äù. At this point, we set the network connection parameters. If twisted pair and a dynamic IP address are used, then there is no need to make changes at all. When using a wireless transmitter, you must specify the network SSID and password. Well, when using a wired connection and a static IP address, you must manually specify the latter.

  10. Now you need to go to the LG app store and create a new entry. After that, you can install all the necessary applications. The multimedia center is now fully configured and ready to use.
