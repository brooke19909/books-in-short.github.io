Reliable watchman "Alligator", alarm, instruction

A modern car security guard is an electronic device with a large number of various service and security functions. New cars come from factory conveyors with a small set of security solutions. However, many car owners prefer to install the Alligator high-quality multi-functional security system in their car. Alarm, instruction, a set of excellent abilities of this "guard" allow you to gain confidence in the complete safety of his wheeled "horse".

General Description of the Alligator Alarm

There are a lot of system modifications; the manufacturing company is constantly improving its alarm functions, adding new features. A security electronic device capable of remotely starting a car engine, notifying the owner of the condition of the car using a key fob with a display, is the Alligator. All messages about what is happening at the moment with the machine are visible in a running line on a small LCD screen. The radio codes are improved, the protection of signals from interference associated with terrain is enhanced. These innovations guarantee the safety of the system from breaking electronic protection. The sound alert signal (in fact, the sound alarm) is valid in a radius of up to 2.5 km and beyond. There is a remote inclusion of lighting in the cabin, windows window control, starter device lock.

Keychain for remote control electronic "guard"

The Alligator alarm is controlled from a distance using five- and four-button remote controls. Commands for door locks, engine starting, switching active and passive protection modes, setting an alarm, clock, machine search functions - all this can be done remotely.



Thanks to the latest technology and two-way communication (auto-owner), any changes in the state of the car immediately become known to its owner through a signal arriving at the two-way remote control. The keychain display shows exactly where the disturbance occurred on the machine body. Alarm system controlled by the Alligator system, the instruction to which describes in detail all the features of the program, saves the car from theft and other extraneous interventions.

Self-timer for battery

Regular or simultaneous start of the engine for warming up can be done using a programmable timer. The system monitors the voltage in the battery and if it is discharged, it will start the engine itself. Simultaneous activation of two timers is possible: monitoring temperature and battery voltage. The necessary process will be launched at the right time for the proper functioning of the vehicle mechanisms. The Alligator alarm operating manual contains the most detailed descriptions of the system setup processes for easy and convenient operation.





Programmable Feature Settings

The car alarm functions distributed between the two menus 23 give the owner the opportunity to access comfortable use of the basic and additional services of the security system. The first menu supports regular constant functions, in the second (programmable) it is not recommended to set up functions on your own, a competent specialist will do everything you need without errors. Otherwise, serious malfunctions may occur. For reliable operation, the Alligator alarm installation instruction recommends that you make the settings in the second section of the menu only in specialized authorized centers.

Safe use of alarms. rules





1. It is dangerous to start the engine from a distance if the car is in a garage or other enclosed space (where there is no ventilation), as it can be poisoned by exhaust fumes.

2. It is dangerous to start the engine of a vehicle in gear from a distance, as this may result in personal injury or death.

3. It is dangerous to leave keychains transmitters in places accessible to outsiders and children.

4. It is dangerous to leave the doors to the room open next to the car parking if the device is remotely started, as exhaust fumes can harm the health of people inside.

5. It is dangerous to trust the installation of the Alligator protective system (alarm). The instruction requires a professional connection procedure.

6 It is dangerous for standard car systems to use low-impedance test devices (logic probes, test lamps), since multi-channel systems can be damaged. Professionals use only a high-quality digital multimeter.

7. It is dangerous during installation of the security system to keep the windows of the salons closed, since you can accidentally be locked in the car.

8. It is dangerous to install the central control unit near the electronic components of the car, as this affects the quality of the system, creating radio interference.

9. It is dangerous to place a siren in the engine compartment near very hot components and parts.

10. It is dangerous to start checking the Alligator security car system until the installation is completed, including connecting the trunk, hood, and parking brake switches .

The instructions for the user explain in detail all the points that need to be studied and clearly remembered so that the use of the Alligator system brings comfort.
