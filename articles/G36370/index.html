Phone "Nokia N73": characteristics, photos, reviews

One of the best communicators of 2006, which combined excellent technical specifications and allowed to solve an impressive list of various tasks, is the Nokia N73. This device continues to be used by many subscribers of mobile operators today. This will be discussed further.





Who was this smartphone aimed at?

At the time of its release, the N73 was the best device on the Symbian software platform. He had the most productive semiconductor chip and excellent technical specifications. All this made it possible to run on this gadget any existing software that was compiled specifically for such devices. Also, support for 3rd generation mobile networks allowed the large file sizes for those times to be downloaded to this smartphone very quickly. All of the previously listed allowed to attribute this device to the premium segment of communicators in 2006. That is, this ‚Äúsmart‚Äù phone was intended for the most demanding users. Now this software platform is outdated. As a result, software updates do not occur, and the phone has turned into a regular ‚Äúdialer‚Äù with data transfer support and two cameras installed.

  • AC-4 Charging Adapter from Nokia.

  • Branded external speaker system HS-23.

The first of them allowed to increase the amount of internal memory by several times, and the second kept the initial state of the device.





Design, usability and ergonomics

For the most part, the Nokia N73 case was made of plastic. The upper part of the case was occupied by a regular display (unlike most current mobile devices, it was not touch!). Under it were the control buttons on the Nokia N73. The top row was occupied by two function buttons, the purpose of which changed depending on which application was open on the display, and a joystick. The latter performed navigation functions and allowed to confirm the performed actions. The second row consisted of buttons to start and end a call. Below were the number keys, which were surrounded by additional function keys (for example, quick launch of the browser) around the perimeter. Above the screen, in addition to the name of the cell phone model and the logo of the manufacturer, there was a speaker. The front camera was located right there. On the top of the gadget was a loud speaker. On the opposite side of the device, all the wired interfaces of the device were displayed. Here was the POP-PORT port for connecting an external speaker system and the familiar round socket for connecting a charging adapter to a smartphone. On the right side of the communicator all control buttons are grouped. This is the regulation of the sound level, this is the control of the camera, this is the blocking of the device. On the back cover of the communicator was an unusual ‚Äúslider‚Äù that protected the main camera from possible damage. It was behind this non-standard component that the peephole of the main camera was located. A single LED was immediately displayed, which made it possible to get quite good photos even with a poor level of illumination.

CPU

Only one computing module was implemented in the central processor unit "Nokia N73". The characteristics of its technical specifications indicated that this is the only core based on the outdated ARM9 architecture. Its clock frequency was equal to 220 MHz. For current high-performance devices, these values ‚Äã‚Äãlook really modest. But in 2006, such hardware specifications for the normal functioning of the Symbian software platform were quite enough. Any application software on such a "filling" was launched without any problems at all. Now this operating system is not supported by the manufacturer itself and is not developing. It is impossible to install any other system software on this device. Therefore, the owners of H73 will have to be content with what is already on the gadget.





Communicator memory subsystem

The amount of RAM in this model of the communicator was equal to 64 MB. And the capacity of the integrated data warehouse was equal to 42 MB. As you can easily guess from the indicated values, this was clearly not enough for comfortable work on this device. Therefore, it was mandatory to install an additional external drive in this smartphone. A slot for installing mini-SD cards was present in the Nokia N73. The characteristics of this group of drives indicated a maximum capacity of 2 GB. It was such the largest amount of memory that could be inserted into the corresponding expansion slot of this mobile communicator.

Display

The diagonal of the non-touch display in this mobile gadget was 2.36 inches. Its resolution in height was 320 pixels, and in width - 240 pixels. The pixel density on its surface was 325 ppi. Even now, you can find entry-level smartphones that have fewer dots per inch. So from this position, this display still continues to be relevant. The number of displayed color shades was 262 thousand. The screen matrix was made using the TFT technology. Its main disadvantage is the distortion of colors when deviating from a right viewing angle. But in 2006 there were no worthy alternatives to this matrix. In any case, the "Nokia N73" screen, both from the position of parameters and from the position of quality, did not cause any complaints. And some of its parameters, as noted earlier in the text, are even better now than some budget entry-level devices.

Cameras

A 3.2 megapixel sensor was on the main camera in the Nokia N73. Photos taken with its use at that time had the best quality among similar devices. Also, in the list of functions of this camera, one can distinguish autofocus technology (in 2006 it was a rarity), digital 4x zoom and LED backlight (even now not every device can be found in it). All this in total even now allows you to get very good pictures on such a communicator. This camera allowed recording video in 352x288 format. By today's standards, this is a disgusting quality, but in 2006 something better could not be found. There was also a front camera in " Nokia N73 ". The photos taken already with its use were of much worse quality. This is not surprising and it had a sensor element of only 0.3 megapixels. Its capabilities were enough only for video calls. And the video quality in this case was far from ideal.

Battery and device autonomy

The battery on the Nokia N73 with a nominal capacity of 1100 mAh made it possible to work in standby mode for 14 days on a single charge. In reality, with an average load, one could count on 3-5 days. With a greater load, the time was reduced to 2 days. Well, with maximum savings on one charge, you could stretch even 7 days. Therefore, from the standpoint of autonomy, ‚ÄúH73‚Äù on the backgrounds of current gadgets is beyond competition. A key role was played by several factors at once, among which one can single out an energy-efficient processor, a small-sized display and a high degree of optimization of the software component of the communicator. The only remark that can be noted in this regard is the lack of such an accessory as an external battery for the Nokia N73. It would not be out of place during a long journey and would significantly increase the already considerable battery life of a mobile device.





Communicator operating system

The most recent Nokia N73 firmware was based on the Symbian operating system with serial number 9.1. Further updates to the system software did not occur. There is a simple explanation for this: newer models of communicators with updated software appeared on the market. It was on them that the manufacturer focused. But even this, albeit an outdated version of system software, is enough to run almost any application software on this device. It is impossible to change the operating system and thereby improve the functionality of the device. Therefore, the owners of this smartphone model will have to be content with what is already available. Separately, it should be noted that this operating system is no longer supported and is not updated by the manufacturer. Unfortunately, this software platform has no future. Although at one time (in 2006), the Finnish company and its operating system were legislators in the market for system software for mobile gadgets.

Application software

Initially, an impressive set of application software was installed on the Nokia N73. There is a built-in browser that allows you to view most web portals even now. If necessary, you can install the Opera Mini viewer. Its main advantage is data compression when receiving and sending. As a result, you can get significant traffic savings. Also, an e-mail client is integrated into the smartphone. You can connect any electronic mailbox to it and receive incoming correspondence from it. Also, this software tool allows you, if necessary and with an established connection to the Global Web, to send an e-mail at any time. For simple calculations on this device there is a calculator built into the OS with a basic set of functions. If necessary, you can expand the functionality of the device by installing an additional of the same product, but based on Java software. In principle, any software that is based on this software platform can theoretically function on this model of communicator. The limitation in this case may be the size of the executable file, which may exceed the capacity of the integrated data and information storage, that is, its maximum size should not exceed 42 MB. In reality, this value is even less: part of the memory is used by system processes. But, as practice shows, for application software platform "Symbian" this size of 42 MB is enough.

  • The device had only one slot for installing SIM cards. The device itself is equipped with transmitters that allow it to work only in cellular networks GSM (the second name is 2G) and UMTS (or 3G). In the latter case, the maximum allowable speed is 384 Kbps.
