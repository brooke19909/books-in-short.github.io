How to turn on your iPhone and start using it

So, you became the happy owner of a brand-new product of the famous "apple" company. How to turn on the iPhone and how to start using it? IPhones of all generations are included as standard - using the Power button located at the top right of the phone. The same button is responsible for locking and unlocking the phone.

But even after turning on, using an iPhone is most often impossible. In order for the device to begin to fully function, it must be activated. If you bought a phone from one of the major mobile operators such as MTS or Megafon, then it will most likely already be activated. But if not, then this operation can be done without problems.

Let's step by step how to turn on the iPhone and activate a new phone.

1. Take out your iPhone and cable for the PC from the box.

2. Download and install the special iTunes program on the computer.

3. Insert the SIM card into the phone, turn it on using the Power button, connect the phone to the computer via the cable that was included.

4. Most often, iTunes starts automatically. If not, run the program manually. She herself will offer to activate the device. We confirm the request.

5. We leave the iPhone alone and wait for the activation to occur. While there is no activation, the device displays a pending message. While the process is underway, you can get acquainted with the capabilities of the device that do not require communication, for example, with a camera, importing photos, menus, etc.

6. When activation ends, try calling. If everything is in order, the phone will dial the number. This means that the full functionality of your iPhone is turned on.

How to enable iPhone 4s

For this generation of iPhones, the activation process was greatly simplified - it became optional to use a computer.

How to turn on iPhone without using iTunes.

1. Insert the SIM card, turn on the phone using the "Power" button.

2. Choose a familiar interface language: Russian-Russia

3. The phone asks if you let him determine the location or not (this will be needed later for using cards, check-ins on Facebook, etc.). If these functions are needed, then we allow it.

4. Next, the iPhone will ask for a WiFi connection. In principle, this step is optional. It can be done later, at any time. We still need the Internet to send information to Apple about the successful activation of the device, and also for the operation of various online services such as iCloud and iTunes Match. For the same purposes, you can successfully use the mobile Internet of a local operator (3G / GPRS).

5. Now the iPhone will register in the network of the mobile operator, receive the necessary settings for MMS, SMS. If you previously had a different iPhone, they will offer to restore the saved previous data from the backup. And if the iPhone is new, then you need to select "Set up a new device." In general, after that the phone can already be used fully. Further steps are, in principle, optional, but highly desirable.

How to turn on iPhone if it is suspended

To do this, use the forced reboot mode: simultaneously press and hold the On button (at the top) and the Home button (the only round button under the screen) for a long time. After that, the phone goes off. If you turn it back on, it will reboot longer than usual, but the problems will automatically be fixed.
