How to connect?

Almost everyone knows how to connect the GS700 tablet to their receiver. Very colorful and clear instructions in the pictures are available on the GS Group official website. Internet and television setup instructions are also available for download.





To connect, just follow a few simple steps:

  1. You need to make sure that the GS E501 server (E502 and beyond), as well as the Wi-Fi router, are connected and configured in accordance with the manufacturer's recommended settings.
  2. On the tablet you need to install (or update, if already installed) "Play.Tricolor".
  3. Connect to a wireless network (its name in this case will correspond to the server name).
  4. The tablet will automatically ‚Äúpull up‚Äù all the settings. After that, you can run the TV viewer without hesitation.

Many who own programming do root. They say that it is very simple. But the bulk still hopes that the General Satellite should resolve this problem in the near future.
