Basic looping options

Choosing among the various methods of looping, you need to consider that there are basic options that can not be bypassed in any case. And there are few such options, there are only two of them. Set from a long thread

In order to start the set, it is necessary to leave a rather long part of the thread. It is on it that all the loops for subsequent work will be dialed. A set of loops can be done on one spoke, or you can use two at once. The fixation of the loops on the spoke occurs precisely at the end of the thread that is left in advance. The set can occur both from one skein of yarn, and from two, when the thread from one fixes the loop, and the main set comes from the second. Often there are cases when the left length of the thread is not enough to set the desired number of loops , and then there is no other way but to dissolve the loop and repeat again, leaving an even larger end of the thread.
