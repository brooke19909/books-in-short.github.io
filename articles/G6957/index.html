Enter BISS keys in the tuner "Openbox x800"

This is one of the most popular devices. Here we will consider the question of how to enter BISS keys into the Openbox tuner. The task will be solved by the example of the device model x800. Everything is quite simple here. This key entry is done manually. First you need to select the channel into which this key will be inserted. It is worth noting the plus of such a tuner - a menu in Russian. Thanks to this, the operations for entering the BISS keys will be many times easier and faster. So, when the channel is selected, on the tuner remote control you must press the "Menu" button. Then the numbers 1117 are dialed. When the data is entered, the user immediately opens a dialog menu in which it is necessary to select the column with the inscription "Byss" using the remote control.





After that, the user enters the next window, where on the right you can see the keys for video and audio. Under each item there is a list of keys. In order to add a new one, you need to press the green button on your remote control. Here we stop for a second and note that in the new key that the user has, it is imperative to delete the fourth and last pair of numbers. Otherwise, if this is not done, the tuner will not show. It is also necessary to take into account that the key is entered as much as two times - for video and for audio.

When the key is entered, you need to press the OK button twice on the remote control. By clicking once, the user saves the key, and the second - binds the key to the selected channel. Next you need to click the "Exit" button. If necessary, this procedure should be repeated with other channels. On this procedure is considered completed. So, we figured out how to enter BISS keys into the Openbox x800 tuner. When the work is completed, the tunable channel will immediately begin its broadcast. This method is suitable for all lines that this tuner broadcasts.

Entering keys in the tuner "Tiger"

The peculiarity of this device is that the entry of BISS keys can be done both through the tuner itself and through a personal computer. First, an example of data input in a standard way will be considered, and then through a PC.

How to enter BISS keys in the Tiger tuner in a standard way

As you can already guess, the input will be made using the tuner remote control. First you need to select the "Menu" button on the remote control. Next, the user must find the second item on top.

A small menu is displayed on the screen, which consists of three items. The user should select the latter. Having opened the next window, you need to click, as in the previous examples, the green button on your remote control to control the tuner. The next step is to enter the data below using the same remote.





The data is as follows:

  1. CAID - at this point you must enter four digits - 2600.
  2. The ProvID field is not populated.
  3. SID This is the ID of the television channel provider. You can recognize it by pressing the "Info" button three times while watching the necessary channel.
  4. The next item is PMT PID. It should also not be touched. He remains unchanged.
  5. ECM PI. At this point, you must enter 1FFF.
  6. EVEN CW and ODD CW. In these two points, you must enter the BISS key.

When all the data is specified, you should select the "OK" button on your remote control to control the tuner. After that, the broadcast of the channel will start automatically.

That‚Äôs all you can tell about how to enter BISS keys into the Tiger tuner. Next, we will consider a method of data input using a personal computer. It is no more complicated than the previous one.

Entering BISS keys using a personal computer

How to enter BISS keys into the Tiger tuner using a computer? Everything here is quite simple and clear. First you need to prepare the necessary constant.cw file. It can be easily downloaded on the Internet. To do this, you must enter the full name of the file and tuner in the search engine. In order to continue to use this file, you must edit it. To do this, you need the WordPAD program on your computer. Further, when the necessary programs are downloaded, you need to edit the file so that it looks like this: CAID ProvID SID PMTPID ECMPID EVENCw ODDCW. Now we upload everything to a flash drive. And she, in turn, is inserted into the tuner jack.





When the user inserts the media, he needs to go into a small menu and select the item called "Update via USB". In the menu that appears, select the last item. We will see a small window in which to enter the following data:

  1. CAID - in this paragraph you must enter only four digits - 2600.
  2. The ProvID field is not populated.
  3. SID This is the ID of the television channel provider. We recognize him by pressing the "Info" button three times while watching the necessary channel.
  4. The PMT PID item remains unchanged.
  5. ECM PI. At this point, you must enter 1FFF.
  6. EVEN CW and ODD CW. In these two points, you should enter this BISS key.

So, the answer to the question of how to enter the BISS key into the Tiger tuner is found. It is in this way that you can enter new data into the device in question using a conventional personal computer. It is up to the user to decide which method to use.
