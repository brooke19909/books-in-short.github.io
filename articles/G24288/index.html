How to make do-it-yourself shambhala

  1. Beads are strung on a cord sixty centimeters long in the sequence that you yourself determine. At the ends you need to make knots.
  2. The base should be fixed with tape on the table so that the cord hangs from it. At the same time, the lower end also needs to be fixed in order to make it easier to work on.
  3. The cord, which is used directly for weaving, must be folded in half so that the ends are the same length.
  4. At the end and beginning of the bracelet you need to tie it in a knot. Between the beads also make one knot. Do not be afraid to tighten them tighter, so the bracelet will turn out even more attractive. It is worth checking beforehand if the bead slides on an unfinished product, and only then cut the end of the thread.
  5. Now you can start making the clasp jewelry. To do this, you need to add a bead to the free ends of the cord, tie knots again and process with superglue to hold better.
