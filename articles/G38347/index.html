Unlock hidden features: custom firmware for PS3 and iPhone 3G

If you are the proud owner of any type of modern computing equipment, whether it be a PlayStation, iPhone or Android tablet, sooner or later you will encounter the fact that you will not be able to install some necessary applications (most often unlicensed ones) or restrictions on use. Then a wonderful thing will come to your aid - custom firmware. With it, you can configure your device as you see fit.



What is custom firmware?

This is a non-commercial software that is created by individual companies or groups of interested people for specific needs or user requests. Custom firmware does not apply to official support of this device. In such software versions, the maximum number of restrictions is specifically removed in order to more conveniently configure the device.

Custom firmware pros and cons

You can install such firmware exclusively manually, at your own peril and risk - in the service center they have the right to install only the licensed version of the software. The minus is the loss of warranty, and the plus is the free distribution of custom updates, so you will not spend a penny on replacing the firmware. But if something goes wrong, then it will be very difficult to restore the device to at least the previous state. So in extreme cases, you will have to contact the same service center where you can be asked for a considerable amount. Therefore, the replacement of the firmware must be approached with the mind and understanding of what you are doing.

Firmware for iPhone 3G



Custom firmware for iPhone 3G is installed in several stages:

1. To get started, launch iTunes on your computer. Now make sure that the custom firmware you have selected is already loaded on your computer‚Äôs hard drive.

2. Connect the iPhone to the PC using USB.

3. Click on the "Restore" button in iTunes, previously holding Shift.

4. Select the firmware file. After that, custom firmware will begin to be installed independently.

5. At the end of this process, the iPhone will reboot and report the successful completion of the installation.

As you can see, everything is very simple. There is only one, but very important feature: when downloading the firmware, make sure that its name contains the inscription "3G", because for different iPhone models there are different versions of the firmware, and they are not interchangeable.

PlayStation 3 firmware

Custom PS3 firmware is installed in approximately the same way as in the previous case:

1. First, select the device from which the installation will be carried out - any one connected to the PlayStation will do.

2. Now in the root folder of this device create the directory "ps3", and in it - "update".

3. Copy the firmware to this folder and rename it (firmware) to "ps3updat.pup", only then the system will be able to detect the update and install it correctly. After that, connect the media to the PS3.



4. Open the settings, and in them - update the system. In this menu there is the item "Update from the storage medium". After selecting it, specify the device on which the custom firmware is located.

5. The system will start the automatic installation, after which it will reboot.

Done, the firmware is installed. You can verify this by going to the "Settings" menu and selecting "System Information".

Installing custom firmware is not a very complicated process, although it requires certain basic knowledge about the operation of the computer and your device. But you need to do this very carefully, because everything can end badly for your favorite device. Therefore, use only verified firmware versions with a large number of positive reviews.
