Prestige Navigator 5056. Instructions for use. Installation and use

Prestigio Geovision 5056 is a car navigator that is perfect for those users who are looking for an inexpensive, but at the same time functional and convenient device. In this material we will talk about the main functions and capabilities, as well as acquaint readers with instructions for using the Prestigio 5056 navigator.





Preparing the device for work

Before using the navigator for the first time, you must familiarize yourself with the installation and operating instructions. From it you will learn how to properly fix the device in the passenger compartment, how to install a memory expansion card, configure the navigator interface and set the route.

Before starting the device for the first time, it must be fully charged from the mains. The approximate time to fully charge the navigator of this model is about two hours. Nevertheless, it is advisable to extend the first charging process up to eight hours - this will favorably affect the capacity and ‚Äúhealth‚Äù of the battery. The indicator will signal the end of charging, changing its color from blue to amber.

For the correct operation of the device, the instructions for using the Prestigio 5056 navigator recommends installing a memory expansion card before turning it on for the first time. In this case, all updates of the maps of settlements, as well as multimedia files will be saved on this media. The next step will be to determine the installation location of the navigator in the passenger compartment. The place near the windshield is best suited for this purpose. In this case, the signal quality from GPS satellites will be better.

Device installation

Following the instructions for using the Prestigio 5056 navigator, we install the device in the cabin. For mounting the navigator on the front panel of the car in front of the windshield, two elements are provided in the kit. This, in fact, is the mount itself and a special bracket. They must be connected, and using a special Velcro to attach to the surface of the dashboard or to the windshield itself. Then, using the lever, lock the mount in the selected position. After that, you can place the navigator on the fastening station. The installation of the device is completed.

The next step, according to the instructions for use of the Prestigio 5056 navigator (see the photo of the device below), will determine the path of movement. At first glance, asking a route can seem like a daunting task. This is especially true for users who are dealing with a car navigator for the first time. In fact, this model has an intuitive graphical interface that you quickly get used to. Even an inexperienced user, using the instructions for using the Prestigio 5056 navigator, can set the necessary route. Below we will describe how to do this in several ways at once.





Routing

If the navigator is running, then after some time your location will be determined using GPS satellites. This point will be the beginning of the route. All you need to do is set the destination on the map by touching the device‚Äôs screen. In addition, you can specify the location where you want to get directions using the exact address. To do this, enter the menu, and there select the "Find" section. Then indicate the desired country, city, street and set the desired address.





Another way to get directions is manual search. To use it, you will need to turn off the GPS-receiver, indicate on the map the point where the route starts and the destination, and then turn on the GPS again. All these operations are described in the instructions for use of the Prestigio 5056 navigator. Customer reviews show that setting a route in this model is convenient and simple.
