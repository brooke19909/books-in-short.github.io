School math. How to find a percentage of a number

The word "percentage" in Greek means the hundredth part of a number. In mathematics, and throughout the world, it is customary to consider the absolute for 100%. Based on this principle, all computational rules are built.



There are several options for tasks related to the purpose of calculating the percentage of the number. Each similar task has its own individual solution principle.

Find the percentage of the number

In the problem statement, a specific numerical value is given and its percentage needs to be found. For example, we have the number 47 and we need to calculate it 25%.

Solution: For the solution, we take the initial number as 100%. After that, this percentage is converted to decimal fraction and we get that 25% = 0.25. We multiply 47 by the percentage expressed by a fraction, and we obtain the desired number 47 * 0.25 = 11.75.

Answer: 11.75 is 25% of the number 47.

Find the number by percentage





The next variety of tasks related to the question of how to find a percentage of a number is to calculate the value from the available percentage. It is given that 57 is 45% of some number. Find this number.

Solution: To solve such a problem, it is necessary to divide the available number by the percentage that it is from the whole. So, we get that 57 / 0.45 = 126.67. To better understand this action, it will be useful to analyze the entire process in detail. 57 is 45%, i.e. to find the value of one percent, you need to divide the number by the number of percent. It turns out that 1% of the integer is 1.2667. Next, to find the integer, we multiply the obtained value by 100.

Answer: The number, 45% of which is 57, is 126.67.

Find how many percent one number is from another



A little more complicated are the tasks in which it is necessary to find the percentage value, which one number is from another. How to find the percentage of the number in this case? The answer is very simple. Consider it with a small example. We have two numbers: suppose these are 45 and 58. To find out how many percent is 45 of 58, you need to multiply it by 100 and divide by 58. We get that 45 is 77.6% of 58.
