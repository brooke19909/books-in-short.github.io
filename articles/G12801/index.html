Details on how to create a folder on iPhone

In this article we will consider how to create a folder on the iPhone. In the case when a large number of applications are installed on the communicator, there is practically no space left on the desktop. In this situation, you should learn how to make a folder on the iPhone. The device has a special program for sorting applications into directories. Before creating a folder on the iPhone, the user first needs to clearly imagine how he wants to organize his programs.

To ensure that access to the application does not cause any difficulties, you must use directories to save the contents of the same type. For example, applications for social networks can be stored in a specific folder, while video can be hidden in a completely different place. Define two programs for future placement in one folder, and then click on the icon of one of them, hold it until you notice that it is swaying. Move one application on top of another and a directory will be created. In order to add other elements to the created folder, you should resort to the corresponding function. Icons for some applications are visible on the catalog surface. It is possible to move folders around the screen. Please note that no more than 12 applications can be placed in each folder. Also, you cannot place another in one folder.

Control





When a new folder appears, the system itself will name it automatically. The name is based on applications stored internally. Renaming a folder to your preferred name is possible. Go into it and click on the name. The keyboard will open. Enter any name convenient for you. To remove an application from a folder, click on the corresponding icon and hold your finger until the icon starts to sway. Drag the content you select beyond the boundaries of the folder. Hold it for a few seconds. When you return to your desktop, you can move the application wherever you want.

Deleting the folder itself is done by pre-cleaning all its contents. After canceling the last application, the directory will be automatically deleted. To remove all content at once, you need to enter the settings, then select the "General" and "Reset settings for the placement of the general table." All folders will disappear instantly. The content from the applications will be placed on the desktop in alphabetical order.

Any user will say that in order to find the right program, it is extremely inconvenient to scroll through several screens. The desktop should be put in the proper order and all applications should be organized in the appropriate folders. To do this, perform the following steps:

  • Identify two or more applications that you intend to put in a specific folder.
  • Hold your finger on the icon of one of the required applications until all the signs begin to sway. This is a signal that you have entered the desktop editing mode. If you want to leave it, press the Home button.
  • In order to create a new folder in the iPhone, drag the icon of one of the selected applications to the designation of another element.
  • As soon as the two parts come into contact, a new folder will be created with the contents of the two programs you have selected.
  • So, we have created a catalog on the iPhone. It will automatically open. Its name will consist of a type of content. At the request of the user, this name can be changed.
  • To rename a folder in iPhone, you need to create an additional area above the contents of the directory. A keyboard will open before you, through which you will erase the automatic name and enter a new one at your discretion.
  • In order to change the contents of the folder you created, drag the selected applications onto the directory area and they will be automatically sorted.
  • To add elements, hold your finger on the icon of the selected program and drag it onto the section image.
  • In order to remove the application from the folder, open it. Select the element that we want to clean, hold the symbol and drag it to the desktop.

Now you probably understand how to make folders on the iPhone. We exit the editing mode using the special Home button.

How to change a directory name

In order to rename a folder, you should hold your finger on its icon and go to the desktop editing options. Having removed the hand, we open the element with a touch. Next, in the name field, write the desired title.
