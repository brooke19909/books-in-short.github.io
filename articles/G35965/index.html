Washing machine error codes

You need to know these codes, as they can help users correctly diagnose a problem in the LG machine.

Error code

condition

Repairs

IE

Inlet water supply error

The initial expected water level will not be reached in 8/25 minutes. You need to check the water taps and make sure that they are fully open. Check hoses for kinks or freezing (in winter). Check the inlet valves. If they are faulty when repairing the LG washing machine, you need to replace them.

UE

Unbalanced load

Redistribute clothes in the car. If the load is too small to balance, add items to the drum that need to be washed. In addition, you need to make sure that the machine is correctly set in level.

OE

Drain error

Water is not removed from the machine within 10 minutes. Check the drain hole for obstruction or hose for kinks and restrictions.

FE

Water Overflow Error

The drain pump operates continuously. Check the valves to make sure they shut off the water supply correctly after filling the drum.

PE

Water level pressure sensor error

Turn off the washing machine. Check the wiring between the water pressure sensor and the main control panel. This repair of LG washing machines can only be done by replacing the water pressure sensor.

dE

Door open error

Turn off the washing machine. Check door for obstruction. If damaged, replace defective parts. If the door closes correctly, check the wiring of the door lock / switch.

tE

Heating sensor error

Turn off the washing machine. Check wiring connections at drum thermistor (temperature sensor). To eliminate this error of the LG washing machine, you need to measure the resistance using a volt / ohm meter. It should measure about 40,000 ohms at 25 degrees. If the thermistor is defective, replace it. Check the heating element for damage. If the heating element is defective, replace it.

LE

Locked Drive Error

Check the wiring harness connections on the engine and the travel sensor (tachometer). Repair any malfunctions in the wiring.

Ee

EEPROM error on main control board

EEPROM (Electrically Erasable Programmable Read-Only Memory) on the control board is damaged. Clear the code by unplugging the machine for 5 minutes. If the errors of the LG washing machine by this code are not reset, you need to contact the after-sales service.

Pf

Power failure

A power failure occurred during the cycle. The cycle may have ended abnormally. Restart the machine.

  4. Need to remove excess soap suds. Why disconnect the washing machine and leave it for 30 minutes. When the foam settles, then turn on the unit again.

  6. Press the START / PAUSE button to start the drain cycle. After completion, remove clothing from the machine.
