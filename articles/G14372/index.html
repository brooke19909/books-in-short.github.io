The subscriber of the Megafon operator has a communication restriction: what to do when a ban is set?

When can the Megafon communication restriction be set?

The inability to make or accept a certain type of call in 100% of cases is due to the fact that the user has activated the corresponding ban on the number. The subscriber could do this either by himself, by entering combinations of characters on the telephone keypad, or through a customer service specialist. An important point is the fact that the restriction on calls is set by an employee of the support service only after confirming the identity of the owner of the number. Thus, the possibility that a communication restriction will be established is excluded. What does it mean? "Megaphone" allows you to carry out any actions with the number only to its owner.





Why is Megafon communication restriction set?

The question of why it is impossible to contact some subscriber or receive an incoming call most often arises for people who have never encountered the Call Barring service. With its help, you can limit the reception or implementation of calls of a certain type. Who can benefit from this? This service will be a real salvation for those subscribers who are going on vacation to another country and at this time want to exclude the possibility of making calls. Since the cost of incoming and outgoing calls is rather high in roaming, this desire is quite justified. It can also be connected in other situations when it is necessary to suspend some kind of connection. For example, this may be relevant for parents who want to avoid writing off funds for sending messages to paid numbers by their child (a similar ban can also be set using a mobile phone).





Description and conditions of the ‚ÄúCall barring‚Äù service

The list of any mobile operator has a number of services that are basic and are provided to subscribers free of charge. Megafon is no exception. The service we are considering, with the help of which you can establish a ban on certain types of communications, is included in this list. Activation of the restriction can be carried out independently by the subscriber, by entering the appropriate commands or by a support service specialist. It should be borne in mind that a fee may be charged for connecting the service through Megafon employees. The communication restriction can be set to the following types:

  • outgoing communication (local calls, calls within the country) - code 33;
  • incoming call (all calls) - code 35;
  • outgoing international communication - code 331;
  • outgoing communication in roaming (blocking calls to all numbers except Russian) - code 332;
  • incoming call in roaming (all calls) - code 351.

To connect a ban, the following conditions must be met:

  • the call forwarding service is not activated on the subscriber‚Äôs number (if available, you must first disable it and then set a ban; the exception is if the call forwarding has been set by the operator);
  • only one restriction can be set for each type of communication at a time (that is, only one prohibition can act on incoming and outgoing communication at the same time; setting the second cancels the previous one).

Activate call barring

Service activation is available to all Megafon subscribers. A communication restriction can be established at any moment of time; disabling is also disabled as necessary. To manage the service, the client will need a special password (for all subscribers the default combination is 0000) and codes of each type of communication, the provision of which can be activated by a ban (they are given).

To enter the restriction, you need to enter on the phone the command * service code * password to manage the service # . You can also set bans on a number of certain types of calls (for example, on SMS messages - code 16, on receiving and sending fax messages, etc.). In this case, the command to set the ban will be slightly changed: * service code * password to manage the service * call type # . Activation of the restriction will occur in 10-15 minutes.





Service Deactivation

Do I have to contact Megafon specialists for deactivation? Communication restriction set, how to disable it? Similar questions may arise for subscribers who need to resume the provision of services by lifting the corresponding ban. Disabling the ban service is as easy as connecting it. To do this, just enter the following combination on the phone: # service code * password to manage the service # . A similar command can be used to deactivate bans on call types (see the full list of call types on the official website of Megafon).





Change password to manage the service

The password, which is used by default to enable / disable the ban, can be changed by the user. This will protect the phone from unauthorized attempts to control the service by other people. To enter a new password, use the combination: ** 03 * 330 * current password * new password # . Its validity period is not limited (until a new sequence of numbers is introduced). However, if you make a mistake three times when entering a combination to change the password, then the use of the service to set the communication ban will not be available. Removing the lock will be possible only when contacting the office of the owner of the SIM card with a passport.

Conclusion

If during a call to a Megafon number he says: ‚ÄúCommunication restriction has been established‚Äù, then with high probability we can say that the Call Barring service is activated on the subscriber. The owner of the SIM card can independently find out about the existence of such a restriction through a technical support specialist (at 8-800-550-05-00). Service management can be carried out both by the subscriber and through a support service specialist. After deactivation of the ban, it will again be possible to use the communication services on the Megafon number. A communication restriction can be established only by the owner of a number who has a personal password for managing this service. It is recommended that you change the default sequence of numbers to ensure the security of your number.
