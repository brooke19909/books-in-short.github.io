Mobile Bank Lock

In case of loss of the card or temporary refusal of the service, customers can suspend SMS informing using the USSD command. This is a brief request to the number "900", which blocks the service until it is renewed by the cardholder. A feature of the method is the complete absence of commission: all SMS to the bank address are free, payment can be charged only at the tariff of the mobile operator.

How to temporarily refuse a Sberbank mobile bank via SMS:

  1. Type one of the "Service Block" or BLOKIROVKAUSLUG in the message. The short code for the operation is "04".
  2. Send a message to "900".
  3. Confirm the desire to get rid of messages using the code sent in the response notification.

Using a USSD command is a temporary way to refuse a Sberbank mobile bank. It suspends the option indefinitely. During blocking, payment for the "Full" package is not charged. To permanently disable the service, you must call the contact center or visit a bank branch.
