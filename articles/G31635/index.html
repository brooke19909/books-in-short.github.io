Pandect (alarm): overview, description, types, specifications and reviews

Alarm Trade Pandect products are developed and manufactured in the Kaluga Region by the Experimental Instrument Making Plant. The Pandekt brand produces both full-fledged security systems and anti-theft immobilizers, which allow controlling your car and using it with a mobile phone.

Manufacturer and models

Automotive safety systems have been manufactured by Alarm Trade since 2004. The company has long and deservedly secured the status of a leader in the field of car-security in the domestic and world markets. Pandect alarms meet all modern world standards and are equipped with high-precision European equipment.

"Pandekt" is an excellent security capabilities, quality, reliability and modern design. And yet, the models of this brand are a vivid example of the successful use of the latest achievements in the industry of microelectronics and security systems for cars. They give the car effective anti-theft and anti-robbery functions.

Today Pandect alarms provide the owner with the most extensive service and security features:

  • determining the location of the car;
  • phone control;
  • remotely customizable sensors;
  • autostart engine - programmable;
  • much more.

Next, we offer a small overview of the Pandekt car alarm models with reviews and characteristics.

Popular security system

The security and anti-theft alarm system for the Pandect X 1170 car is developed on the basis of the popular X-1100 model and is equipped with extended equipment.

The RMD-8B expansion module, which appeared in this model, has expanded the functionality of the alarm system, especially if it is installed on an analog vehicle or a car that requires a larger number of power connections.

Management Methods:

  • Mobile app.
  • Contactless tag.
  • Mobile phone with GSM.
  • The standard car key.
  • Established buttons in the cabin.

In combination with the above module, maximum convenience, miniature dimensions and secrecy of installation are fully preserved. At the same time, the ability to install Pandect alarms on any analog vehicle while maintaining full functionality is added.

Innovative technologies

This deeply modified model was released in the summer of 2017 and has already managed to attract hundreds of car owners.

Pandect X-1900 BT is:

  • IMMO-KEY;
  • GPS / GLONASS;
  • Bluetooth 4.2;
  • 2G / 3G (GPRS);
  • 2xCAN.

In addition, the monoblock system has a record miniature size, the highest functionality and a fairly low current consumption. The Pandect alarm system for the car turned out to be so successful that many of the most experienced anti-theft experts in Russia and abroad consider it now its best security and service system for the car.

The updated version of the model is based on the state-of-the-art microcontroller STM32F413, which made it possible to reduce the power consumption of the system to record levels. The average consumption of the X-1900 BT in armed mode is up to 10 mA.

This model is even more versatile and easy to install. The Pandect X-1900 BT alarm system is quickly installed on modern cars in which power connections are not required. The function of switching to the positive polarity of two timer channels with a load capacity of up to 0.25 A has been added to the system. According to motorists, it is precisely because of this that the model can be fully connected to most modern popular cars.

Micro system Pandect X-3150

This alarm model is a highly functional security and service microsystem with simplified installation on any modern car. This Pandect alarm is equipped with:

  • built-in interface with increased range for dialogue communication 868 MHz;
  • GPRS Internet interface;
  • Bluetooth interface.

The system has a built-in algorithmic keyless crawler of a standard immobilizer. The integrated PWM port allows most modern cars to be launched without a key.

With this Pandect alarm, according to customer reviews, it is convenient to use a smartphone to monitor, configure the system, monitor and control the car through mobile applications.

Miniature model

The Pandect X-1000BT car alarm will be a great choice for those who do not want to use a key fob, but who want to manage car security through a mobile application system, eliminating the additional costs of mobile communications.

Unlike the immobilizer, this model has a multi-system 2xCAN interface for connecting the system, is equipped with a CAN lock function and a code immobilizer. In addition, the system is equipped with a small-sized relay to block the engine and a contactless immobilizer tag for hidden wearing and recognition of the car owner. Thus, this Pandekt system is equipped with the latest anti-theft qualities, has security zones and, if violated, goes into alarm.

You can make settings and manage the system using Bluetooth 4.2 protocol via mobile applications, while the control range is up to 70 meters. The immobilizer label has a button to turn off and turn on the system in case of a discharged phone. And the system provides for an emergency shutdown by entering a secret code.

As for the dimensions of the main unit, it is half the size of the matchbox. With a quality installation, finding a Pandect alarm in a car is very difficult.

The unit is equipped with:

  • 3-axis accelerometer that determines the position of the car;
  • digital sensor of shock, movement, tilt (in case of unauthorized movement of the vehicle, the motion sensor blocks the circuit).

Pandect x-2000

This car alarm with an inexpensive D-0745 remote control is a robust construction combined with good functionality. The presented system is produced using the latest scientific technologies and provides ease of use.

Car Alarm Pandect X-2000 has a large number of service functions:

  • The system provides simultaneous control of ten security sections of the car.
  • It works correctly on the CAN-bus together with the standard pre-start heaters of many cars.
  • Allows you to start the car engine.

The alarm kit includes one LCD key fob and one key fob, ideal for HANDS FREE control and hidden wear.
