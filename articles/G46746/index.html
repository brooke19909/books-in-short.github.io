SMS working principle

SMS is not a direct way of interacting with another person, unlike a voice call, when communication is established directly between two devices. After the message is sent by the subscriber, it goes first to the server, which, in turn, checks whether it can be delivered to the specified number. If the test result is positive, the message is sent to the addressee. Otherwise, it remains on the server for some time, waiting to be sent.
