How to print from phone to printer: instructions

With the improvement of mobile gadgets, the need for digital cameras has disappeared dramatically. Anyone can take a quality shot on their Android phone. Now that we are not attached to a PC and can mobile edit our documents, send and receive files, set tasks, we are freer from the ties of wires, cables and devices. But there are only cases when a certain document needs to be transferred from a digital format to a paper one. Many users do not know how to print photos from a phone through a printer. There are several ways. Let's try to deal with each individually.

Printing through the cloud

If you are a happy owner of a modern printer, then most likely the Cloud Print Ready stamp will appear on it. In this case, it is enough to connect to the Internet and register on any cloud portal. After that, you can print your favorite photos on any device and anywhere without a PC.





Those who have an older model or simply do not have such a function can also use the cloud, only in this case they will have to resort to using a computer. In the Chrome browser settings, you need to add your printer, register with Google, and finally check the history of tasks on the printer itself. Since the main work is performed directly on the ‚ÄúAndroid‚Äù device, you will have to download the corresponding application to your phone, for example:

  1. A virtual printer is a simple official application from Google, which allows you to send files to print, view history, link several accounts and printers.

  2. Cloud Print - in principle, this application is similar to a "virtual printer", with the only difference being that in a particular case you have the ability to print files from much larger sources. For example, SMS, screenshots, contacts, photos from Facebook, etc.

  3. Print Share is the most popular application on the Market, allowing users to understand how to print photos from a phone to a printer, as well as files, contacts, web pages, events. Of the shortcomings - it does not perceive the Cyrillic alphabet, in addition, although initially the application is free, you will have to buy additional options for more functional use.

Each of the above applications has its advantages and disadvantages. All of them can easily cope with the question of how to print photos, documents, and more through a telephone on a printer. But expect more from them all the same is not worth it.

How to print from phone to printer via USB

It‚Äôs worth noting right away that this is not a good method, since not every printer can be manipulated like this. First you need a USB Host output on your phone, then you will need drivers, which, by the way, are not many. You may need to download the USB Connection Kit from the Market. If everything goes well, the phone will see the device as USB, and you will be able to print files.





Of particular note is the ePrint app, which sees all HP devices connected via cable.

How to print from phone to printer via WiFi

If you have set yourself this goal, be aware that this is one of the easiest methods to print files from your phone. Of course, provided that the phone and printer have Wi-Fi connections. All you need to do is go to the settings, turn on Wi-Fi direct and submit the print task.

But such an easy way is possible only on modern devices. Earlier models will have to tinker with. Before you print from your phone to a printer, you will need to connect to the network, and directly to the router.

How to connect a router to a Wi-Fi printer directly, via WPS

First you need to clarify the support for WPS-connection on your router, make sure that MAC address filtering is turned off and the network is protected by encryption. To configure some models, you need to know the network name and password. And also your PIN code (it is on the box of the router in the form of eight digits located under the serial number).





Next, enable WPS on the router, enter in the browser 192.168.1.1., In the username and password field - admin. In the "Security" section, you must set the ENABLE parameter.

Take a closer look at your router, it should have a network search button on it. It should be pressed until a connection is established. If there is no button, you will have to use the software of the device itself. It‚Äôs easy - just click on the Wi-Fi Protection setup in the ‚ÄúNetwork‚Äù section. Do not forget that both devices must be turned on for 2 minutes.

How to print files using Dropbox

Before you print from your phone to a printer using Dropbox, you need to create an account on the site. Next, download the appropriate application to your phone and computer from the Market. It remains only to save the files in the "Dropbox" that you want to print, then, logging in through the computer, send them for printing.





Listing files from Apple devices

Suppose you have not an android in your hands, but an ‚Äúapple‚Äù. In this case, to print files on a printer, you will need special applications designed for iOS devices.





  • Apple airPrint - and nothing more. No cables or drivers, just a few touches, and the virtual photo becomes quite real. The main thing is the availability of a compatible printer, which, unfortunately, is very problematic in our country.

  • Handy Print is a wonderful replacement for the above application, which is free for the first 2 weeks, then you will have to deposit funds for further use. Perhaps the only minus of the application is that it requires a loaded PC on which it will be installed.

  • Printer Pro - significantly different from the previous two. You will have to print documents directly from the program. Just click "Open in" and select the application you want, for example, Dropbox.

  • Epson iPrint is a proprietary application developed by the Epson printer manufacturers themselves . The program automatically finds available devices and connects to them on a wireless network. It can also print via email.

  • HP ePrint Enterprise is another similar application created by HP programmers. It also connects to a wireless network, works with cloud storage and accepts tasks from e-mail.
