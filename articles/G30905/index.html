How to prepare iPhone 5 for Hard Reset? Hard Reboot Process

Unfortunately, even the new iPhone may freeze and stop working. This can happen if you get a virus infection or a system crash. The reason may even serve as a load of RAM. It is in this case that the actual question arises of how to make Hard Reset on the iPhone 5. It is advisable to perform a reset of all data in order to remove a possible cause of the problem - a virus.

Description

This type of reboot will be able to reanimate the phone if it freezes and does not respond to keystrokes. Users note that it works in almost 100% of cases. However, it is recommended that you back up all data before executing. After all, as mentioned above, when performing a reset before Hard Reset on iPhone 5, all information will be deleted.

When to do a hard reset?

This function will be necessary if the phone slows down, freezes. It will help, even if the smartphone just works slowly. If any errors occur, a hard reset is also allowed. Hard Reset on iPhone 5 will be useful if the user has problems with the Internet connection, some utilities, applications and keyboard.

If the function is performed without resetting the data, the data will not be affected. However, if the cause of the above problems is a virus, then a hard reset will not help. Therefore, it is worth pre-resetting the settings and data, so the maximum expected effect will be achieved. Below we‚Äôll look at how to make a backup.

How to back up?

There are two options that will allow you to save data. The first involves the use of iTunes, and the second - iCloud.

  • You must be connected to the Internet. Next, find the iCloud program on the device. After that, go to "Settings", activate the function to create a backup copy of the phone. It remains to wait only a certain amount of time for the process to complete successfully.
  • When using iTunes, you must connect iPhone 5 to your computer. Hard Reset can only be performed after the data has been completely saved. Next in the program you need to find a function that allows you to create a copy. The process will last no more than five minutes.

How to reboot?

In order to perform a hard reset, you must keep the device turned on. Some users think that you need to hold three keys in the iPhone 5. Hard Reset is done a little differently. Only two keys are involved in the process. These include the "Home", located under the screen, and on / off. Keep them pressed until the screen goes out and turns on again.

Summary

The article describes how to make a Hard Reset on iPhone 5. You need to do it only when the device is turned on and the battery is at least 50% charged. Otherwise, malfunctions may occur. If a hard reset was performed when the phone was charging in the off state, then the device most likely will not turn on. You need to disconnect from the cable and hold the power key for a long time. If this does not help, then you can do a hard reset again.

The above describes how to make Hard Reset using the "Home" and lock keys (on / off). It should be understood that no other method exists.
