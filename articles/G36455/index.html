Wi-Fi router setup: step-by-step instruction

This article will discuss the step-by-step configuration of a WiFi router without specific reference to the model or manufacturer. The main specifications of such devices and recommendations on the selection of such routers will also be given. In addition to this, the basic requirements for choosing the installation location of various network nodes will be given. All this will allow the potential buyer to choose exactly the solution that will most closely match his needs. Also, with the help of this material, a novice specialist will be able to configure the computer network properly.

General characteristics of routers. Recommendations for selection. Equipment

Any modern router is a universal solution that is able to create both a wireless segment and a wired section of a computer network. Also, some of these most advanced devices are equipped with a USB port. In this case, the router can perform the functions of an archive server or documentation. Also, in addition to the same port, you can install an external modem and use it to implement the backup connection technology to the global web.

For home use, a router that has four LAN ports with a speed of 100 Mbps is enough. With two antennas for creating a wireless network, the transmission speed is already limited to 300 Mbps. But in the case of the office, it is recommended to purchase more advanced solutions that have an additional 2 USB ports and an increased transfer rate in the wireless segment of the computer network.

In most cases, the configuration of such network nodes includes the following:

  1. Router
  2. Power Supply.
  3. A disk with software.
  4. An extended user guide that details the features of such a device. Also, in some cases, it describes in stages the setup - Mi WiFi router from Xiaomi, for example.
  5. Communication wire for reconfiguring a network device.

Types of Routers

Conventionally, WiFi-enabled routers can be divided into two large groups. The first of these is universal devices that are not tied to the equipment of one particular provider. They have a higher cost, and they need to be configured properly. For example, setting up a Tp-Link WiFi router can be done either using the wizard or using the WEB interface. It is easier for beginners to apply the first method. But for professionals, the second of them is more suitable. By the way, it is recommended that Xiaomi be configured with the second method also.

The second large group of routers includes specialized solutions. They are sold by providers to their customers. Such devices are already initially configured to work as part of the computer network of a company. But such routers will not work with the equipment of other providers. In this case, you only need to switch the device on and on.

General tuning algorithm

In general, the WiFi router is configured as follows:

  1. Unpacking the device. Installation of antennas in case of their separate execution. In some cases, they are not removed, and there is no need to install them.
  2. Choosing an installation location. The supply of communications.
  3. Network node switching.
  4. Turn on the router. Correction of the software shell of the device. Its reboot.
  5. Health Check.

Training

At the preparatory stage, the WiFi-router should be removed from the shipping package along with the documentation. The latter is recommended to study in detail. It is also necessary to inspect all the stickers on the device and find the name of the network and access code for it.

Another point is the contract with the provider for the provision of services for connecting to the global web. It indicates the parameters that must be entered into the software shell of a network device. This is the type of connection, this is the login, and a special password, and the type of addressing, and DNS. All these data are recommended to be written separately on a blank sheet and then used in the process of reconfiguring the network node.

Choosing an installation location. General recommendations and wiring

Setting up a WiFi network starts by choosing the right place to install the router. The first important condition is that the network access point should be located as close as possible to the center of the room. If there is not enough coverage, then it is necessary to additionally acquire the so-called repeaters and already use them.

Another important point is metal surfaces. Near the router, they simply should not be. If this condition is not met, then the level of the received signal and the quality of the coating will significantly deteriorate.

Also, in the installation site of the network node should be a socket for the organization of its power. The second important point is the ability to supply twisted pair from the provider to the installation location of the router. It is based on these conditions, and we choose the installation location of the router.

Commutation

The next step is to start connecting the network node. To do this, the power supply is installed with the built-in plug into the outlet, and the cord from it is inserted into the device‚Äôs socket. Then we connect the twisted pair cable from the provider company to the port marked WAN or Ethernet. It is usually blue.

Further, if necessary, a wired part of the computer network is created. Using twisted pair, the ports of various devices are connected to the LAN connectors. As a rule, there are 4 such nests in routers and they are yellow.

In some cases, at the final stage, the MFP, printers, additional network drive and modem are connected to the USB port.

Reconfiguring the network host shell

If the router is provided by the provider, then its shell is configured, and there is no need to make any changes to it. It is only enough to carry out switching and turn on the device.

Well, in the case of using a conventional router, you must properly adjust its parameters. To do this, after turning on the network point, you need to connect any wireless device to it. In this case, the access code and SSID can be specified on the sticker from the router.

After establishing a connection, launch the browser and go to the address of the router. This is usually 192.168.1.1. But there may also be 192.168.0.1 or 192.168.10.1. In any case, this information, like a password with a login to gain access to the parameters of the router, can be clarified in the instruction manual. After navigating to the address of the network point, we enter the previously specified values. Next, in the interface that opens, you need to set the following parameters:

  1. WAN / Ethernet Tab: Connection Type, Address Assignment Method, and Subnet Mask.
  2. WiFi tab: the new name of the wireless network, the connection code to it and the type of encryption WPA2.

After each change, the parameters must be saved. Actually, on this, the WiFi D-Link 300 setup, and not only, is over. Now you can exit the network point configuration interface.

Testing

At the end of the software configuration of the network node, it is necessary to check its operability. To do this, activate the WiFi transmitter on any device, wait for the connection to be established and launch the browser on it. After that, open any information resource from the global web in the last window.

If after this the site appeared, then everything is in order. Otherwise, it is necessary to additionally check the switching and parameters. After that, re-check the operability of the created computer network.
