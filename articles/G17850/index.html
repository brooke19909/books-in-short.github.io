So what is a trigger?

By definition, it is an element with two stable states. Its simplest non-electronics example is a switch. It also has two stable states - on (on) and off (off). For a trigger, these states will sound accordingly - in a single state (on) or reset (off). In a single state, the output of the trigger has a voltage slightly less than the supply voltage, in the reset state - a low voltage level. From here comes its name, i.e. an element is in one position. Translated from English, the trigger is a latch.

The simplest will be an electronic trigger on transistors. For its manufacture, two transistors, several capacitors and resistors are required. The first use of a trigger will be to use it as a switch.

Often you have to perform a simple action - press the button and then release it. So, usually this action is remembered by a trigger. When a button is pressed, it is set to a single state, and then the device already understands that it needs to work in the mode specified by the pressed button. To exit this mode, press another button, then the trigger will reset and a new operation mode will be set.
