What is the ISO in the camera and how to work with it?

On such a parameter as ISO, everyone at least once "came across" in the menu of his camera. However, not everyone knows about him. Let's see what ISO is in the camera.



Under this parameter is the camera's photosensitivity. The sensitivity is higher, the higher the value of the specified parameter, and from a high value in the photo a lot of noise appears. That is why experts recommend using a low ISO value to get high-quality photos without noise. When shooting in low light without a tripod, increase the value of this parameter. If the aperture is also fully open, you can increase the shutter speed so that the photo does not work out blurry. If we talk about what ISO is in the camera, it should be noted that when shooting at night, you should still use a tripod, and also set the value of this parameter below. When using the flash, its high value will not be necessary.



It is worth noting that the amount of noise in the photo with a high ISO value directly depends on the size of the matrix used in the camera. If these are compact cameras, then with ISO 800 you should not expect a good result. Poor lighting will cause the photo to be smeared completely. In this case, the camera should be held on a tripod using a low parameter value. When working with professional or semi-professional equipment, you can set its value four times higher, while the quality of the pictures will be simply amazing.



Speaking about what ISO is in the camera, it is worth recalling such an opportunity as Auto ISO. It can be noted that this function is quite moody. If you use a soap box, then you should set the ISO 100 and not bother. When using a SLR camera, you can limit Auto ISO to a certain extent. For example, you set the highest value to 800, then the camera will be unable to raise it higher, even if it is appropriate.
