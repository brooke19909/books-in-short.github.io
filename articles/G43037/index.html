Which regions and which operator does the 920-th area code service?

This article will answer what operator 920 has a digital code and in which regions of the Russian Federation it can be found.





Who!

First, let's figure out which operator 920 has in its arsenal. Today, he is hiding a single operator who belongs to the so-called "Big Three". More specifically, this is MegaFon. But a strictly defined region was not allocated for this telephone code and, as a result, the telephone database of subscribers with such numbers has a rather large geography.





Dialing order

After we received the answer to the question: “Which mobile operator does 920 have a code?”, We will deal with the rule for dialing such telephone numbers. When making a call within the country, you need to dial the following combination: +7 (RF code) - 720 (one of the Megaphone codes, which is discussed in this article) - XXXXXXXX (this is the digital subscriber number). Then you need to start the call by pressing the appropriate button. The dialing order will slightly change when making a call from Belarus and Kazakhstan. In this case, instead of “+7” you need to press 8 and wait for the appearance of a loud and continuous beep. Then we dial the rest of the number in the same way as inside the Russian Federation. All over the rest of the world, to make a call to these numbers, it is recommended to use the combination “007” instead of “+7”. The rest of the subscriber’s number will not change in this case.

Regions

Now we know which operator 920 telephone code within the Russian Federation serves. This, as mentioned earlier, is “Megaphone”. But you can meet the phone numbers of this series in the following regions: Central, North-West and Caucasus. In the first case, these are Nizhny Novgorod, Kaluga, Oryol, Tula, Kursk, Bryansk, Vladimir and Ryazan regions. In the second, you can find this phone code in the following regions: Yaroslavl, Tver, Smolensk, Ivanovo and Kostroma regions. Well, in the Caucasus region these numbers are assigned to Belgorod, Tambov, Voronezh and Lipetsk regions. So all these numbers are tied to the European part of the Russian Federation.





Northwest Region

Now we will figure out which regions of the northwestern part of the Russian Federation use the code 920. Which telecom operator is hiding behind it is MegaFon. First of all, it is necessary to highlight the Yaroslavl region, which in this case is represented by the number base 1,000,000 ─ 1109999; 1120000 ─ 1499999 and 6500000 ─ 6599999. In the beginning, as you might guess, do not forget to add +7920. The same nuance is taken into account in all other cases further in the text. Another region that may be hiding behind this phone code is the Tver region. In this case, the mobile phone numbers should be in the following ranges: 1500000 ─ 1999999; 6,800,000 ─ 6999999. The Smolensk region also fell on this list, for the needs of which such a number base was allocated - 3,000,000 ─ 3399999; 6600000 ─ 6699999. This list is supplemented by the Ivanovo region, where you can find the following numbers with this phone code: 3400000 ─ 3799999; 6600000 ─ 6699999. Finally, we should also note the Kostroma region, which boasts such a number base: 3800000 ─ 3999999; 6400000 ─6499999.

Well, the dialing order is fully consistent with the previously given phone number. The only thing at the end is to press the call button.

Caucasian

As noted earlier, in the Caucasus region, you can also see the telephone code 920. The mobile operator remains unchanged - this is the same MegaFon. In the Belgorod region, you can find such numbers with this phone code: 2,000,000 - 2099999; 5500000 - 5599999. It is difficult to say what the management of the mobile operator was guided by when it attributed this region to the Caucasian branch of MegaFon. It would be more logical to attribute it to the center, but, alas, for some unknown reason, such a strange option was chosen to divide the regions of the Russian Federation. The following number base has been allocated for mobile communication subscribers in the Voronezh Region: 2100000 - 2299999; 4,000,000 - 4,699,999. Again, it is not entirely clear the division of regions from the perspective of geography, but geographically this region is subordinate, like the previous one, to the Caucasian branch of MegaFon. Another area that was not quite logical in this company is Tambov. The numbers 2300000 - 2399999, 4700000 - 4999999 are provided for it. The Lipetsk Region also fell into this company, for the needs of which the numbers 2400000 - 2499999, 5000000 - 5499999 were allocated. The only thing that can be definitely noted is that the geographical location of the region is not always consistent marketing considerations. And this division of the Russian regions is a vivid confirmation of this. The dialing order in this case does not change and remains exactly the same as in previous cases.
