What should be a mobile phone for the elderly

Each person in old age should have in his arsenal a specialized telephone, convenient for use. Over the years, vision becomes worse, hearing can begin to disappear, fine motor skills of hands suffer, so the use of modern devices with small buttons becomes simply impossible. It is worthwhile to understand the question of what should be a mobile phone for the elderly. You can consider each parameter.



The size of the buttons of the device should be comparable with the size of the pads of the fingers of a human hand, and preferably even more. So a pensioner can not miss the dialing number, as well as not cling to the surrounding buttons.

A mobile phone for the elderly should display a large font on the screen, it should also be used on the keyboard. It should be easy for the pensioner to find the right button and read information from the screen. Some manufacturers supply their devices not only with large numbers, but also with special markings in the form of convex icons, which allows you to find the necessary keys even by touch. If such a device is in the hands, it should be assessed how successfully this opportunity was implemented.



Considering a mobile phone for the elderly, it is worth mentioning about the big screen. If a pensioner will be forced to read or write SMS, then the entire message should fit on one screen so that there is no need for scrolling.

It is worth evaluating the degree of need for additional functions. Some retirees like Tetris, but most prefer to use the phone exclusively for calls. Such an apparatus does not need any additional programs, an elderly person will not be sitting on the Internet for hours, there is no need for a large amount of memory. A mobile phone for the elderly is needed for calls, but features such as a flashlight and radio will not be out of place. There is one important requirement - it should be immediately clear how what is included. For the flashlight, you need to place a separate button on the case.





An important function will be the emergency call button. This should be a separate button that is easy to find. After pressing it, a distress signal should be sent to several numbers entered in advance, usually 8-10 contacts.

For adjacent buttons, the colors should be bright and contrasting, then dialing will be as simple as possible.

The device, designed for a pensioner, should have a loud call and transmit the speaker's voice in a high-quality manner. This is especially true for hands-free calls. If the phone transmits your voice with extraneous noise, then it will be simply difficult for grandparents to use it.
