UIP - what is it in a payment order? Unique Payment ID

A payment order, or, as it is called in the slang of bankers, entrepreneurs and legal entities, payment, is one of the documents for transferring money. All fields of this form must be filled in, including the TIP field, in which incorrect information can cause the transfer of funds to another account. This may cause late payment.

The concept of documents used for payment

Payment documents are used to confirm payment for products. They can be cash and sales checks, strict reporting forms, payment requirements and payment orders.

Cash and sales checks are issued to customers upon the purchase of a product. A strict reporting form is issued to a client receiving a certain type of service instead of a cash receipt.





A payment order is used when it is necessary to make non-cash payment using a bank account. This document is a confirmation to the person who made such a payment.

The concept of UIP in the payment

UIP - what is it in a payment order? This is one of the most important details of this document. Literally, it means a unique identifier for a payment. In the absence of information on this code, the field corresponding to it must be filled with zeros. If the code is incorrectly generated, the money can be credited to another account, and if paid to state organizations, this can lead to additional penalties.





In this regard, you need to take into account some subtleties when filling out the code in the field.

The purpose of the UIP

With its help, the transfer of funds is streamlined.

Government agencies have the ability to track payment documents, which is used for statistical research purposes.

Provides the holding of the amount necessary to complete a particular transaction.

UIP and UIN

In the payment order, in addition to the UIP code that is not related to budget transfers, the UIN code is given, which is decrypted as a unique identifier for charging. Under this code, all types of transfers are made to the budget. The numbers entered in this field should also be checked for correct input, otherwise the funds may not be where they were planned to be transferred. In this case, it will be necessary to return the money, and then transfer it again with new details, which will take a certain time.





Thus, the codes of a unique accrual or payment identifier should be used in different cases. They must be put in the same field, which is called the Code.

The purpose of the appearance of these fields since 2014 is to improve and optimize the work of civil servants to make payments more quickly credited. This is due to the fact that when specifying the code there is no need for a TIN, PPC or CSC of the organization.

In what cases is the UIP mandatory?

Indication of this detail in the payment order is not always mandatory. Cases of mandatory indication of UIP are regulated by the Bank of Russia.





There are two such cases:

  1. If this number was assigned by the recipient and the payer was informed about it in fulfillment of the terms of the contract. The procedure for generating the code and its verification by the bank are regulated by the Central Bank.
  2. When paying taxes, fees, various contributions. Since the above payments should be performed by everyone, it is useful to know everyone about the UIP and where to get it. However, it must be borne in mind that for these types of TIP is required not for ordinary payments, but for payment of arrears, penalties, fines imposed by the Federal Tax Service.

UIP regulatory framework

The legal regulation of the UIP is carried out using two documents:

  1. Appendix 2 to the Order of the Ministry of Finance No. 107n, which defines the mandatory indication of the UIP in case this identifier is present.
  2. Provisions of the Central Bank of the Russian Federation No. 383-P, according to which the bank should not accept a payment order if the UIP column is not filled. Mandatory code is indicated, in the absence of an identifier in field 22 must be 0.

Code Writing Rules

Besides the fact that you need to know what it is - UIP in a payment order, you also need to be able to write it down correctly.





  • The number constituting the code of the identifier in question cannot include less than 20 and more than 25 digits.
  • These figures indicate the payer and the basis of payment.
  • The numbers related to this code must be clarified by the payer.

We find out the code

What is this UIP in a payment order, we found out. Where can I get this code to fill out the document correctly?

First of all, such a code must be reported by the party in whose favor the payment is made. You can also find it in some government agencies that control the payment of payments to the federal budget. Such organizations include FNI, Federal Tax Service, PFR and Customs.

For example, in a paper receipt, this code will be indicated as the index of the document, which is located above the bar code.

In "Sberbank online" in your account enter payment information, click on the "Information" button, and then receive a check where the number will be indicated. Each transfer of funds is accompanied by a separate receipt of the number.





The next way to find out the necessary code is to use special directories. At the same time, the codes are given with the decryption in them, so it is necessary to carry out the identification correctly, which will avoid problems when transferring the payment.

What to do if the payment went with the wrong code

If the code in the field of the payment order, which corresponds to the UIP code, is missing, it may not be filled out, but zeros can be entered there. If the wrong code was written, then it is necessary to perform a series of sequential actions:

  • It is necessary to rewrite the payment order in which to indicate the correct TIP, and then make the payment again.
  • To the organization to which the payment was made with the wrong details, you need to write a statement stating the reason why she should return the money to the sender. In this case, the reason for the return is indicated as erroneously entered details.

The application must indicate the details of the current account where the funds will be returned.

Waiting for a refund may be delayed for a sufficiently long period of time. It depends on where exactly they got.

Remember to check the UIP codes

With the annual dynamics of document forms, codes can also undergo changes, including UIP. Therefore, when making a payment, it is best to always check not only the correctness of the code at any particular point in time, but in order to know for sure that there were no changes, you need to check for amendments made to the codes.
