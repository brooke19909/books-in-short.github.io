How to put a melody on a contact on the "Android" (Android)?

The question of how to put a melody on a contact on an “Android” device worries a lot of users who are just starting their acquaintance with this mobile platform. Fortunately, this operating system does not create any particularly difficult obstacles to achieving an appropriate result. Probably, among all three OSs (and we are talking about Android, Windows Mobile, IOS), the first is the easiest to set your own ringtones for various purposes. In view of this, the question of how to put a melody on a contact on “Android” becomes even easier. In the meantime, it's time to get down to business from words.

How to put a melody on a contact on an “Android” device





Many users are unhappy with the default set of ringtones and sounds that the operating system offers them. They can be applied to several types of categories at once. We are talking about incoming voice calls, an alarm clock, text messages and much more. Nevertheless, such a set of alerts does not suit, for example, young people who want to set original songs and music on calls. That is why the developers of the operating system provided a similar opportunity. It turned out that even without using third-party software (for example, Ringtone Maker utility), you can set your ringtones to calls. And to do this is not so difficult. Thus, the time has come to answer the question of how to put a melody on a contact on “Android”.

Create folders





The first step towards achieving the appropriate goal will be to search for a specific directory. To do this, open the file manager in the device itself . If you have already installed special software for these needs (we are talking about file managers from various developers), you can successfully use it, there is no fundamental difference here. Of course, FMs differ in functional part and interface, but the meaning always remains the same, so the actions in each of the programs will almost copy each other. Thus, using the file manager or file manager, we go to a directory called SDCARD.

Further search





In fact, after you go to the folder called SDCARD, you are in the root directory of the memory card. This is where all the other folders that are written on it and, of course, each file are located. Next, we need to find a folder called DCIM. When you find it, click and open this directory. There may be other folders in it, but we do not pay attention to them. Just create another directory and give it the name Media. After the operation is completed, you can see a new object resting in the DCIM directory with the corresponding name. So we took another step towards the answer to the question of how to put the melody on a separate contact. “Android”, as we see, is extremely simple in this regard, and then you will need to open the newly created folder and create another directory in it, which we will call Audio. Consider that half of the operation is already behind you.

Android operating system: how to put a melody on a contact’s ringtone

So, in the Audio directory, we need to create several more folders: alarms - for sounds set to the alarm, ringtones - for voice calls, notifications - event melodies, and ui - sounds of the operating system interface. Actually, you will need to download tunes to these directories. This can be done using a microUSB cable, as well as wirelessly.

After all the melodies are arranged in their directories, it remains only to go to the contacts menu and select the necessary person. In the context menu, you can find the sub-item “Set ringtone”. After you click on it, a list of tunes to choose from will open. There you already select the song you need.
