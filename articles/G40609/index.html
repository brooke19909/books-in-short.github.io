How to update Android

The Android operating system tops the list of the most accessible and common, so developers regularly release various updates and improvements for it.

If the question arose, how to update Android, then we can suggest a universal step-by-step instruction.

Warning: along with any update, all files, contacts, messages and other personal information are lost, so first you need to save all the data to a memory card and duplicate it to your computer if possible.

So how to update Android:

1. Connect your Android device to the Internet, go to settings, find the system data section. Most often this is ‚ÄúGeneral System Settings‚Äù, ‚ÄúAbout the System‚Äù.

2. Turn on automatic updates, if possible. If systematic downloading of updates is undesirable, then just click "Update." If the server has a fresh version for your type of Android, then it will be installed automatically.

Warning: the update "weighs" quite a lot, so it is better to use unlimited Internet or connect to a free wi-fi connection.

3. How to update Android if the above automatic update did not happen? It is best to use the services of the Android Market. There you can find updates for specific models or use an application that will independently search for the necessary updates.

4. Another option: on the official Android page you will definitely find the answer to the question ‚ÄúHow to update Android‚Äù. If the update, in principle, has already been created, then it is freely available there, and it can be downloaded manually. More recent firmware can be safely downloaded and installed as a regular application.

5. If you are unsure how to update Android firmware yourself, you can always use the services of the corresponding service centers. There they will perform an update for little money, and at the same time they will find out the reason why it did not happen automatically.

Now, let's take a concrete example of how to upgrade Android OS to version 2.3 of Gingerbread for HTC Desire.

1. We save all personal data on the memory card, on the computer and reset the settings to the original, in order to free up all possible memory space. If you do everything right, you get 136 MB of free space.

2. Download the firmware, best from the official site of HTC. After downloading, 4 files will appear in the archive: the firmware archive itself, a game with a ball, a flashlight and an archive of background images

The developers specially made standard applications in the form of separate files, since the OS itself takes up a lot of space. If you do not need these applications, then you can not install them.

In general, the difference between the factory version 2.2 and the updated 2.3.3 is 8 MB, which, you see, is not critical at all.

3. Unpack the archive file with the firmware - there will be two files in it: one with instructions in PDF, the second - the actual installation EXE. After starting the installation file, it is automatically unpacked into a temporary folder.

4. Connect the NTS to the computer via the USB cable. It is best to connect directly to the system unit, and not to the monitor, otherwise you may receive an error about the wrong connection. If synchronization is successful, a notification appears that the computer "saw" your smartphone. If you have not previously synchronized these devices with each other, then download and run the HTC Sync program.

5. Check the battery charge of the smartphone - it must be at least 30%, otherwise an error message will appear.

6. Run the firmware file, then proceed according to the dialog box. In the windows that appear, check the box and click "Next."

7. After the third dialog box, the connection of the smartphone to the computer is checked. If everything is in order, the ‚ÄúUpdate‚Äù button appears in the next window. Press this button, a new window appears, in which the current version of the OS and the one you are going to change it are registered. Click ‚ÄúNext."

8. After that, the last dialog box before installation appears, in which using ‚ÄúNext‚Äù you need to confirm that you really want to install the new OS.

9. The installation process begins, which takes about 10 minutes. At this time, you should never disconnect the USB cable.

10. When the process completes successfully, the ‚ÄúFinish‚Äù button will appear. Click her.

11. Now it remains only to restore all contacts, settings, personal data and enjoy the new operating system,
