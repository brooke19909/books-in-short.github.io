What is this technology?

Since its introduction in 2014, the Apple Pay payment system has been developing quite successfully. It is based on a concept that boils down to the fact that for the purchase of a product there is no need to carry a wallet with money, but just take a smartphone. The device must be equipped with an encoding chip, an NFC transmitter, and special software. All this will allow you to wirelessly make a transaction, for example, with the iPhone 6 Plus. Again, the security level of this technology is quite high and it‚Äôs easy for an attacker not to hack it - financial means are adequately protected.

Initially, this technology was used only in the United States, but now it has found its application in most parts of the world. In addition to Apple, other companies have provided similar programs: for example, Google and Samsung. That is, this technology will find more and more application every year.

Separately, it should also be noted the fact that you can use this payment service not only on smartphones, but also on other devices of this eminent manufacturer. Support for Apple Pay is present on both the Apple Watch and iPad tablets. This approach makes it a universal tool for shopping both online and offline stores.

Mobile Device Requirements

The list of key requirements for a mobile device to work with this payment system is as follows:

  1. The presence of an NFC transmitter that allows you to transfer data to a payment terminal.
  2. Installed Wallet application. It is in it that payment cards are added.
  3. Latest software version for more security.
