So, we connect the device to the PC via USB and launch iTunes. We go to the "Edit" section, then "Settings" and "Devices", after which we erase all backup copies of your iPad so as not to get lost in them. Next, make a new backup in iTunes, selecting the desired device in the sidebar, and click "Create copy now."

Next, switch to iBackupBot and go to System Files - HomeDomain - Library - Preferences. There we are looking for a file called com.apple.springboard.plist. For security, you can save and copy the original file by right-clicking on it, choosing Export Selected Item.
