How to put a password on an application or folder on an iPhone?

Every year in Russia becomes more and more happy owners of the iPhone. In this connection, in an effort to protect their personal data from prying eyes, Apple product users are wondering how to put an application password on an iPhone or password protect a photo folder.

Program Password

There are several ways to set a password for an application installed on your phone on an iPhone. But the device provided is not very convenient to use. One of its few advantages is the lack of the need to install additional software, allowing you to save more free space.





To set a password in the standard way, go to the "Settings" menu, where you should select "General" and "Limitations". In the window that opens, the "Enable restrictions" button will be active by default - after clicking it, you will be prompted to enter a 4-digit code. This password will be requested each time the selected applications are launched. To determine the list of programs that will be protected, it is necessary, scrolling below, to mark the necessary ones from the proposed list. It is worth noting that in addition to opening programs, you can password protect the installation or removal of applications on the device.

Alternative methods

More convenient to use is the method with the installation of special applications. How to set a password for an application on an iPhone depends on the selected defender program. However, for most of them, the principle of operation and adjustment are the same. As an example, it is worthwhile to figure out how to set a password on one of the most popular applications - iAppLock, which is freely available in the App Store.

To set the security code, you must run the defender program and select the type of protection and the necessary applications in the settings. One of the advantages of using additional software is the ability to select the type of protection: graphic code, digital PIN, or regular password.

Folder Password

Studying how to put a password on a folder, you can see that it is possible to also use both standard iPhone tools and install special programs. By default, you can password-protect any folder in the same way as applications. In the "Limitations" section, at the bottom of the list of applications offered for setting a password, you must select the appropriate folder.





It is worth noting that such a case will protect access to files in the folder only when entering through the standard file manager. Any other application trying to access closed files will be able to do this without calling the PIN code input field.

Blocking Analogs

As for applications, there are several dozens of different programs for setting a password on folders. Each of them allows in a certain way to block free access to files inside the directory. One of the most common means of locking is Folder Lock, which can be downloaded for free on the App Store.





After the installation of the application is completed, the corresponding item appears in the Settings menu, in which you can select folders and the type of protection. The program also allows you to password protect a number of other iPhone elements, including "Settings", a list of contacts and a camera.

Other protection methods

However, to protect your personal data, you do not need to know how to put the password on the iPhone to the application or folder. If you just want to close access to some files or programs by strangers, you can hide them from the main menu or file manager. Thus, you can not be afraid to forget the password or pattern for accessing data at a crucial moment. To open hidden folders or applications, just return their display in the menu. For greater convenience, there are special programs that allow you to do this in a few clicks, but using them will increase the likelihood that data will be detected.
