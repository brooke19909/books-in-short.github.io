Work with windows

To do this, you need a computer on the appropriate system, a cable for connecting and the smartphone itself. The system will open a window with autorun. Here you need to select the appropriate mode. It is recommended that you select "Open device to view files."

Work with Windows 10

How to transfer photos from iPhone to a computer with Windows 10? The previous method is not always effective. Sometimes a device refuses to work as a file manager. Therefore, you need to use third-party programs. One of those is in Windows 10.

It will open, and all synchronized images will appear in front of the user. In the upper right corner, select the "Import" command. Next, you need to select a device that is connected via USB. The iPhone will appear in the list.

Now we select those photos which need to be saved in system. To do this, you can select one or more, and then click the "Continue" button. Upon completion, all current pictures can be found in the system.
