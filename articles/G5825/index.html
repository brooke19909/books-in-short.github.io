Lenovo B50 30: installing Windows 7. Step-by-step instructions

Lenovo has long been known to the vast majority of users as a manufacturer of high-quality and inexpensive laptops. Laptops have good technical specifications. However, there are incidents and this manufacturer. No one is safe from this. For example, on the Lenovo B50-30, installing Windows 7 by default was not possible. Why did this happen and how to install the "seven" on this laptop, if it is just out of the store? We will understand this now.

Why did this happen?

Lenovo had no back thought. Just this laptop model was developed and sold in 2013. Let's recall what Microsoft did at this time? Right. In every possible way (at times very aggressively) they promoted their new operating system - Windows 8. Some manufacturers were forced to somehow limit the ability to install the "seven" on their devices. On the company's official website for the Lenovo B50-30 model, Windows 7 drivers are completely absent.





The most interesting thing is that updating the BIOS did not solve the problem in any way (although it should have). Curses and all kinds of calamities rained down on the company. But all this did not last long. Craftsmen quickly solved the problem. It turned out in the BIOS settings. By default, the manufacturer preinstalled options that made the installation of the "seven" impossible. And if you consider that for most BIOS users - a dark forest, a kind of "feint with ears" quite worked for Lenovo B50-30. How to install Windows 7 on this model? Easy peasy. And now you will see this.

BIOS settings for installing the "seven"

First of all, you need to go into the BIOS settings. To do this, when loading a laptop, you need to press the key combination FN and F2. Now go to the General Settings tab. Here you need to select Load Default Settings and click OK. Next, go to the Boot tab and in the line OS Optimized Defaults select the item Other OS. Now it remains to exit BIOS, not forgetting to save all the changes.





In some cases (especially difficult), you still have to install the "eight" and use it to update the BIOS firmware. Sometimes this is necessary, since the stock BIOS does not always give the user the opportunity to change the operating system in the boot options. But after the update everything should be fine. Next, you can restart your Lenovo B50-30. Installing Windows 7 is now possible.

Installation process

It is not much different from the standard. In fact, no different at all. But it is better to describe it all. So, the installation algorithm of the ‚Äúseven‚Äù on the Lenovo B50-30 after all manipulations with the BIOS:

  • We insert the USB flash drive or disk with the Windows installation files.
  • During loading, hold down the key combination FN and F12. A dialog box will appear.
  • In the window, select the media (CD-DVD for disk or USB-HDD for flash drive). After that, the installation of the system installer will begin.
  • In the next window, select the installation language and country of location. Click Next.
  • Now select the item "Install Windows Only" and create a disk layout. Remember that the "seven" is placed only on the NTFS partition. The remaining sections can be adjusted after the installation of the system.




  • Click the "Next" button - the installation process begins. The laptop will restart several times.
  • After the last reboot, you will be prompted for a username and password.
  • Click the "Next" button and wait. After a while, the installed system will start.

You have dealt with your Lenovo B50-30. Windows 7 installation is complete. However, before using the system, you will have to configure something else. Device drivers

What is the Lenovo B50-30 laptop problem? WiFi drivers (Windows 7) or any other drivers for the "seven" on the official website are missing. And Wi-Fi is critical. Indeed, without the Internet, other drivers cannot be downloaded. There are several options here. You can download drivers from the G8. In some cases, they are suitable. Or you can go to the site of the manufacturer of the component and download the latest version from there. But in any case, you will need a second computer with Internet access.





Sometimes on the websites of equipment manufacturers there is no software for the Lenovo B50-30 model. However, the driver for Wi-Fi will be anyway. It is worth putting only him. And further problems can be solved through the "Device Manager" in the installed OS. It allows you to update hardware drivers from official Microsoft servers. While the "seven" is officially supported, they will not go anywhere from there. The main thing is to get the Internet on this piece of hardware. Once you do this, living right away will be easier.
