Set-top box MAG 250: specifications, instructions, settings and reviews

The MAG-250 is a set-top box that works over IP. A compact but powerful device provides a wide range of multimedia services.

Description of IP Set-Top Box

The case of the MAG-250 is small and stylish. The front panel has a USB connector, an infrared receiver for the remote control, and a standby indicator . On the rear panel there is an additional USB, HDMI, Ethernet connector, composite video and audio output, digital audio output and a power connector. Slots are made on the side panels for ventilation of the device.

In addition to the MAG-250, the kit includes:

  • RCA audio and video cable;
  • remote control;
  • two AAA batteries;
  • external power supply.




MAG-250: specifications

The device allows you to:

  • play high definition video;
  • View multicast streams
  • create a list of television channels;
  • Download a pre-prepared list of channels from the network;
  • preview shows
  • Get access to Internet services such as YouTube, Picassa, etc .;
  • connect external devices via USB, including flash drives, Wi-Fi adapter, keyboard, mouse, etc .;
  • play audio and video of various formats;
  • viewing photo standards JPEG, PNG, BMP, GIF, RAW;
  • decode MPEG2, H264, WMV9, MPEG4P2, VC-1 video streams;
  • decode MP3, AC-3, MPEG2-audio, MPA, M4A, WAV, Ogg, AAC standards audio streams;
  • convert the video format to a format supported by the TV;
  • play media from UPnP servers;
  • play media from USB.

Watching video in HD-format is only possible on television receivers with an HDMI connector. In addition, you will need an HDMI cable, which is not included in the set-top box.

Using composite video output does not allow viewing channels in HD format. At the same time, they are displayed in SD quality.





Connector Assignment

The following outputs are located on the rear panel:

  • USB 2.0 for connecting a flash drive, Wi-Fi adapter, keyboard, mouse, etc.
  • Ethernet for connecting a wired LAN of 100 Mbps.
  • HDMI for connecting to a television receiver.
  • S / PDIF for connecting to speakers.
  • AV - composite video output for connecting to a TV via an RCA cable.
  • DC - 5V power connector




Network connection

The next step in the preparation of the MAG-250 is to set up an Internet connection. There are two options:

  • direct network connection;
  • through the router.

When connecting the MAG-250 through a router, the procedure is as follows.

  • Connect the Ethernet network cable to the corresponding connector of the console and the router.
  • Connect the HDMI cable to the MAG-250 connector and the TV. If there is no HDMI input, use the AV cable and the composite input of the TV.
  • Connect the power source.
  • Configure a router.
  • Turn on the TV and use the remote control to select the signal source (SOURCE, HDMI or INPUT).




If the MAG-250 is connected via an HDMI cable, select the HDMI source after pressing the INPUT or SOURCE button. If through composite, then on the remote control you need to press AV.

Channel setup

The next question MAG-250 users are asked is how to add a playlist? To do this, save the m3u file to a flash drive and insert the latter into the USB port. Go to the viewing form on the USB drive and select a playlist. After the offer to save it in UTF-8 format, confirm by pressing the OK button. Add playlist completed.

It is also possible to add a list of TV channels by specifying a direct link to it. To do this, go to the "TV channels" section, press the "Menu" button, press the ‚Üí key and select the "Download Playlist" section. Set Reminder

You can set a reminder for the program or movie that you want to watch. When it is time to start transmitting, the MAG-250 will offer you to switch to the specified channel.

To do this, follow these steps:

  • select a channel in the menu and go to the list of nearest shows;
  • Using the navigation keys on the remote control, select the program of interest;
  • set the reminder with the "OK" button.

A mark appears on the program. To cancel, click "OK" again. The list of reminders can always be viewed by pressing the blue button on the remote control.

If the channel has several audio tracks, then you can switch between them with the yellow button on the remote control.

To select the video format mode, press the green button on the remote control one after another.

  • step value in seconds;
  • duration of playing content;
  • indicator of the current position.

Each press of the ‚Üê and ‚Üí buttons changes the current position by 1 step, the value of which can be changed with the ‚Üë and ‚Üì buttons from 5 to 900 s. The default step is 60 s.
