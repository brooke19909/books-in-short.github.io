What is a transistor?

A transistor is a radio-electronic component created from a semiconductor material. Usually it has three outputs, it is able, using a small input signal, to control a significant amount of current in the output circuit. It can be used for amplification, generation, switching, and conversion of various types of electrical signals. Now the transistor can be considered the main device, the vast majority of electronic devices.
