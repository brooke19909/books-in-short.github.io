How to remove ads from your phone in several ways

Varieties of advertising inserts

Depending on the implementation method, advertising inserts are divided into several groups. The first includes pop-ups that appear when browsing the Internet. Here you can also attribute various automatic redirects (redirects in the browser) to unwanted resources.

The second group includes advertising modules built into the programs that cause the content to be displayed on the phone screen when there is access to the Web. And finally, the third, most unpleasant group, is a spontaneous redirect to third-party sites in any installed browser. It is easy to guess that the answer to the question of how to remove ads on an Android phone depends on the way it is displayed.

Dummy Address

One of the ways that has long been successfully used on desktop systems is editing hosts. This special file contains instructions on how to perform redirects for the entered website addresses to internal IP. The structure is extremely simple: on the one hand, a list of names of network resources is presented, and on the other, Internet addresses.

For those of them that need to restrict access, compliance 127.0.0.1 is prescribed. When requested from the browser, the operating system first checks the hosts file, and if there is a match there, then there is no data exchange with the resource. How to remove ads from the phone in this way? For this, the easiest way is to use special programs. One of them is Ad Away. First you need to install the program and run it. An important nuance: making changes to hosts requires root rights.
