Change Apple ID: step-by-step instructions, tips and tricks

Apple ID is the name of the apple profile. It is required to work with all the applications and features of Apple products. You can create an account for free. But what if it needs to be changed? Can I change my Apple ID? And if so, how to do it? Each owner of the "apple" device should deal with this issue. Otherwise, you can lose the account with all its data or not clean the device from its downloads before selling.

Possibility of adjustment

Can I change the number in the Apple ID? Yes. Changing Apple ID on Apple devices is possible in many ways.

Firstly, it seems possible to change the e-mail used to work with the account. This technique will help to save data for further use with another email.

Secondly, you can reconnect a new account on the device. In this case, the data from the previous Apple ID will be erased from the phone or tablet.

Next, we will try to study all the existing scenarios. They can be useful to every owner of "apple" products.

Mail address

Need to change your Apple ID? Then for this it is necessary to clarify what exactly will be subject to adjustment.

To begin, consider the process of transferring data from an "apple" device to another email. So the user will be able to free previously occupied Apple ID e-mail.

To cope with this task, you will need to perform the following steps:

  1. Open the Apple ID profile management page.
  2. Carry out authorization in the system under your name.
  3. Click on the line with the inscription "Profile Management".
  4. Select the "Change" option located near the e-mail.
  5. Dial the new email address.
  6. Confirm saving adjustments.

After the steps taken, the user will change the Apple ID login. It is very convenient, especially if you want to free a previously occupied email address from your "apple" account.

Reset from device

How to change Apple ID on iPhone or other Apple device? Usually, such a question begins to bother users before selling the corresponding device. First, you need to reset the previously used Apple ID, and then log in to the new profile.

To implement the task it is recommended to adhere to the following algorithm of actions:

  1. Go to the settings of the mobile device.
  2. Click on the Apple ID.
  3. Select the "Exit" option.
  4. Confirm request processing.

There is nothing difficult or incomprehensible in this. After the user reset the old account, he will be able to log in to the new "apple" profile on the device.

  1. Connect the "apple" device to the computer via USB.

Account Password

How to change the Apple ID in the iPhone, found out. First you need to reset the old profile, and then just log in to the new account. What to do if a user has forgotten his password? And if necessary, changes to this component?

It's time to think about resetting your Apple ID password. You can change it by e-mail or by using security questions. First, consider the first trick. It allows you to reset the password if the user has access to the mailbox used for the "apple" account.

If you want to change your Apple ID password, you will need:

  1. Go to the Apple ID management and authorization page.
  2. Click on the "Forgot your password?" Hyperlink located under the authorization log.
  3. Indicate your "apple" identifier.
  4. Choose a password change method. In our case, click on the line with e-mail.
  5. Go to the email account used for Apple ID.
  6. Open an email from Apple tech support, and then follow the password recovery link.
  7. Create and print a new password.
  8. Save the received data.

Fast, easy and very convenient! But what if the Apple ID is known, but there is no access to the box? It's time to think about changing the password for the account through security questions.

Recovery Questions

If you want to change your Apple ID, but the password is forgotten, help questions will help.

To achieve the desired, you will need:

  1. Repeat the first 3 steps from the previous manual.
  2. Click on the block "Security Questions".
  3. Answer the questions asked.
  4. Click on Continue.
  5. Create a new password for authorization in the system.

By repeating the appropriate combination and saving it, the user will get access to the Apple ID profile. Now, if necessary, you can change the email address used.

Reset questions

And you can change the control questions in Apple ID. This method is effective if the user knows the password for his account.

Want to use a similar technique? In this case, you must adhere to such an algorithm of actions:

  1. Go to the Apple ID website and select the "Forgot ...?" Option.
  2. Indicate the command "I want to reset the questions."
  3. Enter your Apple ID password, as well as its ID.
  4. Identify the user. To do this, just follow the prompts that pop up on the site.
  5. Indicate new control questions along with answers to them.
  6. Press the button responsible for saving the new data.

This operation is also not very often used in real life, but it can be useful to every owner of the "apple" device.

Recovery from the device

If you need to change your Apple ID (password or security questions), you can resort to using the mobile interface. This technique usually occurs when working with a PC is not possible.

To get into the mobile data recovery form from Apple ID, you need:

  1. Open the AppStore and iTunes Store section in the settings.
  2. Click on Apple ID.
  3. Click on the block with the signature iForgot.
  4. Write an "apple" identifier, and then select the option you need.

After that, you can resort to the help of the manuals we have already studied. From now on, it is clear how it is proposed to change data in Apple ID.

Important: if the user is unable to cope with this task on their own, it is recommended to write to the support service. To help her, she will have to prove ownership of the mobile device.
