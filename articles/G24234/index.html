ASUS P5B Plus. Motherboard review

Choosing a motherboard is always a very difficult matter. And choosing a budget motherboard is a hundred times more difficult. There are many factors to consider. The main one is the ability to connect all modern devices. The type of processors supported is also extremely important. In this regard, the motherboard ASUS P5B Plus looks very attractive. Let's analyze this product in more detail.





What is this fee?

Motherboard ASUS P5B Plus Xeon - a budget solution for weak computers. This board came out back in 2007 and has already managed to become obsolete. A sign of the "old age" of the product is the presence of a special IDE connector for a floppy disk, which no one has been using for twenty years. Nevertheless, this motherboard is quite suitable for creating a computer with good multimedia capabilities. But for games such a solution does not fit. Only the earliest versions can be launched.

Specifications

ASUS P5B Plus is based on the LGA 775 socket . The chip used is Intel P965. This means that it quietly supports processors such as Core 2 Duo. But the newfangled Core is not available to her. The "chip" of this motherboard is in a very high-quality sound chip. Typically, manufacturers supply boards with an inexpensive and not very high-quality chip from Realtek. It also has a more advanced chipset from Analog Devices, which features high-quality sound. The motherboard supports it in standard 7.1.





Let's move on to a deeper analysis of the ASUS P5B Plus. The characteristics of this board are as follows: the supported processors are the Duo family from Intel, some models from AMD, the system bus frequency is 1066 MHz, the memory type is DDR2, and the Ethernet data transfer speed is 1000 Mbps. Features are pretty standard for 2007. Such options were common for inexpensive motherboards. Some of them are relevant in our time.
