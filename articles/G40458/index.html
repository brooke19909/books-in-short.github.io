We write letters from the tablet: mail setup on iPhone

If you have purchased any of the iPhone product line, then you will undoubtedly want to use its capabilities to the maximum. One of these features is synchronization with the mail server for receiving and sending messages. By default, this feature is not available, but proper mail setup on the iPhone will help. There is nothing complicated here - we will now analyze the process in stages.





How to set up mail

First way

If you have an account in one of them, then everything is quite simple - you just need to select the appropriate item and fill in the necessary and intuitive input lines. After that, the mail is automatically configured on the iPhone.

Second way

If you do not have an email account, then you can create one. The most optimal solution, in our opinion, is Gmail - mail from Google. Its interface is quite easy to use, so it is very popular among users.



Do not want to open an account with one of the proposed providers - then select "Other". In this case, setting up mail for iPhone may take a longer time.

First you need to configure the SMTP and POP3 protocols, which are used so that third-party applications can access the mailbox.

In the column "Host Name" enter exactly the address that was specified in the settings of your account. For the username, enter the address of your mailbox. In the next field write the password.
