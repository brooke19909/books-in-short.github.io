How to remove a ringtone from an iPhone? Required:

  1. Download and install iFile.
  2. Open the main menu of the device and launch the mentioned application.
  3. Go to var / mobile / media / iTunes_Control.
  4. Go to the Ringtones folder.
  5. Swipe left on the desired document.
  6. Select the "Delete" function.
  7. Confirm the process.

iTunes to help

We found out how to remove ringtones from an iPhone without a computer. Perhaps this is the only solution that can be advised. In all other cases, you have to use a PC.

Let's start with the simplest and most correct approach - using iTunes. Every owner of an apple device is familiar with this application. The program serves to work with devices from Apple in different directions - from restoring settings to downloading new documents.



How to remove a ringtone from an iPhone? You can act like this:

  1. Install the latest version of iTunes on your PC. It is important that it is not older than 12 assembly. Otherwise, the user will not find the necessary menu item.
  2. Connect the iPhone to the computer using the USB cable.
  3. Run the mentioned application.
  4. Create a library ("File" - "Add to Library ...").
  5. Go to "Devices" and select the connected smartphone.
  6. Open the General tab.
  7. Check the boxes for "Manually processing music and video", "Synchronize only marked songs."
  8. Go to the "Sounds" section.
  9. Check the box next to "Sync."
  10. Click on "Apply."
  11. Do the same with the "Music" section.

Thus, it will be possible to remove ringtones and songs from the "apple" devices. It is important to understand that all documents not added to the library are subject to cleaning. In addition, you can get rid of this way only from documents downloaded via iTunes.

iTools and ringtones

How to remove a ringtone from an iPhone? Required:

iFunBox

You can use the iFunBox program to translate ideas into reality. Typically, in this case, users work with iOS and MacOS.

The guide to removing music and ringtones is as follows:

  1. Install iFunBox on the computer.
  2. Connect iPhone to PC using USB.
  3. Launch AiFanBoks.
  4. Click on "My Mac".
  5. Select "Raw File System".
  6. Go to var / mobile / media / itunes_control / ringtones.
  7. Click RMB on the desired document.
  8. Click on "Delete."

After a moment, the ringtone will be deleted. There is nothing difficult or incomprehensible in this!
