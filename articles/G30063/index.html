MIUI is very user friendly, as it always finds a smart solution to every problem. Disabling ads with Mi File Explorer:

  1. Open Mi File Explorer.
  2. Click the 3rd character icon in the upper corner of the Mi File Browser.

  4. To disable the advertising identifier, go to "Advanced settings" - "Privacy" - "Advertising services" - "Use advertising identifier" - "Disable".
