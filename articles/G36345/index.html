Is it worth buying the Apple Watch: gadget specifications, benefits, reviews

Apple Watch is a device that has appeared relatively recently. Of course, everyone already knows about him, and many managed to get to know him first hand. But some still doubt whether to buy an Apple Watch. How justified can such a purchase be?

About device

Apple Watch is a wristwatch with additional functionality. The first time they became known in 2014. Of course, they can work independently, but for full work, iPhone 5 and newer versions are required.

But not everyone has decided for themselves whether to buy an Apple Watch. It seems to many that this pampering can be dispensed with without hours. Some found the best assistant in this accessory, which became indispensable.

Since 2015, Apple has launched the release of this device. By 2018, four versions of smart watches are known. Of course, each model has its own characteristics. Owners of new Apple smartphones choose mainly the latest modification. But to understand whether it is worth buying an Apple Watch, you need to consider all versions.

Options Apple Watch

While someone was thinking about buying an Apple Watch, many were already considering the packaging of the device. She was noticeably large. But having removed the cover, it was possible to find a plastic pencil case, in which there were the treasured clock.

The device is placed in a special compartment, under which there was documentation and a charger. An optional shorter strap was supplied.

Appearance of Apple Watch

At first glance at the device, the question of whether to buy an Apple Watch 1 immediately disappeared. The gadget looked expensive and modern. The screen got rounded edges and a glass coating.

Since this is a watch, it could not have done without a dial. It was customizable, so you could install anything you like: from landscapes to cartoon characters.

For the body, aluminum was selected. The whole foundation is made of it. The glass that covers the display protects against scratches and other damage. At the bottom is a sensor that collects heart rate data. This means that the user receives not just a watch, but also a fitness bracelet.

Special attention should be paid to a hypoallergenic strap. It is simple, does not have any decorations. It is delivered in different color schemes. What remains special is its fastening - with the help of magnetic latches.

Apple Watch Features

Since this was the first watch model, it can be called somewhat experimental. The manufacturer needed to understand what the buyer wants in order to create the ideal in the future.

The first series acquired a 1.53-inch screen with an OLED matrix and a resolution of 390 x 312 pixels. The clock is always displayed on the main screen. If you swipe the display from top to bottom, a curtain with notifications appears, from bottom to top - the Glances menu.

In this menu, you can control all additional functions. For example, music on a smartphone or using a fitness bracelet. But there is also a main menu, to go to which you need to use the wheel on the side.

A cloud of icons will appear on the screen. It contains all the programs that are installed on the smartphone. From here they are easy to manage. You can use the button under the wheel. It is customizable, so thanks to it you can, for example, configure the call of certain contacts and send messages.

The equipment of the second series has not changed, so you can skip this moment. It is interesting that the appearance has not particularly changed. Features of Apple Watch 2

But some changes nevertheless affected the novelty. Despite the fact that the elements remained in their places, they began to work differently. Now the wheel did not call the main menu, but was only a tool for scrolling data. It also allowed you to call the application menu or return to the main screen.

The mechanical button has stopped working with contacts. Her task was to call application widgets. All other functions could be called using the touch screen.

Of course, the watchOS operating system has changed a bit. Many users began to talk about the fact that new options were missing in the previous gadget. Applications began to work much faster, because the system optimization has changed.

The manufacturer has focused on the health of the owner. Sensors began to work more accurately, and more fitness apps appeared in the Apple Store.

The screen of the novelty has not changed. Everything is also AMOLED 1.5 inches with a resolution of 390 x 312 pixels. The new building continues to protect the owner from water, so you can use the device in the pool and at sea.

The manufacturer decided to install a new SoC - Apple S3. Also appeared feature LTE. It was decided to increase the battery capacity. The changes affected the operating system.
