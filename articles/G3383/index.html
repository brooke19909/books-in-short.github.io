Mi Drop Xiaomi: what is this application and how to use it?

Xiaomi continues to delight its users with interesting "chips." It all started with the MIUI 7 firmware version. So, on the devices of this company, a new Mi Drop Xiaomi application has appeared. What is this utility, how to use it, is it possible to remove it? The article will give answers to these and other questions.

Xiaomi Drop Xiaomi - what is it?

In short, this program provides users with the ability to easily and quickly send and receive different files: pictures, videos, music and so on. The application works through a Wi-Fi connection. Both devices (the one that sends the file and the one that receives it) must be connected to the same Wi-Fi network. Each of them must also have access to the Mi Drop program.

However, not only Xiaomi device owners can use this service: the application can also be downloaded for free on Google Play and owners of smartphones of other companies on OS Android. You can use the service without an internet connection. The process of transferring and receiving files will be considered using the example of the Redmi 4A device.

What is Mi Drop on Xiaomi Redmi

The developers assure: transferring files via Mi Drop is faster than Bluetooth 200 times. In fact, this application is an analogue of SHAREit from Lenovo and AirDrop, a program for devices from Apple. How to enable Mi Drop on Xiaomi? You can start using the program by clicking on the application icon, which is located on the smartphone‚Äôs desktop (for example, in the photo below it is located in the fourth row).

If the user wants to share the file, you must click "Submit". Receive a file from another device - "Receive".

Or even simpler: after selecting a file from the gallery, player or file manager, you should click "Submit". Several options will be offered, Mi Drop - among them.

At this point, on the second device you need to go into the program and select "Receive." After a few seconds of searching, pairing between the devices will be established: the device icon with its name will appear. You need to "tap" on it - and the file will be transferred.

In fact, the program interface is very simple, and file transfer pleases with high speed. Even interrupted data transfers can be resumed.

Can I remove Mi Drop?

It is possible, but since this utility on Xiaomi smartphones is part of the firmware, it will reappear the next time the device is updated. Moreover, to remove it, root-rights are needed. Those who do not have them should not get rid of the application: it does not take up much space in memory, and the processor "does not bother". Nevertheless, if the user decided to remove the program, he needs to find MiDrop.apk via the file manager along the path / system / priv-app.
