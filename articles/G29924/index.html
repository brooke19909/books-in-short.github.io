How to find out ifns code: three ways

The IFTS code is a strict sequence of numbers that identifies a specific tax service inspection . The characters in this cipher are four Arabic numerals. The combination of the first two is the code of the subject of the Russian Federation, and the last two are the number of the particular tax office. And now we will pass directly to how to find out the IFNS code.

Method 1: on the website of the Federal Tax Service

The easiest way is to go to the FTS website. So, how to find the IFTS code at the address in this case:

  1. Open the main page of the site, find the "Electronic Services" section. Click on the "All Services" button.
  2. Among all the variety you need to find the "Address and details of your inspection" and select this service.
  3. The window that opens illogically asks you to enter the code of the IFNS you need. Without entering anything, click on "Next."
  4. In the next window, enter the region. In the "district" field, you can enter the name thereof, if it is appropriate, if not, then skip the line. Next, enter the name of the city. Then the name of a smaller settlement - village, village, district center, if relevant. In the case when you live in a large city, you must also indicate the street on which the inspection is located.
  5. Click on "Next" again.
  6. How to find out ifns code now? In the next window, in the subheading "Details of the IFTS" you will see the four-character code you need. That's all!




Method 2: by TIN

How to find the IFTS code if you are filling out a declaration or other document for the tax service? There is another fairly simple way - take a look at your TIN, an individual number that is available to each taxpayer - a private or legal entity. The first four characters are the inspection code in which you are registered.

Method 3: by reference

How can I find the IFTS code if you cannot use the service? In this case, downloading the SOUN-directory containing codes of tax authorities, as well as other information useful to the taxpayer, will help you on your PC. In addition to the cipher you are looking for, in this collection you can find out about the full name of the inspection or other organization of the Federal Tax Service, its address, current phone number and information about the reorganization.





That's all the methods that help to find the code of the inspection of the Federal Tax Service. It is also possible to obtain information about him by directly contacting her, calling there or on the hotline, using the SMS service of the service.
