Samsung Galaxy Note 2: characteristics, instructions, reviews, photos. Samsung Galaxy Note 2 N7100

A large display diagonal, a powerful processor, an impressive amount of memory and a fairly capacious battery - all this is in the Samsung Galaxy Note 2. The characteristics of its hardware and software components will be given as part of this material.

This device is positioned by the manufacturer itself as a premium gadget. Accordingly, the equipment should be appropriate. But still, there are certain comments on it. In addition to the mobile device itself, the boxed version of the device includes such components and accessories: a battery, an interface wire for connecting to a personal computer, a charger, a stylus, and an instruction manual. This list clearly lacks headphones, a protective film, a cover and a memory card. In the first case, there is even a corresponding port, but there are no stereo headsets in the delivery set. Although the front panel of the device is protected with 2nd generation Gorilla Eye glass, an additional film to preserve its original state would clearly not be superfluous. The situation is similar with the sides and the back cover - they are also best protected, but only with the help of a cover. Although the capacity of the built-in drive is equal to an impressive 16 GB, you can expand it by installing an external flash card. This will allow more rational use of the memory subsystem in Samsung Galaxy Note 2. The characteristics (instructions indicate this nuance) of the external drive are as follows: the type of micro-SD card, and its maximum capacity is 32 GB. This will definitely be enough to place the user's personal files (photos, videos, books, music) here.

Device memory

Samsung Galaxy Note 2 N7100 boasts a well-organized memory subsystem. The amount of installed RAM in it is 2 GB. The size of the integrated internal drive is 16 GB. At the same time, the user can use only 11 GB to install his programs and store personal data . This is quite enough for comfortable and normal work. The rest of the space is occupied by system software. If 11 GB of free space of the internal drive is not enough for the user, you can install an external flash card of 32 GB. The corresponding slot is in this device. This certainly will solve the problem of lack of free space in the device. Well, the most valuable information is best stored on some cloud service. This will allow it to be restored in the event of a breakdown or theft of the device without problems.





Autonomy

The dimensions of the device are impressive (one display diagonal of which is only worth 5.55 inches) and with a small battery, it would not have been able to boast an impressive degree of autonomy. As a result, the battery capacity is 3100 mAh in the Samsung Galaxy Note 2. The characteristic, of course, is far from outstanding today. But this value is quite enough so that the device with an average level of load on one battery charge could last 2-3 days. Taking into account the diagonal of the screen of the device and the hardware filling, this value cannot be significantly increased. If you run some resource-intensive application (for example, Asphalt 8), then the battery life will be reduced to 6-8 hours.

Data exchange

An impressive list of ways to exchange data with the outside world is implemented in the Samsung Galaxy Note 2. Characteristics in this regard, he has the following:

  • Support for mobile networks GSM and 3G. True, the device has only one slot for installing a SIM card, and it cannot work in the "DUOS" mode. In the first case, you can only view unpretentious sites on the Internet and exchange messages on social networks. But with the help of 3G technology, all these restrictions are removed, you can even watch online video.
  • Korean developers have not forgotten about the "Bluetooth". It is indispensable in cases where you need to connect wireless headphones to a smartphone or share small personal files with a similar mobile device.
  • Also in the device there is a Wi-Fi module. It, just like 3G, allows you to receive and send data to the global web in any quantity and at good speed. In this case, the most "heavy" sites will load quickly enough.
  • For navigation in the device provides two systems. One of them is tied to satellites and can work with both JPS and GLONASS. Well, the second is based on the use of the location of mobile communication towers and is called A-JPS.
  • Among the wired interfaces, the presence of the previously mentioned micro-USB and audio port can be distinguished. The first is for charging the battery or connecting to a PC. The second is used in cases where you need to output an audio signal to an external speaker system.
