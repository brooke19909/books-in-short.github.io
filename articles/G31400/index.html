Service Description

The service of "transferring" calls, or "forwarding", has been popular almost since its inception. Indeed, this function can be useful at any given time to any subscriber of a mobile operator. Its essence lies in the fact that all calls from one number can be forwarded to another. At the same time, when setting call forwarding, you can set the condition under which they will be forwarded. In other words, before making a call forwarding to MTS, you should determine the situation in which you need to redirect calls to another number.

There are several options:

  • if the number is busy;
  • if the device is not registered in the network or is simply turned off;
  • no answer;
  • all calls.

How to set up call forwarding to MTS for each specific condition will be described later.





Features of using the service

  • If you select the conditions for the redirection of incoming calls "No answer", you can independently set the time interval, after which the call will be sent to another specified number.
  • A ‚Äúredirected‚Äù call is paid: the cost depends on the price per minute of communication from the number from which it was originally made (more will be described later).
  • Before leaving the home region, you should disable the service. In the event that there is a need for it, then it is necessary to replenish the balance of the main number in advance.

Will forwarding help in roaming?

When in roaming, the call forwarding service will continue to operate, however, the cost of calls will increase sharply and will consist of the following items:

  • cost of a minute of an incoming call at roaming rates;
  • cost of a minute of an outgoing call at roaming rates in the direction of the redirected call.

How much does installation and temporary use of call forwarding cost?

The service is provided free of charge and does not imply a monthly fee. All charges that can be made are only possible with the acceptance of forwarded calls. The cost of such a call is equal to the price of one minute of communication with an outgoing call from the main number.

Thus, it is recommended to disable the service immediately after the need for it has disappeared - this will avoid unplanned costs for mobile communications.
