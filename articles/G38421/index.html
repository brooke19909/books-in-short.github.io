How to enter BIOS Lenovo G50? Recommendations

Lenovo laptops are extremely popular in Russia and other countries. This is due to the fact that laptops are reliable, unpretentious, productive and have an adequate cost.

The products of this Chinese manufacturer literally flooded all the shelves of computer stores. But along with all the advantages they have one small minus: in some models it is completely unclear how to enter the BIOS. And this is necessary when installing the system or making any settings. If laptops from other companies respond to keyboard buttons, Lenovo refuses to do this. Especially the G-series models. This begs the question: "How to enter the Lenovo G50 BIOS ?" We will try to answer it.





Why do I need to enter BIOS?

BIOS is a basic input / output system. It is she who allows you to communicate with a PC or laptop at the hardware level. BIOS settings greatly affect the performance of the computer as a whole. It is also necessary to go into it before installing the operating system in order to select the boot from the installation media, and not from the hard drive. In general, it is very useful to know such information, since it can be useful in the future. And at the most inopportune moment. Therefore, it is highly recommended to know how to enter the Lenovo G50 BIOS or some other equally capricious laptop model.

In addition, BIOS can be incredibly useful for restoring a computer. It is he who can reset the hardware to factory settings. Among other things, it is the settings of the basic input-output system that govern the operation of the hardware of the laptop. Therefore, we will analyze in more detail the question of how to enter the Lenovo G50 BIOS .

How to enter BIOS?

On regular laptops, the Enter or F2 buttons help. But a completely different story with the Lenovo G50-45. How to enter BIOS on this laptop?

  • loading into BIOS;
  • normal load;
  • boot menu;
  • System Restore.

In our case, we need to select the boot in BIOS. Here is such a tricky button.





Pay attention to the fact that on some Lenovo laptop models (for example, B580 or B590) this button can be located next to the power button of the device. However, on the above models there is another way to enter the BIOS - you need to press Enter at boot, see the line in the menu and press F1 to access the BIOS settings. In particular, this applies to the model B580. As for the Lenovo G50-30, we already know how to enter the BIOS of this laptop. Using the small and inconspicuous Novo key. Let's move on to the main BIOS settings on Lenovo laptops.

Settings for installing Windows 7

Recently, Lenovo laptops come preloaded with the Windows 8 operating system. But not everyone is happy with it, and when you try to install the "seven" on your laptop, the laptop starts to sprinkle with errors. A competent BIOS setup of the device can save the situation.

Consider the situation on the example of a laptop Lenovo G50-30. We already know how to enter BIOS, so let's move on to the settings right away. We go to the Exit tab and look for the OS Optimized Defaults item there. There is Windows 8. There we need to change it to the "seven". That's all!





However, after that it is necessary to load the optimal BIOS settings. So you need to do this before each installation of the operating system:

  • To do this, on the same Exit tab, select Load Default Settings;
  • then you need to click OK;
  • press F10;
  • restart the laptop.

After performing all these manipulations, the installation of the seventh version of Windows will be possible. No more problems.

How to change the boot order?

Sometimes you need to change the boot order of the device. That is, to make sure that the laptop is loaded not from the hard drive, but from the installation media (CD-ROM or flash drive). Consider the Lenovo G50 laptop :

  • We already know how to enter BIOS, so skip this step.
  • After pressing the notorious "New" button, a menu will appear where, among other things, there will be a Boot Priority item. This is what we need.
  • We go to this point and see a list of devices that are currently available for download. If you install the operating system using a USB drive, select USB-HDD.
  • And if an optical disk is used as the installation media, then select "ODD-device-name".

After that, the computer will start loading not from the hard drive with the preinstalled operating system, but from the bootable media. And you will be able to start installing the system.
