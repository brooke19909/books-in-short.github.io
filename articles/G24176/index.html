How to block a card of Sberbank?

How to block a card of Sberbank? It would seem that a simple question at first glance can cause a lot of difficulties. Never before having faced a similar problem, a person can come to a standstill. If a person has lost a Sberbank card, he should use one of the options below for blocking it.

How to block a Sberbank card - help desk will tell

One of the options for blocking a plastic card is to call the help service of Sberbank. This is one of the easiest ways. 8 800 555 5550 is a number that everyone should remember. Of course, not everyone is able to remember, therefore it is better to write it in your notebook. This reference service allows not only to lock the card, but also to unlock it. Among other things, it allows you to get the necessary information on the map. There you can get additional information on how to block a Sberbank card.

If your card has the Mobile Bank service activated, then you have the opportunity to send an SMS message to a simple number 900, in which you indicate the following:

1. Write the word BLOKIROVKA.

2. Indicate the last digits of the card (five should be written).

3. Write one of the proposed numbers:

  • if the card is lost - 0;
  • if the card is stolen - 1;
  • left a card at an ATM - 2;
  • another situation is 3.

The code that you will receive in the reply SMS message must be sent within five minutes to the number 900.

The card may stop working due to incorrect three-time input of the PIN code. But in this case, the card unlocks itself after 24 hours.
