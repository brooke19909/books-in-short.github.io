Knitting description

  4. Loops for a sleeve act on one additional knitting needle. Each sleeve will include 2 loops from raglan. The main part of the canvas 23 cm is knitted (62 rows will be obtained).
  5. Legs for overalls are knitted with socks. Knits 27 rows (about 16 cm). Cuffs are made at the edge - 17 rows are sewn with a 2 x 2 elastic band. Then, the 18th row is folded in half so that the elastic band goes inside.
