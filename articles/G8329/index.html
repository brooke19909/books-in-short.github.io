Hardware equipment

In Huawei GR3, the performance characteristics are quite consistent with the segment to which it belongs. The model is based on a processor from MediaTek. This is a fairly common practice. Each core is capable of overclocking to 1500 MHz. The manufacturing process is 28 nm, which is currently not the best solution. The tri-core accelerator Mali-T720 MP3 was chosen as the coprocessor. This video card operates at a frequency of 450 MHz. Two gigabytes of RAM supplement the performance characteristics, which is pretty good. The size of native memory is limited to 16 GB.
