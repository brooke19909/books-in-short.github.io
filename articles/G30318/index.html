How to close applications on Android: tips and tricks

When working with mobile devices, a certain amount of battery power is consumed. In addition, running programs load the operating system, which leads to software initialization errors, as well as slow phone or tablet operation. Next, we will talk about how to close applications on Android. It is this OS that people use most often. Therefore, it will be useful for everyone to know about the correct completion of work with programs.



Ambiguity

In fact, there is no single solution to the problem. It all depends on the phone model, as well as the software used.

Home and Background

How to close applications on Android? Some people prefer to click on the "Home" button. If you act in this way, then the program used will really close. More precisely, it will cease to be displayed on the screen, but nothing more.

Instead, the application will go into the background. Or, as some users say, it will be minimized. Some programs when you click on the "Exit" button go into the background. This fact must be taken into account. How to close background applications on Android?

Quick completion

Implement the idea will help the application manager. With it, on many Android devices you can finish working with programs and games.

The action algorithm is reduced to the following manipulations:

  1. Call application manager. For this purpose, a special button is usually placed on the phones. For example, at Samsung it is located to the left of the Home button.
  2. In the list that appears, wipe in the direction from right to left (or vice versa) near the desired programs.
  3. Click on the chart image. A list of active applications that have not completed their work will appear.
  4. Click on the "Finish" button.

How to close applications on Android? The answer to this question will no longer be difficult!



Dispatcher

But there is one more trick. It will help to complete the work with all background applications running on the device.

To use it, you need:

  1. Press the Home button and go to the "Settings" section.
  2. Find the "Task / Application Manager" there.
  3. In the list that appears, click on the button responsible for terminating all running programs. Usually it is in the lower right corner.

Some, thinking about how to close applications on Android, use the so-called "task killers". These utilities promise to complete all running processes with one click. For Android, such programs are harmful. You can use them, but doing so is not recommended.
