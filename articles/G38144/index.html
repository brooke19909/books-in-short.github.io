Software Update Ways

Working with pc

Let's start with the fastest and most reliable way. It's about working with iTunes. This program is known to all Apple product owners. It is extremely understandable, working with it is as easy as shelling pears.

How to upgrade iPhone 4 to iOS 8.1 via computer? The action algorithm is reduced to the following steps:

  1. Install the latest version of iTunes on your computer. It is advisable to download the program from the official Apple website.
  2. Run the appropriate application.
  3. Using the USB cable, connect iPhone 4 to the computer. The mobile device should be turned on at this point.
  4. Wait for the synchronization of the PC and iPhone through ITuns.
  5. In the left menu of the program, select the line indicating the connected smartphone.
  6. Find the item "Update iPhone ..." in the field that appears on the right side. Click on the corresponding inscription. Wait.
  7. Click on "Download and Update." This message will appear on the screen when it detects updates to the operating system.

Quick, easy, convenient. How to upgrade iPhone 4 to iOS 8 if there are no updates? In this case, you will have to change the firmware and independently find and download the software necessary for initialization to the mobile phone. In practice, such a need is extremely rare.



Wi-Fi Actions

  3. Go to the "Settings" section on your mobile device.
  4. Open the tab "General" - "Software Update".
