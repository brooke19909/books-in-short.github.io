Router Tenda N301: instruction, reviews, settings

They are suitable for undemanding users who want to install a router at home. This article will focus on the Tenda N301 router . We will tell you how to configure it for correct operation. Product Description

This device is one of the typical in the range. When transmitting data over the air, it gives about 300 megabits per second. Three different connection protocols are supported. There are two antennas. Both work with 5dBi. Device security is supported using standard WEP, WPA-PSK, and WPA2-PSK features. There is WEP with a connection speed of 128-bit. You can use WPS mode. The operating modes are identical. There is a router, a bridge. Built-in repeater. You can clone mac addresses, and Tenda N301 can also be configured via the global network. A repeater for this device, called the Tenda A301, has also been developed.



Tenda N301: connect and enter settings

Before you start working with a router, you must install it and connect it. This process is completely dependent on which device will be configured. If you are using a laptop or a regular computer, you must use a network cable connection. It can be found in the kit. Next, connect the router to power. The cable from the provider must be installed in the WAN port.

If you want to connect to the Tenda N301 via a Wi-Fi network, you just need to turn on the device. By default, the device is configured to distribute a factory network that does not have a password. It will need to be installed later in the configuration process. If you still have a password request, you must enter the numbers from 1 to 8. Sometimes, on some devices, the manufacturer sets such a code. Thus, you can connect to the device even from a tablet or phone, then directly from them and perform the configuration. The main thing is that the device is connected to the network.

For those who do not know how to work with a router and do it for the first time, the main thing is to understand that there will be no Internet immediately after connecting the device. To get the correct work with the network, you must configure the device to connect to the provider.

In order to enter the settings of the router, the Internet is not needed. On the computer, do not enable any connections. But then how to configure the Tenda N301 router ? To enter the settings you need to launch any browser and go to a special address. It is standard and has the form 192.168.0.1. Next, a page opens where you need to log in. There you should enter the same character set in the column password and login: admin.

Sticker Options

It should be noted that the address for entering the panel, the factory password (if any) from Wi-Fi, as well as mac addresses are indicated on the sticker, which is located on the bottom of the device. Next, click OK. If you can‚Äôt get into the control panel, you need to slightly change the computer settings. Typically, this problem occurs if the device does not automatically receive IP addresses. Next, the owner will see the main settings. Here you can specify the type of connection, change the password, set any other parameters that are necessary. It is advisable to open the basic settings and advanced options. This is where the adjustment itself will be carried out.



Setting Up Internet Connection (WAN)

First you need to make sure that the Internet cable is connected to the WAN port. This wire can be purchased either with a modem or directly from a service provider. Next, you need to do the most important thing - setup. You should set the parameters, thanks to which the router can connect to the Internet without any difficulty.

If these Tenda N301 parameters are set incorrectly, a Wi-Fi network will be created without access to the Internet. It is imperative to find out what type of connection a particular provider uses. Next, you will need to specify data that depends on the specified parameter. You should also check with the company if it uses MAC address binding. If the connection is dynamic and address binding is not used, then no additional settings need to be made. The router will automatically set the desired type of connection and the Internet will be available. Recall how to connect the Tenda N301 described above.

If we are talking about binding by MAC address, then in the settings, press the clone mac address key. If this is difficult, then you can call the company itself and ask them to register new data in the settings. Additionally, the operator will need to dictate the address of the router itself. After that, you can proceed to configure the network itself.



PPPoE setup on Tenda N301

You need to go to the panel in the "Configure Wan Connection" tab. Next, a menu appears where you need to select the type of connection correctly. It is necessary to indicate the necessary, then register the username and password that are set by the provider. As the address mode of Tenda N301, you should choose a dynamic type. Next, check the box next to "Connect automatically." To save the settings, click "OK". If these parameters were set correctly, then access to the Internet will appear.

Configure PPTP or L2TP

In the "Configure Wan Connection" tab, you must select the type of connection. Next, specify the server address. Just below you need to specify a username and password. In order to save the settings of the Tenda N301 router (reviews of which are described below), click OK. In PPTP or L2TP modes, you do not need to enable a high-speed connection. A similar function is activated automatically. The main thing is to properly configure the connection. As soon as the router starts connecting to the Internet, you need to configure the network.



Wi-Fi setup

If you need to change the name of Wi-Fi and set a password, then you should go to the ‚ÄúWireless Mode‚Äù tab. There you should enter an arbitrary name in the network name tab. If you want, you can leave the standard naming that was set by the manufacturer. If you wish, you can also activate the guest network here. To save the settings, click "Save". Next, go to the "Protection" menu. This is where you can change the password and set encryption. As a code, you can think of any set of 8 characters. Next, you need to save the settings. More detailed instructions for the Tenda N301 are included.

After the router is rebooted, a network with a password and a new name will appear. In order to reboot the device, press the corresponding key in the control panel. This completes the configuration of the router. It is advisable to go back to the settings and change the factory password to access the control panel. Otherwise, someone else can get into the device parameters and change its options.



Change admin password

You can change the password used in the System Management tab. There is a function for changing the password, displayed in a separate column. Next, you just need to specify the old code, and then two times new. Do not forget to save the changes. It is advisable not to forget the new password, otherwise you will have to reset the Tenda N301 wireless router again. In order to do this, you must press the RESET key, and then hold for 10 seconds.
