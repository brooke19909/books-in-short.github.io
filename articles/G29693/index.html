How to put a ringtone on an iPhone. Instruction manual

The iPhone, like any other phone model, has a set of standard ringtones that can be set on a call, SMS and alarm clock. But there is one caveat. If in simple phones, in addition to standard sounds, you can put any downloaded melody in .mp3 format on a call, then in an iPhone you won‚Äôt be able to do this simply. Yes, yes, this modern gadget is cunningly designed - Apple developers, as they say, tried their best. Therefore, we will not be surprised at all that you were interested in how to put a ringtone on an iPhone not from standard sounds. You will learn the answer to this question by reading this article to the end.



Set ringtone on iPhone

To get started, let's learn how to change the sounds that alert us to an incoming call. To do this, do the following:

  1. Go to the "Settings" menu item, then select the "Sounds" section.
  2. Listen to all standard tunes and, having chosen your favorite one, set it as a ringtone.

But for sure you are pretty fed up with standard sounds and you are not averse to somehow stand out from the crowd. So let‚Äôs now find out how to set the ringtone for iPhone, which will be downloaded from your computer. But keep in mind, you have to conjure a little over it, because a regular song in .mp3 format will not be installed on a call.

Create a ringtone in iTunes

This application is already installed on your device. But, unfortunately, not everyone can afford to use it, since there all the melodies are paid. But do not be discouraged if you have a computer / laptop, you can create a ringtone from your favorite song absolutely free. To do this, do the following:



  1. Install the iTunes application. You can download it from the official website of Apple developers.
  2. Launch iTunes and click on the "Album" tab.
  3. We select a composition from which we will create a ringtone.
  4. Click on it with the mouse and select "Details".
  5. In the window that opens, click on the "Settings" tab and mark the necessary interval for the future ringtone. Keep in mind, its duration should not exceed 30 seconds! Save the changes by clicking the "OK" button.
  6. Right-click on the file you just created and select "Create version for AAC format." After this action, a second track appeared on the list. The name of these files will be one, but the playing time is different. But that's not all, because before you put a ringtone on your iPhone, you need to redo it in a different format.
  7. Click on the file with the extension .aac and select the option "Show in Windows Explorer" / "Show in Windows Explorer". Thus, the track we created will open in Explorer.
  8. Copy it to your desktop, and remove it from your iTunes library.
  9. Change the file extension from .m4a to .m4r. To do this, just rename it.
  10. Open the Ringtones section. Drag and drop the .m4r file into it from the desktop.
  11. Connect your iPhone to your PC and select it in the program window.
  12. Open the Sounds tab, mark the created melody in order to synchronize it with your gadget, and click the Apply button.
  13. Disconnect the device from the PC.

If you did everything correctly, the ringtone you created will already be in the iPhone and displayed in the "Sounds" section. Well, how to put a ringtone on an iPhone, we examined a little higher.
