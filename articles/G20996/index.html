How to reset a security code in a Nokia smartphone

This article offers you several ways to reset a security code on your Nokia smartphone. Each phone of this company comes with a default code of 12345. If you care about the security of your smartphone or the personal information contained in it (for example, contacts, photos or anything else important), this function may be required. You can make settings on your phone so that access to the SIM card is blocked for third parties.



Therefore, it is important to change the default code. Thus, you protect your device. However, it happens that some users forget their Nokia security code. Often this happens for the reason that they are not used too often. If this happens, Nokia support will not be able to help you recover the code. Therefore, there is nothing left to do but disable this security feature.

How to reset Nokia security code: first way

You can perform a hard reset of the gadget. This is not quite the same as a return to the factory configuration. Such a hard reset will erase all data in the phone‚Äôs memory. Therefore, if you have access to the contents of your smartphone (if it is not blocked), please back up. Also, make sure that the phone‚Äôs battery is fully charged before performing a hardware reset.

To reset your Nokia security code in this way, press and hold below 3 keys at the same time:

  • For phones in the classical style - call button + * + 3.
  • For fully touchscreen phones - call button + Exit button + Camera control.
  • For touch phones with a QWERTY keyboard - left SHIFT + Space + BACK.
  • For Symbian ^ 3 phones (Nokia N8, C7, E7, C6-01, X7, E6) - sound reduction button + Camera control button + Menu.

After these key combinations are held down, press and hold the power button until you see the message ‚ÄúFormatting‚Äù on the screen. Release all buttons and wait until formatting is complete. After this operation is completed, all data from the phone will be deleted, including captcha.

How to restore security code in Nokia: the second way

It is possible that this method will not work with your phone. But it is worth trying before you make a hard reset of the device, because when you use it, the information is not deleted.

Download and install Nemesis Service Suite (NSS). Do not install it on the C: drive, as PC problems may occur. Better to choose drive D.



Connect your smartphone to your computer using Ovi Suite or in PC Suite mode. Close Ovi / PC Suite if the service starts automatically. You do not need it.

Open the Nemesis Service (NSS) package. Click on scan to find new devices (upper right part of the interface). Select ‚ÄúPhone - read only memory‚Äù.

Now the program will read the contents of the memory of your smartphone and save it on your computer. To view this data, go to the Nemesis Service Suite (NSS) installation directory, and then go to D: NSSBackuppm. In this folder you will see a file named {YourPhone'sIMEI}. Right-click on it and open it using Notepad. Now find [308] in this file. On the 5th record (5 =) in section [308] you will see a password. It will look something like this: 5 = 3 1 3 2 3 3 3 4 3 5 0000000000. Delete all the digits through one, starting with the first (first, third, etc.). Then remove the zeros written at the end. In this example, the standard Nokia security code is 12345.
