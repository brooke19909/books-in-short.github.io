How to enable GPS on iPhone 5s: instructions, tips, tricks

GPS navigation on modern mobile devices is used quite actively. And this is despite the fact that with the option turned on, the phone‚Äôs battery runs out faster. Next, we have to figure out how to enable GPS on the iPhone 5s. Why do we need this option? How is it useful? Even an inexperienced owner of "apple" smartphones will be able to bring the idea to life.



Description

Before you turn on GPS on your iPhone 5s, you need to understand what this is all about. This option allows you to use maps and determine the location of the phone. It helps to find the iPhone when it is lost, for example.

If this option is disabled, the user will not be able to use maps and navigation on the smartphone. Therefore, most often people think about how to turn on the GPS-receiver on the iPhone 5s and configure location in applications. Make it easier than it sounds!

Menu activation

Let's start by directly connecting the option. Initially, GPS is disabled in all "apple" smartphones. To start using it, you will have to follow a small instruction.

How to set up GPS on iPhone 5s? To do this, you will need to enable the so-called geolocation services. They allow you to use GPS in various applications, but only if you have an Internet connection. Without it, navigation will not work.

Turn GPS on as follows:

  1. Open the Settings menu when the iPhone is on.
  2. Go to the section entitled "Privacy / Confidentiality".
  3. Click on the line "Location Services".
  4. In the window that appears, put the switch in the "On" state. Further it is offered to choose a GPS operating mode. For example, "Always."

You can close the settings. After this, the GPS navigation will be fully enabled. But that is not all. Now you need to configure it in various applications. As a rule, if the program works with GPS, then at the first start it will require access to geolocation services. Just click on the "Allow" button to use all the options of this or that software.



Using the Maps service

Now it‚Äôs clear how to enable GPS on the iPhone 5s. Next, we will talk about how to work with this service. For example, about setting up location in the Maps program.

To bring the idea to life, you will need:

  1. Go to the iPhone main menu. To do this, click on the "Home" button.
  2. Find the Maps app on the screen. Click on the corresponding icon.
  3. Click on the curved map screen. A menu with settings appears.
  4. If you press the "Satellite" button, the image of the map from the satellite will pop up on the display. The blue pointer is the position of the subscriber.
  5. Select "Hybrid". This function superimposes street names and house numbers on satellite imagery.
  6. If you click on the button with an arrow in the lower left corner, a compass will appear. An orange nose points to the north.
  7. How to set up GPS on iPhone 5s? It should be noted that when working with the "Maps" application, the card rotates in the direction that the mobile device is pointing.

This is all you need to know about working with Maps. The main thing is to include the Internet and geolocation services. After that, the GPS-navigator will work without any problems.
