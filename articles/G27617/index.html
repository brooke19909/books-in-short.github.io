How to call Kiev to make it cheaper

Call to Kiev from your home phone

Let's try to figure out what you need to know for this, and what order of dialing will be correct.



First, consider how to call Kiev from your home landline phone. To do this, carry out the following actions:

  • dial eight - this is necessary to access long-distance calls;
  • wait for the beep - otherwise the connection will not take place;
  • code 10 gives us access to an international line;
  • using code 38 we get in touch with Ukraine;
  • dialed number 044 is the code of Kiev;
  • Finally, we dial the subscriber number and wait for the answer to our call.

Call to Kiev from a cell phone

Phone calls from a cellular phone differ from calls from a home device primarily in the chain of dialed numbers. We‚Äôll learn how to call to Kiev using a mobile phone. The sequence of actions should be as follows:

  • mark the + sign - this is a signal for international calls;
  • we dial the international code of Ukraine - 38;
  • Kiev code 044 you already know, dial it too;
  • if you make a call to the mobile number of the Ukrainian operator, then you must enter its code instead of the Kiev code;
  • finally, we dial the seven-digit number of the Kiev (mobile) phone.

If necessary, call to other cities instead of 044, you must enter the code of this settlement. By the way, how to call Kiev can be found on many Internet sites or by calling the help desk.

Call from Kiev to Russia

Since October 2009, calls to Russia from Ukraine have been carried out according to the new dialing procedure. It all depends on the code of the city where you will call.

When making a call to a landline phone, we dial 0 and after the signal we press 0 again - this is the way to international communication. Next, enter 7 - the code of Russia, then the code of the settlement and phone number.

When making a call from the Ukrainian capital to Moscow, you should first clarify which code you need to dial, since two codes are used for international calls to Moscow - 495 or 499.

When making a call from Ukraine to a mobile phone in Russia, they first dial 0, and after the signal they press again 0. Next, they indicate the operator code and the subscriber number of seven digits.

You can call Russia using the services of a call center. True, in this case, the call tariff may be 1.5 times the cost of a call from a landline phone.
