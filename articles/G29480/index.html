How to install iTunes and start using the program: a guide for "dummies"

Every year, Apple products are becoming increasingly popular. Especially for owners of fashionable technology, iTunes was developed. Many have heard about it, but few know how to install and use it in the future.





The first thing iTunes attracts attention to is a stylish and bright, “tasty” interface. Using it is very convenient, but according to statistics, all users need some time in order to get used to the program. Essentially, iTunes is designed to exchange files between a computer, as well as iPod, iPhone and iPad. In addition, the program supports the functionality of a media player.

To get started, let's figure out how to install iTunes. First of all, you need to download the installation file. It is best to take it directly from the official Apple website - so you will be 100% sure of the program’s performance and not expose the system to the risk of a virus attack. You can configure to receive the latest news and updates directly to your e-mail. The program downloads for free.



Further, guided by the prompts of the installation wizard, we choose where and how to install iTunes. Most likely, the computer will ask for permission to install new software. You can safely agree - if you downloaded the installation file from an official source, it carries absolutely no threat.

In fact, installing iTunes is the same as installing any other program. Before you can start working with a media player, you will also need to agree to the terms of the license agreement.

Another popular question: "How to install iTunes on multiple devices at the same time and synchronize the created library?" There are several subtleties to starting using it on multiple devices at once, which you should be aware of.

So, first of all, you need to hold the Shift key and run the program itself. The system will display a message asking you to select an existing media library or create a new one. Now we need the “Create” option. Then everything is simple: we come up with a name and save the created file in the appropriate folder. After that we add the folders and files of interest to us in the created library. In the future, it will be possible to switch between the data of different media libraries in the same way that we used, opening iTunes to create a new one. Namely - opening the program with the Shift key held down. The only difference will be that instead of the "Create" button, you will need to click the adjacent "Select" button.





And finally, the last topic I want to cover is installing applications through iTunes. Perhaps for owners of Apple technology this is one of the most burning issues.

In fact, everything here is much simpler than it seems: when registering, you should immediately enter your credit card number, from which the funds spent by you in the store will be debited. However, if you do not want to spend money, then you can not do it. A huge number of games and applications in the App Store can be downloaded for free or use the demo versions for review. If you wish, you can register several accounts at once. What is it for? The fact is that some files available, for example, in Europe or America, for Russian users can be closed. The creation of two accounts will easily solve this problem and freely download music, films and clips from around the world.

Understood how to install iTunes? Then welcome to the limitless world of entertainment and fun!
