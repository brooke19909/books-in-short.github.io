How to reset all settings to factory defaults on a smartphone?

This article will be devoted to how to reset all settings from the phone. It often happens that a smartphone starts to fail unreasonably. Or it happens that, when setting up the device, actions were taken that worsen its operation. For such cases, there is a function for resetting to factory settings. This will be discussed in our article. All features and methods will be discussed in detail below.

Reset settings via the smartphone’s main menu

How to reset all settings on Android to factory settings? There are three ways to do this. Let's look at the first one.

In order to reset the smartphone to factory settings based on the Android OS, you need to go to “Settings”, select the “Privacy” section there and click on “Reset Settings”. Next, select one of the items - “Data Archiving” or “Auto Recovery”.

These items are necessary so that you can cancel the restore and return the settings of the smartphone before resetting the settings.

Next, click "Reset Data." This will lose some data. For example, a Google account, as well as all installed applications and games will be deleted.

All that will be lost from the smartphone when resetting the settings is indicated in the notification before this event. After reading the information provided, you must confirm the selected action and wait for the end of this procedure.

Service code

How to reset everything to factory settings using a service code? About the second method later in the article.

On each device with the Android operating system, there are a number of service codes for various purposes. One of these codes is used to reset the data of the smartphone to the factory.

In order to perform a hard reset, you need to do the following:

  1. Go to dialing.
  2. Enter the code necessary to reset.
  3. Click on the call.
  4. Wait for the reset procedure to complete.

Each phone model and version of the Android system has its own reset code.

Code entry examples:

  • * # * # 7378423 # * # *;
  • * 2767 * 3855 #;
  • * # * # 7780 # * #.

Use Recovery (Reset with special keys)

How to reset all settings? Each phone has its own combination of special keys.

The following combinations are possible on Android:

  1. "Volume down" + "Turn on the device." This is one of the most common combinations used on many phones. Try it first. If this does not work, then see the list further. The combinations below may help.
  2. “Volume up” + “Volume down”.
  3. “Turn on the device” + “Home” button + “Volume up”.
  4. “Volume up” + “Volume down” + “Turn on the device”.
  5. "Volume up" + "Home" button.

These are standard button combinations for resetting smartphones on Android. If none of them worked, see the instructions for your phone model. They must be held for several (2-5) seconds.

After pressing these keys, the system menu (Recovery) will open. Using the "+" and "-" buttons (increase and decrease the sound), you need to find the line that says: wipe data / factory reset, Clear eMMC or Clear Flash. Select it, click on the "Home" button. Select Reboot System. After that, all settings will be reset to the factory settings.

Returning to the factory settings on the iPhone

How to reset iPhone to factory settings? On Apple devices, you must use the Reset function. You will need to go to the "Settings" - "General" section and select the "Reset" item.

Now let's analyze what is in this section:

  1. "Reset all settings." This function is very convenient for those who want to delete all entered settings from the start of using an iPhone, without losing any other information. By choosing this option, all installed settings will be reset on the device. They will be reduced to factory. That is, to those that were installed upon the acquisition of the iPhone. In this case, your personal data will not be affected, as well as all media files created up to this point (photos, videos, music, etc.). It is selected when unwanted settings were entered, and it is possible to return everything back as it was.
  2. Erase content and settings. This type of reset is applied when the iPhone is put up for sale, or given to someone for a short time. After “Erase content and settings” was selected, all personal data and media files will disappear on the “iPhone”, and the settings will be returned to the factory form. After clicking on “Erase content and settings”, a warning will be displayed and that all data from the “iPhone” will be erased. Confirm the reset (“Erase iPhone”), after which your device will automatically reboot. This is a very simple tool for those who want to get rid of all the data on the smartphone at once. It will take about five to ten minutes. After rebooting, the screen appears a request to enter user account information. To do this, you will need an Internet connection. After that, you need to enter the initial settings at your discretion.

Reset Network Settings

This type of reset is necessary when the iPhone does not catch the network, the Internet or wi-fi. You can pull-insert a sim card. But on the "iPhones" this requires a special clip. Therefore, it is easier to use “Reset Network Settings”. In this way, in most cases, problems with mobile network, Internet or Wi-Fi connections are solved.

If the password is forgotten ...

If you forgot your password, how to reset all phone settings? Now we will understand this issue. Many smartphone users put individual passwords on their gadgets. This greatly helps in protecting the phone from other people. However, it happens that this password has been forgotten and the phone is locked. To avoid entering this password, you must turn off the phone in emergency mode and immediately turn on the settings section when turning on, without waiting for a lock or a request to enter a password.

Next, you need to reset all phone settings and restart the phone. Password will be reset automatically. Regardless of what kind of smartphone you have, each can be reset as necessary. Using this function, you can reset the phone type settings (Samsung, iPhone, Lenovo and others).

For some, a password may be required. In most cases, it is standard - 0000. If this password does not work and an error is entered in entering it, then you need to contact the phone’s manual or the service to reset the settings.

It so happens that the wrong actions were taken with an independent attempt to reset all the settings of the smartphone to factory settings. In this case, you need to contact the smartphone repair service of your model (iPhone, Samsung, Lenovo, etc.).

It will cost some amount of money. But the reset will be performed reliably and without errors.

Important information!

When wondering how to reset all settings, remember: absolutely all the information on your smartphone will be reset.

In this case, it will be deleted: saved phone user accounts, multimedia files. They will also be permanently deleted: photos, videos, music, etc. (personal information, messages, contacts, dialed numbers, time zone, location and other user data entered).

In order to reset to factory settings and not lose all this data, a backup copy of the account is created. Information about the account and all files stored on it get into it.

This is necessary when the reset must be canceled, or return everything as it was before the operation.

For example, this action may be performed by mistake. Then at any time you can return everything to its place and restore the account with all the lost data.

User Tips

A backup is done before resetting. Otherwise, when restoring the account, the last backup will be used, which can be done for a long time and it may not contain many files, records and other necessary information.

In this situation, unrecovered data is lost. And in order to return them, you will need to contact the service center, where with the help of additional software you will recover all the lost information.

In addition to all this, installed applications will be deleted, among which may be purchased. In this case, to return them and reinstall, you will have to pay again for the purchase of this program.
