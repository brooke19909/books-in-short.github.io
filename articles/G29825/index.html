Crochet loop: basic knitting principles and uses

Crochet today at the height of fashion. Thus, they create not only sweaters, vests, but even swimwear, summer hats and toys. If we talk about ordinary knitting, then everything starts with a chain of air loops. If you knit in a circle, then here you need a sliding loop, crochet knit. Let's figure out how to do it.



Benefits of this technique

You can start knitting with the usual loop, but in the end the center of the product will be with a small hole. A crochet sliding loop provides a denser canvas, things hold their shape better. This is especially important when it comes to knitted toys. They are also called amigurumi. Therefore, the second name of this loop is amigurumi. When a toy is stuffed with synthetic winterizer or cotton, it is important that it does not stick out at the place where the knitting began. To master this loop is quite simple. Here is a detailed guide.

Start knitting

So, you only need a hook and any thread. First try on a thick thread to work out the technology:

  1. Wind the thread on two fingers so that a whole circle is obtained.
  2. Remove the thread from your hand, holding it with your fingers so that it retains the shape of a large loop.
  3. Place the tail of the thread under the circumference, pull the thread crochet.
  4. Pull the ends of the thread and tighten the loop. It‚Äôs a sliding loop (crocheted), so it can stretch and contract like a tie.
  5. All you can start knitting air loops.

For beginners and continuing

There are more advanced techniques for crocheting a sliding loop. For example, you can wrap the thread around your fingers twice, and then stretch the second thread through the loop formed by the first. This is a more complex description, far from always understood by beginners. It is easier to use the algorithm indicated above. To check that you have followed the instructions correctly, try tightening the loop. The loop should slide freely along the thread, decreasing and expanding as you wish. If a knot is formed, then the thread can be cut, and the loop is better to bandage. After the sliding loop is ready, you can start knitting a chain of air loops.

Simple circle

To start knitting, make a sliding loop. Now you need to knit three air loops using the first. Now connect them in a circle, tightly fastening the thread. Start knitting in a circle as follows:

  • First row: Double crochet a double crochet in each loop.
  • Second row: double-knit every second loop.
  • Third row: double-knit every third loop.

In this way, continue knitting, constantly increasing the distance between those loops that you knit twice. So you knit only when you need a flat circle. If a ball or sphere is needed, adjust the shape of the object by reducing and adding loops. So you can even tie a snowman without interrupting the thread, if you know how to make a sliding loop with a hook.

Knit a ball

To make a ball, you should begin to reduce loops in the middle of knitting. To do this is simple: first knit one loop to the middle, then immediately the second. Finish knitting these loops by passing the thread through two loops at the same time. You need to reduce the loops exactly according to the same principle as you added them. If you knit a toy, do not forget to leave a hole for filling the mold. Keep the packing tight. Once the figure has taken the desired shape, you can tie it to completion.

Knitting patterns

Here are just a few examples of how to use this technique.

  1. Slingbuses . Bright beads will decorate mom, and will also be the best toy for a child. Make them simple. Tie the wooden beads with colored threads. How to start, you already know - use a sliding loop, and then knit in a circular pattern, gradually adding loops.
  2. Rattle Making a children's toy is very simple. All you need is a plastic chocolate egg mold in which you put a couple of dried peas or beads for ringing. It remains only to tie the toy with bright threads and add a pen.
  3. Toys All such products are knitted in a circular pattern. Each element is created separately, and then everything is sewn together or attached in a different way. With this loop, your toys will keep their shape, and cotton wool will not leak through the hole, where knitting starts.
  4. Hats . Whatever thread you use, when crocheting a crochet loop is useful to you. Next, knit in a circle, alternating air loops and crochets for summer options, as well as using continuous knitting for winter models.

Conclusion

Applying this technique is simply necessary if you plan to create things for sale. So, your knitting will look just professional. In crochet, there are many different patterns and loops. However, if you know how to crochet a sliding loop, then you are guaranteed a good start to knitting. As a result, any thing will be more accurate.
