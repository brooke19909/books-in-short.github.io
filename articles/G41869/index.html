How to reinstall drivers to sound: methods and instructions

It often happens that after installing Windows, the sound disappears. This, of course, is connected with many things, but often it is the drivers. The article will discuss how to reinstall drivers for sound in Windows. Various methods will be considered, and detailed instructions will be attached to each of them.

Driver definitions

Before undertaking reinstallation of the driver, it is necessary to understand which one we will reinstall. To do this, you can use many utilities and programs. But why download something if you already have what you need on your computer?

We need to get into the device manager. There are two ways to do this.

  • Click on the "My Computer" shortcut with the right mouse button and select "Management" in the menu that opens. A window will appear. It is necessary to select the tab "Device Manager".
  • Enter the Start menu and enter ‚ÄúDevice Manager‚Äù in the search bar. After that, the system will find the utility we need, and it remains only to click on it with the left mouse button.




So, we reached the first goal - we entered the device manager. Now you need to find out the name of the driver. Due to the fact that we are analyzing how to reinstall the drivers on the sound, and the example will manipulate it. Find the Sound, Gaming, and Video Devices tab and open the drop-down list. Now you have a whole list of drivers for sound. As a rule, if some do not work correctly, a yellow icon will be drawn next to them.

Reinstall using device manager

After completing the steps above, you find yourself in the device manager and find the names of the drivers that need to be reinstalled. First reinstall those with a yellow sign next to them.

To do a reinstall, you need to:

  • Right-click on the driver and select the "Properties" line in the menu.
  • In the window that appears, go to the "Driver" tab.
  • At the bottom of the window, click the "Delete" button.
  • In the confirmation window, install a bird next to the line "Uninstall driver programs for this device" and click "OK."

Now the incorrectly working driver is removed, and in its place you need to put a working one. To do this, repeat the manipulations until the "Properties" window appears with the "Driver" tab open. In the window now, instead of "Delete", you must click the "Update ..." button. You will be asked where to search for a driver. If you downloaded it in advance to your computer, then select the first item. Otherwise, you will need to perform a search over the Internet.





Now you know how to reinstall the drivers for sound in the device manager. But this is only the first of many methods, so move on to the next.

Search for driver by ID

We have already figured out how to reinstall the drivers to sound using standard Windows utilities and using programs. Now let's talk about how to find the right driver on the Web by ID.

  • In the Device Manager, enter the properties of the required driver.
  • Click the Details tab.
  • In the drop-down list, click on "ID Codes"
  • Copy any line from the list.
  • Go to the DevID website.
  • Paste the copied code into the search box.

Now it remains only to download and install the driver.
