Via iTunes

First, we will look at how to recover deleted SMS on an iPhone using a specialized program called iTunes. This method is suitable only for those who made a timely backup of the data using the named application. It is important to know that after recovery all new SMS will be erased. The smartphone will return to state on a given date.





To restore SMS on an iPhone, you have to:

  1. Back up your data through iTunes in advance.
  2. After deleting messages, connect the iPhone to the computer via USB.
  3. Launch iTunes. Select your device in the list of devices. Sometimes you need to go to the "Sync" menu item to connect your phone to a PC, but usually this step is skipped.
  4. Go to the "General" tab.
  5. Click on the "Restore iPhone ..." button on the right side of the window.
  6. Confirm action. To do this, you will have to enter data from AppleID.
  7. Select a backup copy of the recovery information.
  8. Wait for the process to complete.

After the steps taken, the iPhone will reboot. All data on it will be restored. Messages and other settings will return to their original state.

Work with iCloud

And how to restore SMS on iPhone 5? In the same way as on all other versions of the smartphone. If you do not want to work with iTunes, you can resort to iCloud. This service allows you to backup data, upload it to the "cloud" and restore, if necessary.

Typically, the "apple" gadgets in the settings included the option to automatically create backup data in "iCloud" when connecting devices to the network. Nevertheless, it is recommended that you independently create a "rollback point" in iCloud.





And to solve the problem of how to restore SMS on an iPhone through a data cloud, you need to do the following:

  1. Erase all data on the phone. You can do this by visiting the sections: "Settings" - "General / Basic" - "Erase All".
  2. Confirm the operation by typing the password from AppleID.
  3. Wait until the smartphone reboots.
  4. Select "Recover from iCloud" in the menu that appears.
  5. Re-enter the data from the Apple ID profile.
  6. Select a backup to restore.

Now you just have to wait a bit. The device will reboot - data will be returned on the specified date.
