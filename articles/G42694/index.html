Activating options on “Tele2”: how to activate the “Who called” service?

Did your cell phone go off due to low battery, did it activate air mode, or did it fail? Nothing wrong. Even if for some time your number was unavailable, the Tele2 company will notify you of who tried to call during the “absence”. You can not only get information about missed calls, but also find out how many calls and at what time was made on Tele2. How to activate the “Who called” service, how it works, how to configure it correctly - this is what this article will discuss.





Service Description

Before you talk about how the activation of the service and its configuration occurs, I would like to give a general description of it. “Who called” - this option appeared in the mobile operator under consideration for a long time. At the same time, the principle of its functioning for the current day remained the same. The only thing that has changed is the fee for its use. In the first time after appearing on Tele2 (we will consider further how to activate the “Who called” service), the service was provided for free.

It is included in the basic package of services and is activated automatically on all numbers (when purchasing a Tele2 number, be sure that you will find it in the list of options connected by default).





“Who called” service (on “Tele2”): cost

You can clarify the actual cost of the service for your region on the mobile operator’s website by going to the appropriate section with the list of services. However, it should be noted that the connection is free. There is a daily service charge. For example, in the Tula region it is 50 kopecks.

On the tariff plans that are used for the Internet, there is no subscriber at all. pay. But for the TP Black line, it is included in the available fee under the tariff plan.

Activation on “Tele2”: how to activate the “Who called” service?

There is no need to connect the service, since it is a basic option and is included in the list of connected services by default. However, if you do not receive messages about missed calls, then several options are possible:

  • the option has been deactivated (by a user or customer support employee);
  • Call forwarding was previously configured on the number (if there is a valid call forwarding on the number, it is impossible to use the "Who called" service);
  • since the “Who called” service on “Tele2” is paid, then in the absence of funds in the account sufficient to write off the daily payment, it is suspended.

The solutions to each situation are as follows:

  • activate the option again, for this you can use the command * 155 * 331 # (or use the web assistant or its analogue - an application for mobile devices);
  • restore redirection to the “Tele2” service number for the correct operation of the service (for each region this number is individual, the full list can be viewed on the operator’s portal or by calling the contact center number);
  • deposit funds into the account.




If the above tips did not help restore the correct operation of the service and notifications of missed calls are still not received, then you should contact the contact center for advice (at 0611).

Conclusion

In this article, we found out that the service we are considering is the basic one on Tele2. How to activate the “Who called” service if it is not on the number? This can be done independently using the tools of the web cabinet (or the functionality of the mobile application, which is its analogue) and standard USSD requests.

Also, a customer service specialist can help with setting up and activating the service: you just need to call 0611 and explain the situation.

We hope you find these tips helpful.
