Verification Methods

There are several options that will allow you to achieve a positive result. We will consider all available methods and provide the necessary instructions. And first of all, we note the most effective and simplest option associated with calling the operator. To implement it, it is enough to do the following:

  1. Pick up the phone.
  2. Dial 611, press the call button.
  3. An answering machine will answer you and ask you to perform several actions.
  4. After a few minutes, the operator will answer.
  5. Explain the situation to him and request the necessary information.

This method is the simplest, however, it requires several minutes of free time. If suddenly you do not want to call, but you need to know the tariff, you can use another option. The following method involves using the official resource of the mobile operator:

  1. First you need to launch a browser to access the Internet.
  2. Go to the official website.
  3. Click on the "Enter your account" button in the upper right.
  4. Indicate your mobile number.
  5. Enter the code that will come to you in SMS.
  6. As soon as you get to the main window, there you will immediately see the name of your tariff.

This method is not the most popular, but in most cases it helps to resolve the issue. And if you want to use the standard option, then you need to know the USSD command. For convenience, we have compiled instructions for you:

  1. Pick up the phone.
  2. Enter the command * 107 #, press the call button.
  3. A window will immediately appear on the screen of the mobile phone, where the name of your tariff will be indicated.

This option is convenient only if you just want to find out the name. To get additional information about the cost and packages, in any case you have to visit the site or call the operator.
