How to set a password on iPhone



Almost all mobile devices for security are equipped with the ability to protect personal information from the wrong hands, including gadgets on iOS. In the device settings, in order to be able to set protection, the tab "Password protection" is provided. There you can set any protection configuration, for example, set the period of time after which the device is blocked. An interesting feature is the ability of the device to self-destroy all data if the password is entered incorrectly 10 times. However, it is not recommended to use this function for those who have a lot of necessary information stored in the device. This menu item appeared on the iPhone 5, in iOS firmware version 7.1, which appeared in March 2014.

  • After 6 incorrect entries, the device displays a message stating that it is necessary to try entering in a minute.
  • After 9 erroneous code entries, the device will be temporarily locked for 1 hour. The message will look like this: ‚ÄúiPhone is disconnected, repeat in an hour‚Äù.

How to install iTunes



In order to download the installer of software for working with the device through a personal computer, you need to go to the official website of the manufacturer Apple. How many times can a counter be reset



You can carry out the procedure any number of times. Users have verified that the reset occurs during any interaction between the gadget and iTunes. Data backup

You can create a copy of information from any gadget in two ways:

  • Through the application in the iCloud device itself.
  • Using iTunes.

For the first method, you will need to register an Apple ID account, information will be stored on a "cloud" server. Synchronizing a device with it is possible only with the help of authorization in the application. In order to keep a copy of the information in iTunes, you must synchronize the device with it. The developers recommend that owners do this procedure periodically so that the information on the computer is up to date. If the iPhone is disconnected and its recovery is necessary, the data will not be permanently lost.

Putting the device into DFU mode

Device Firmware Update Mode - device debugging mode, it is possible to restore the device's factory settings in it. When switching to this mode, it is possible to install the device‚Äôs operating system from scratch. In the instructions for restoring the device‚Äôs health, if the iPhone asks to connect to iTunes, this mode is often mentioned.

To go into recovery mode, you must perform the following manipulations:

  • Connect the device to the PC and start iTunes.
  • Press and hold the power button for a few seconds.
  • Then, while continuing to hold the power button, press Home and hold them together for a while.
  • After about 10 seconds, release the power button, and continue to hold Home for some more time.

If all the manipulations were performed correctly, then iTunes will display a message stating that the device was detected in DFU or recovery mode.

How to restore access to a device using recovery



After you enter the gadget in the DFU state, iTunes will first respond and a message will appear in the program window that the found device must be restored before use. To start this process, click OK. Once again, you need to confirm your decision and start the recovery process on the side panel, which displays the model and technical data of the device. After launch, you must confirm your agreement with the provisions of the license. The recovery procedure is not quick, during the recovery download a message may appear that the iPhone is not connecting to iTunes. Do not take any action until the update download is complete. Then put the device back into recovery mode and update the device firmware via iTunes. As a result of the work done, you can again use your device. Initially, it will be absolutely clean: applications, photos, contacts and more will disappear. If you have a backup copy of this data, then a few minutes are enough - and your phone will become almost the same as it was before the password was forgotten.
