How to use iTunes for iPhone? Some features of the program

Those who changed their mobile phone to Apple products, at first it is quite difficult to use new devices. The difficulty lies not in the differences between the usual operating systems from the iOS OP, but rather in the inability to download videos, music, books and install applications from the Internet. In order to do this, users need to familiarize themselves with the iTunes program. This is a kind of file manager that has various functions, including a media player. There is a lot of information on the network about how to use iTunes for iPhone, but for a beginner it is best to use articles in the form of step-by-step instructions.





Where to download iTunes?

The very first mistake owners of brand new iPhones is to try to buy the program on paid sites. iTunes is distributed free of charge from the official portal of Apple. In the upper right corner of the main page window there is a blue button - ‚ÄúDownload‚Äù, clicking on which anyone will receive the original version of the necessary software for the American smartphone.

Before downloading iTunes for iPhone 4, as well as other versions of the gadget, users should familiarize themselves with the requirements for computer operating systems, the amount of RAM, and the amount of free space on the hard drive. 2 minutes of reading the instructions will help to save an hour (and sometimes more) that would have to be spent on unsuccessful attempts to install the wrong version of the program on your computer.





How to use iTunes for iPhone?

After downloading and installing, it would be nice to understand what operations can be performed using this program. Launch iTunes. Unlike the first time, all subsequent launches of the program will be accompanied by proposals for updating the software version. Users who are satisfied with the installed option can check the box on the menu and not receive such messages at all, but the latest versions provide more options to organize files, make it easier to work with videos and music.

After starting the program, the user can access the tools for adding various media files that are located on the computer‚Äôs hard drive. You can transfer and install music, movies, and books using the Media Library menu.





Tuna, music, device

Consider the example of adding the selected composition, moving the file from the computer to iTunes for iPhone 5, and then transferring it to the gadget itself.

  1. In the upper left corner of the program there is a small icon in the form of a square divided into two parts of different colors. Next - an arrow, clicking on which, the user will see a drop-down menu. In it, you need to select a line with the name "Add file to your library."
  2. In the explorer window that opens, specify the path to the selected composition.
  3. After adding the file, you must connect the iPhone to the computer using the USB cable. An icon with the image of a smartphone will appear in the upper right corner of the computer - this is the direct menu for working between iTunes and the device.
  4. You need to click on the icon, but not on the arrow next to it. In the window that opens, the user will see the main page, from which it is necessary to go to the "Music" section.
  5. The program will provide a selection of some parameters. For the first time it will be enough to mark the line ‚ÄúSelected artists, genres, albums‚Äù and decide on a playlist.
  6. In the lower right corner of all sections of the program there is an icon "Synchronize". Clicking on it will be the last step to adding the selected composition to the device.

The synchronization operation takes about 2 minutes, after which the iPhone can be disconnected from the computer and enjoy the sound of your favorite song.

How to use iTunes for iPhone if you want to watch a movie on your gadget? Just as in the case of audio files, only for transferring video to the device you will need to select the ‚ÄúVideo‚Äù tab. To add books - go to the section with the appropriate name.





iTunes Store

The iTunes Store is a convenient online service in which owners of apple smartphones can also purchase by bank transfer and download, without paying, various films, albums of favorite artists, software products for their devices. At some point, the user will definitely want to download applications, games, music from the Apple store. iTunes for iPhone 5s provides such an opportunity. But to buy apple company products, you need to register an Apple ID. The easiest way to do this is on the official website. When registering, you must be prepared for the fact that you have to enter a credit card number. Use it to buy any product on the iTunes Store.
