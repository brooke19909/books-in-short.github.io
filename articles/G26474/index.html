Theoretical Foundations of Electrical Engineering: Nodal Stress Method

The method of nodal stresses is the calculation of electrical circuits in which the voltage values ‚Äã‚Äãin the nodes of the circuits relative to the base node are variables. The equations are compiled on the basis of the first Kirchhoff law, which allows us to reduce the number of system equations to a value of k-1, where k is the number of chain nodes. This method is best used when the number of branches of an electric circuit is more than two. The method of nodal stresses has found application in computer programs for modeling electrical circuits, due to the simplicity of the algorithm for generating equations of nodes.



Nodal voltages are the voltages between an arbitrary reference node (in it, the potential is taken equal to zero) and each of the nodes. In the diagrams, the reference node is displayed grounded.

Consider various methods for calculating electrical circuits

The essence of this method is to solve a system of equations by which the potentials of each circuit node are determined with respect to the reference node. After that, the circuit is calculated using Ohm's law, that is, the values ‚Äã‚Äãof the currents of all branches are determined.
